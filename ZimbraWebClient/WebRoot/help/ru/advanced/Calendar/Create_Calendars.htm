Создание ежедневников

Для пользователя создается один ежедневник, также можно создавать дополнительные ежедневники для ведения учета функций различных типов.  Например, можно создать ежедневник для рабочих, личных мероприятий и мероприятий компании.  Можно выбрать вид Расписание для одновременного просмотра ежедневников.

Cоздание ежедневника

  1. Перейдите на вкладку «Ежедневник».

  2. В области просмотра ежедневника щелкните значок или щелкните на панели инструментов стрелку   и выберите пункт Ежедневник.

  3. В диалоговом окне Создать новый ежедневник введите название ежедневника и выберите цвет для отображения мероприятий данного ежедневника.

  4. Если просмотр другими лицами мероприятий ежедневника нежелателен, установите флажок Исключить этот ежедневник при составлении отчета о свободном/занятом времени.

  5. Чтобы создать ежедневник и синхронизировать встречи с удаленным ежедневником, установите флажок Синхронизировать встречи из удаленного ежедневника. Введите допустимый URL-адрес файла iCal удаленного ежедневника.

  6. Нажмите кнопку ОК. Новый ежедневник появится в списке ежедневников.

Удаленные ежедневники периодически обновляются в соответствии с интервалами опроса удаленного ежедневника, установленными системным администратором.
