Сохранение содержимого вашей  учетной записи

Пользователь имеет возможность экспортировать все данные учетной записи, в том числе сообщения и вложения, контакты, ежедневник, задачи, папки документов (блокнотов) и портфеля, или экспортировать отдельные элементы учетной записи.  Данные можно сохранить на компьютере или в другом месте в виде архива. Данные учетной записи сохраняются в архивном файле формата Tar-GZip (tgz), который можно импортировать обратно для восстановления учетной записи.

  •  В случае экспорта отдельных папок контактов они сохраняются в файлах формата csv; а отдельные папки ежедневников сохраняются в файлах формата ics.

Данные только копируются, но не удаляются из учетной записи.

 Экспорт файла можно выполнить, открыв вкладки Настройки >Импорт/Экспорт.

Как экспортировать данные

  1. На вкладке Настройки   откройте папку Импорт/Экспорт для той учетной записи, которую нужно экспортировать.

  2. Экспортируйте все данные учетной записи или укажите для экспорта папку определенного типа.

    • Чтобы экспортировать все данные учетной записи, выберите пункт Учетная запись. Примечание. Корзина и папка «Спам» не экспортируются.

    • Чтобы экспортировать список контактов или определенный ежедневник, выберите соответствующий вариант. В случае экспорта ежедневника или списка контактов необходимо указать формат файла для экспорта. См. раздел Экспорт элементов из учетной записи.

    • Чтобы выбрать несколько разных типов элементов для одновременного экспорта, установите флажок Дополнительные настройки. Окно приобретет расширенный вид. Установите флажки для элементов данных, которые требуется экспортировать. В указанную папку будут включены все элементы данного типа изо всех папок. Если требуется экспортировать данные определенного типа, для указания типа данных можно воспользоваться фильтром поиска. См. раздел Описание языка запросов и Примеры поиска.  

  1. Нажмите кнопку Экспорт и выберите место для сохранения файла.

  2. Нажмите кнопку ОК.

  3. Экспортированные данные учетной записи можно просмотреть с помощью программы архивирования, например WinRAR .

 
Как импортировать сохраненные данные учетной записи

Учетную запись можно восстановить из экспортированного файла.  

  1. На вкладке Настройки   откройте папку Импорт/Экспорт.

  2. В разделе Импорт, тип выберите тип элемента для импорта: учетную запись, ежедневник или контакты. В случае импорта ежедневников или контактов укажите формат для файла импорта. См. раздел Импорт контактов.  Если указан тип  «учётная запись», в учётную запись может быть импортирован только ранее экспортированный файл формата tgz.  

  1. Нажмите кнопку Обзор и найдите файл для импорта.

  2. Укажите место назначения.

  3. В случае импорта данных учётной записи выберите соответствующий вариант в полях Разрешение конфликтов дубликатов.

    • Игнорировать. Конфликт дубликатов остается неразрешенным. Добавляются все файлы, а существующая версия элемента сохраняется.

    • Заменить. Элементы, изменённые после экспорта учётной записи, возвращаются в исходном состоянии в экспортируемый файл. Добавляются старые элементы. Новые элементы, добавленные в учётную запись после создания экспортируемого файла, сохраняются.

    • Сброс. В случае выбора этого параметра все элементы указанного типа в выбранной папке удаляются перед импортом новых элементов.

  1. Если  требуется импортировать данные определенного типа из файла формата tgz, установите флажок Дополнительные настройки, чтобы придать окну расширенный вид, в котором можно выбрать типы элементов для восстановления.

В разделе Типы данных выберите для импорта нужные типы элементов из сохранённого файла формата tgz. После нажатия кнопки «Импорт» импортируются только элементы указанных типов.

  1. Нажмите кнопку Импорт.
