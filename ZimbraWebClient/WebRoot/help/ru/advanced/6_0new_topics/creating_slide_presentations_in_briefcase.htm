Создание презентаций в Портфеле (бета-версия)

Работая в Портфеле, можно создавать презентации. Другие программы для этого не требуются. При создании презентации можно:

  • выбирать цвет из меню тем;

  • вставлять изображения, которые хранятся на компьютере пользователя;

  • пользоваться окном ввода текста для компоновки макета слайда;

  • форматировать слайд, выбирая разные шрифты и стили.

 

  1. В Портфеле выберите пункт Новая презентация, в открывшемся окне введите имя презентации. Нажмите кнопку ОК. Будет отображен первый слайд.

  2. Выберите цвет фона страницы презентации, щёлкнув ссылку Темы.

  3. Добавьте содержимое. Щёлкните окно текста заголовка и добавьте заголовок, затем щёлкните окно текста содержимого и введите текст. Можно выполнить следующие действия.

    • Изменить размер окна текста и добавить дополнительные окна для ввода текста.

    • Изменить тип, стиль и размер шрифта с помощью панели инструментов, расположенной над слайдом.

    • Щелкните пункт Вставить изображения и добавьте  изображения.  Изображения сохраняются в папке Портфеля.

    • Щелкните пункт Вставить слайд и добавьте еще один слайд.

  4. Нажмите кнопку Сохранить, чтобы сохранить слайд.  Чтобы закрыть слайд, щёлкните X на вкладке браузера.
