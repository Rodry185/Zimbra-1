Импорт  элементов в учетную запись.

На вкладках «Настройки->Импорт/Экспорт» можно выполнить импорт всех данных учетной записи или отдельных папок, ежедневников и списков контактов.

Импорт контактов

Можно импортировать списки контактов,  сохраненные в файле с разделенным запятыми текстом (csv). Импорт можно выполнить, открыв вкладки Настройки>Импорт/Экспорт.

  1. На вкладке Настройки откройте папку Импорт/Экспорт.

  2. В разделе «Импорт»  выберите тип данных Контакты.

  3. В раскрывающемся меню рядом с пунктом Контакты выберите тип импортируемого файла контактов.

    • Контакты Yahoo!

    • Контакты GMail

    • Контакты Outlook 2000

    • Контакты Outlook 2003

    • Контакты Outlook

    • Контакты Thunderbird

 В списках контактов разных учетных записей данные контактов отображаются по-разному. Выбор правильного типа импортируемого файла контактов позволяет определить порядок полей для корректного отображения данных в адресной книге.

  1. Нажмите кнопку «Обзор» и найдите файл формата csv для импорта.

  2. Нажмите кнопку Импорт. В диалоговом окне Выбрать адресную книгу выберите существующую адресную книгу для добавления контактов или создайте новую адресную книгу, в которую будет импортирован файл контактов.

  3. Нажмите кнопку ОК. Содержимое файла контактов будет импортировано.

Импорт ежедневника iCal

Пользователи могут импортировать ежедневники iCal, сохраненные в формате .ics. Ежедневник iCal является стандартом для обмена данными ежедневников в интернете, а формат .ics является стандартным форматом для данных ежедневников iCal. Импорт можно выполнить, открыв вкладки Настройки>Импорт/Экспорт.

Можно импортировать ежедневник iCal в существующий ежедневник или создать новый ежедневник, в который будет импортирован файл ics.

Как импортировать ежедневник iCal

  1. На вкладке Настройки откройте папку Импорт/Экспорт.

  2. В разделе «Импорт» в поле Тип  выберите пункт Ежедневник.

  3. В поле «Файл» укажите файл ics для импорта.

  4. Нажмите кнопку Импорт. В диалоговом окнеВыбрать папку выберите папку для импорта ежедневника: выберите один из существующих ежедневников или создайте новый ежедневник.

  5. Нажмите кнопку ОК. Ежедневник импортируется в учетную запись.

Ежедневник iCal периодически обновляется в соответствии с интервалами опроса удаленного ежедневника, установленными системным администратором.
