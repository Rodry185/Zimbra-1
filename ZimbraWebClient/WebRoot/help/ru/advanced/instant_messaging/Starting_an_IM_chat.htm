Начало чата МС

После добавления приятелей можно начать с ними чат МС.

Как начать чат

  1. Дважды щелкните имя контакта, с которым необходимо начать чат. Откроется окно чата МС с именем приятеля в заголовке.

  2. В нижней части окна чата введите текст сообщения и нажмите клавишу Enter или щелкните Отправить. Сообщение отобразится в окне чата под вашим именем.

  3. Когда приятель отправляет ответное сообщение, оно отображается в окне чата МС под его именем.

  4. Чтобы завершить чат, нажмите клавишу Esc или щелкните красный значок «X». Окно закроется.

Запись чата по умолчанию сохраняется в папке «Чаты», которая находится в области обзора на вкладке «Почта». Для получения сведений об изменении параметров сохранения см. раздел Настройка параметров мгновенных сообщений.

Использование смайликов в мгновенных сообщениях

В чате МС можно использовать смайлики Yahoo!. Щелкните стрелку на кнопке с желтым смайликом, чтобы открыть раскрывающееся меню со смайликами.

Щелкните необходимый смайлик для его вставки в текстовое поле. Введите сопровождающий текст для смайлика, либо нажмите клавишу Enter или щелкнитеОтправить, чтобы отправить только смайлик. Смайлик появится в окне чата.

Другие способы начать чат МС

  • Щелкните правой кнопкой мыши имя приятеля и выберите пункт Новый чат.

  • Дважды щелкните контакт в списке приятелей, чтобы открыть окно чата.
