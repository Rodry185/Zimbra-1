Настройка параметров чата МС

Предусмотрен целый ряд параметров для настройки и улучшения функции чата @ClientNameShort@.

Изменение формата текста

Настройка параметров текста

  1. Щелкните Изменить формат. В окне чата отобразится панель инструментов форматирования.

  2. Выберите необходимые шрифт, размер шрифта, форматирование текста (полужирный шрифт, курсив, подчеркивание) и цвет текста.

  3. Щелкните Изменить формат еще раз, чтобы закрыть панель инструментов форматирования. При необходимости можно оставить ее открытой. Теперь отображение текста настроено согласно вашим требованиям.

Отправка записи чата по электронной почте

Запись чата можно отправить своим контактам по электронной почте с помощью окна мгновенных сообщений. Отправка чата по электронной почте

  1. Нажмите кнопку Отправить по почте. Откроется окно создания нового сообщения с записью чата в теле письма.

  2. Заполните необходимые поля. Также можно добавить текст в тело сообщения для указания необходимой информации.

  3. Нажмите кнопку Отправить, чтобы отправить запись чата контакту (контактам) в сообщении электронной почты.

Также можно переслать сохраненный чат из папки «Чаты» на панели навигации вкладки «Почта». Выберите запись чата, которую необходимо отправить по электронной почте, и нажмите кнопку Переслать.

Пометка чата как важного

Можно пометить чат как важный, щелкнув звездочку в окне мгновенных сообщений, чтобы она стала зеленой. Щелкните звездочку еще раз, чтобы снять пометку.

Закрепление окна

Закрепление окна мгновенных сообщений позволяет видеть окно чата, даже когда не открыта вкладка МС.

  • Для закрепления окна мгновенных сообщений щелкните значок булавки в его правом верхнем углу, после чего булавка должна стать цветной и горизонтальной.

Теперь можно открывать другие вкладки, а окно чата МС будет отображаться поверх интерфейса @ClientNameShort@.

  • Чтобы отменить закрепление окна, щелкните значок булавки еще раз независимо от того, какая вкладка открыта на данный момент. Окно мгновенных сообщений вернется на вкладку МС.
