Отправка файлов из портфеля

Во время работы в портфеле можно отправить ссылку на файлы, содержащиеся в портфеле, или отправить собственно файл в виде вложения в сообщение. Чтобы открыть файл, получатели должны иметь соответствующее приложение.

Отправка сообщения со ссылкой на файл

Можно отправить сообщение со ссылками на файлы, содержащиеся в портфеле.  Для работы с этими файлами получатели должны иметь соответствующее разрешение. Все вносимые в файл изменения отображаются в файле, который находится в папке вашего портфеля.

  1. Откройте портфель и щелкните правой кнопкой мыши те файлы, ссылки на которые вы хотите отправить.

  2. Выберите пункт Отправить ссылки. Отобразится сообщение для подтверждения. Нажмите кнопку Да.  Откроется страница создания сообщения с URL-адресом для каждой ссылки в теле письма.

  3. Добавьте адрес электронной почты получателя и завершите сообщение.

  1. Нажмите кнопку Отправить.

Отправить файл из портфеля

Элементы портфеля можно отправлять непосредственно из портфеля в виде вложений в сообщения.  

  1. Откройте портфель и щелкните правой кнопкой мыши файлы, которые нужно отправить. Выберите пункт Отправить вложения.
    Откроется страница создания сообщения с вложенным файлом.

  2. Добавьте адрес электронной почты получателя и завершите сообщение.

  3. Нажмите кнопку Отправить.
