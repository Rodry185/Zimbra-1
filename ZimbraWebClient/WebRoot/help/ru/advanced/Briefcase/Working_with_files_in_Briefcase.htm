Работа с файлами в портфеле

Открытие файлов в портфеле

Файлы в портфеле можно открыть с любого компьютера. Для открытия. чтения и обновления файла необходимо соответствующее приложение. Такие распространенные форматы файлов, как текстовый, HTML и изображения, например . gif или . jpg, можно открывать в различных программах.

Чтобы открыть файл, щелкните его правой кнопкой мыши и выберите пункт Открыть.

Примечание. Чтобы открыть файл, можно также дважды щелкнуть его.

  • Если тип файла неизвестен, откроется диалоговое окно с запросом действия: сохранить файл на диске или открыть в указанном приложении.

  • Если к портфелю предоставлен общий доступ, можно извлечь файл, чтобы его не могли редактировать другие пользователи.

    • Щелкните файл правой кнопкой мыши и выберите пункт Извлечь файл. Файл можно открыть или сохранить его на своем компьютере.

    • После внесения изменений загрузите файл. Если имя загружаемого файла совпадает с именем файла, который находится в папке портфеля, откроется диалоговое окно «Конфликт загрузки». Если выбрать «Да», файл будет загружен и создается новая версия файла. Если выбрать «Нет», файл не будет загружен. Следует изменить имя файла или загрузить файл в другую папку портфеля.

  • Если это документ Zimbra, дважды щелкните его, чтобы открыть в браузере. В этом виде невозможно изменить документ Zimbra.  

Изменение документа Zimbra

  • Щелкните правой кнопкой мыши документ Zimbra и выберите пункт Изменить. Файл откроется на новой странице браузера.

  • Внесите изменения на странице и нажмите кнопку Сохранить и закрыть, чтобы закрыть страницу. Если изменения сохранять не нужно, закройте вкладку браузера.

Примечание. При нажатии кнопки Сохранить или Сохранить и закрыть создается новая версия файла.

Перемещение файлов

Файлы можно перемещать из одного портфеля в другой.

  1. Выбор файлов для перемещения

  2. На панели инструментов щелкните значок .

  3. Выберите папку назначения или создайте новую папку в портфеле.

  4. Нажмите кнопку ОК.

Присвоение тегов файлам

Файлы можно помечать.

  1. Выделите файл и на панели инструментов щелкните значок .

  2. Выберите тег. Теги видны только на виде «Подробный».

  3. Нажмите кнопку ОК.
