Установка общих настроек

Страница Параметры > Общие содержит следующие изменяемые настройки.

  • При входе в учетную запись открывается расширенная версия веб-клиента, но здесь можно указать тип клиента по умолчанию.

    • Расширенная версия клиента предлагает полный набор функций для групповой работы в сети. Веб-клиент работает лучше всего с более новыми версиями браузеров и быстрым подключением к Интернету.

    • Стандартная версия клиента рекомендуется для медленных соединений с Интернетом, при использовании старых версий браузеров или для упрощенного доступа.  

  • Тема — это цвет фона для интерфейса почтового ящика. Цвет фона можно выбрать из списка на странице «Общие» в разделе «Темы».

  • Язык, используемый в веб-клиенте Zimbra, можно сменить на тот, который вам больше подходит.

  • Часовой пояс по умолчанию. Этот параметр определяет дату и время отправки и получения сообщений, а также время, отображаемое для встреч ежедневника.  Этот параметр может отличаться от настроек часового пояса на вашем компьютере. Если показан неверный часовой пояс, измените его.  

  • Если отображается параметр Изменить пароль, можно изменить пароль. Если при смене пароля возникают проблемы, обратитесь к администратору, чтобы узнать правила в отношении пароля. Если этот параметр отсутствует и нужно изменить пароль, обратитесь к системному администратору.

  • В меню Поиск можно указать, чтобы папки «Спам» и «Корзина» автоматически включались в каждый поиск. По умолчанию поиск в этих папках не выполняется.

  • В разделе Язык поиска установите флажок Всегда показывать язык расширенного поиска на панели инструментов поиска для отображения подробностей строки поиска в текстовом окне поиска на странице клиента.  

  • Прочее

    • Чтобы во время прокрутки страницы содержимого отображалось более 50 строк, измените Количество отображаемых элементов при прокручивании.

    • Установите флажок Отображать флажки для быстрого выбора элементов в списках (требуется обновление), чтобы отображать флажок для каждого элемента в области содержимого. Если этот параметр включен, используйте флажок для выбора одного или нескольких элементов, над которыми нужно выполнить одно и то же действие, например удалить, переместить, пометить как прочитанные или непрочитанные.

    • При печати сообщения по умолчанию используется шрифт размером 12 пунктов. Другой размер шрифта для печати можно выбрать из списка Размер шрифта печати.
