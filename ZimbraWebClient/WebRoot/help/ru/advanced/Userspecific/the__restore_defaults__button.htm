Восстановление настроек по умолчанию

На каждой вкладке Настройки, за исключением вкладки «Фильтры почты», кнопка Восстановить значения по умолчанию на каждой странице возвращает системные настройки по умолчанию для всех измененных параметров, действовавших при первом входе пользователя в новую учетную запись Zimbra. Чтобы подтвердить эти изменения, нужно нажать кнопку Сохранить.

  • Функция переустановки не действует на изменения пароля.
