Настройка параметров почты

Вкладка «Почта» содержит следующие изменяемые настройки.

  • Проверять новую почту каждые. Выберите интервал опроса. Это частота, с которой компьютер выполняет проверку новой почты. По умолчанию опрос выполняется каждые 5 минут. Для этого параметра не рекомендуется устанавливать значение меньше 5 минут, так как частая проверка почты ведет к перегрузке почтового сервера и снижению производительности. Если вы ожидаете почту, можно в любое время нажать кнопку Получить почту и сразу же ее получить. Если выбран параметр Никогда, то для получения новой почты требуется нажимать кнопку Получить почту на панели инструментов.

  • При нажатии кнопки «Получить почту». Этот параметр определяет, изменяется ли вид почтового ящика при нажатии кнопки «Получить почту».

    • Если установлен флажок Запустить поиск по стандартным критериям, то при нажатии кнопки Получить почту запускается настройка поиска почты, установленная в поле «Поиск в сообщениях по умолчанию».  Если по умолчанию поиск выполняется не в той папке, которая отображается после нажатия кнопки «Получить почту», то автоматически выполняется перенаправление в вид папки по умолчанию. Обычно это папка «Входящие».  

    • Если выбран параметр Обновить текущий вид, то при нажатии кнопки Получить почту в любом виде папки запрос на получение новой почты ищет обновления для текущего вида папки и новую почту для других папок. Текущий вид папок при этом не меняется.  При получении новой почты в другие папки имена таких папок выделяются полужирным шрифтом и обновляется количество непрочитанных сообщений.

  • Показать почту. Выберите пункт «В формате HTML (когда возможно)» для отображения сообщений, которые имеют формат HTML, например текста, выделенного полужирным, цветным и другими шрифтами и стилями. Выберите пункт В виде текста для отображения сообщений в формате обычного текста. В формате обычного текста теги HTML отображаются, но не применяются для разметки текста сообщения.

  • Параметры Просмотра сообщений

    • Показывать фрагменты сообщений. Если этот параметр включен, то при наведении курсора на тему сообщения отображается первая строка сообщения. Если этот параметр выключен, то отображаются только первые несколько слов сообщения.

    • Если включен параметр Дважды щелкните, чтобы открыть сообщение в новом окне, сообщение будет открываться по двойному щелчку в новом окне.

  • Включите параметр Изображения, чтобы загружать изображения автоматически в сообщения в формате HTML. Если этот параметр включен, щелкать изображение в сообщении для просмотра нет необходимости.

  • Параметр В режиме чтения сообщения в области чтения указывает порядок пометки сообщений, просматриваемых в области чтения, как прочитанных или непрочитанных. Сообщения могут быть помечены как прочитанные немедленно, спустя определенное количество секунд просмотра сообщения в области чтения или всегда оставаться непрочитанными при просмотре в области чтения.

  • Параметр После перемещения или удаления сообщения устанавливает порядок навигации в области содержимого при чтении сообщений.

  • Параметр Поиск в сообщениях по умолчанию определяет параметры поиска, выполняемого при входе. Результаты этого поиска отображаются в области содержимого при входе в почтовый ящик. По умолчанию поиск выполняется в папке Входящие. Т. е., при входе в почтовый ящик отображаются результаты поиска в папке «Входящие». Для отображения в этом поле при первом открытии веб-клиента Zimbra, можно выбрать другую папку, тег или сохраненный профиль поиска. Синтаксис см. в разделе «Описание языка запросов».

  • При поступлении сообщения. Можно указать режим уведомлений при поступлении сообщения: воспроизведение звука, мигание вкладки почты, мигание заголовка браузера. См. раздел Настройка уведомлений.

  • Переслать копию. Укажите электронный адрес для пересылки. Исходные сообщения могут удаляться из почтового ящика Zimbra.

  • Укажите, нужно ли отправлять автоответ на входящие сообщения, и укажите дату начала и дату окончания.  Эта функция часто используется в качестве сообщения об отсутствии. Она позволяет извещать отправителя о том, что вас нет на работе, и сообщать ему другие дополнительные сведения. Также можно указать дату начала и дату окончания для использования этого сообщения. См. раздел Сообщение об отсутствии.

  • Подтверждение прочтения. Укажите, как следует обрабатывать полученные сообщения с флагом, требующим отправки подтверждения прочтения отправителю.

  • Настройте раздел Сообщения от меня для обработки отправляемых вами сообщений, которые содержат ваш адрес в списке получателей или которые включены в адрес, например список рассылки, участником которого вы являетесь.

  • Параметры спама. Можно указать до 100 адресов, письма с которых нужно всегда блокировать и не добавлять в папку «Входящие», и до 100 адресов, письма с которых должны всегда добавляться в папку «Входящие» и не отправляться в папку «Спам».

  • Если используется клиент POP3, можно указать параметры  доступа по протоколу POP во время загрузки сообщений ZWC. Возможны следующие настройки доступа по протоколу POP.

    • Разрешить загрузку всей почты, в том числе старых писем

    • Разрешить загрузку почты только с настоящего момента.
