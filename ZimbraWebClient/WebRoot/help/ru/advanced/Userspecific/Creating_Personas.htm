Создание образов

Образ — это личные данные почты, используемые в поле От отправляемых сообщений. Создание различных личных данных почты позволяет использовать несколько адресов электронной почты в одном почтовом ящике. Это может понадобится для управления различными ролями на работе и в личной жизни с помощью одной учетной записи. Имя учетной записи по умолчанию определено в основных личных данных.

Кнопка Добавить внешнюю учетную запись позволяет определить внешние учетные записи POP/IMAP для получения почты в веб-клиенте с других учетных записей. Также можно установить, чтобы в ответах на эти сообщения использовался соответствующий адрес POP/IMAP.

Чтобы создать образ, выполните следующие действия.

  1. На странице Настройки, Учетные записи нажмите кнопку Добавить образ.

  2. В разделе Настройки образа введите имя образа. Это имя используется для идентификации образа в списке «От:» при создании сообщения.

  3. В текстовом поле От введите имя, которое должно быть указано в поле «От» исходящих сообщений, например Иван. Это описательное имя, котрое будет отображаться перед вашим электронным адресом. Во втором окне выберите электронный адрес, с которого будут отправляться сообщения. В зависимости от настроек может быть доступна возможность создания другого адреса. Сделайте это далее.

  4. Если ответы должны отправляться на имя и адрес, отличные от указанного в поле «От:», установите флажок Обратный адрес. Введите имя и адрес для получения ответов для этих личных данных Например, ответы на сообщения о тренировках, отправляемые Иваном, должны отправляться Тренировочной команде по адресу: training@xample.com.

  5. Можно выбрать подпись для этого образа. Если вы не создали подпись, вы можете вернуться позднее и отредактировать этот образ. Если вы не хотите использовать автоматическую подпись, выберите пункт Не включать подпись.

  6. Если нужно, чтобы этот образ автоматически использовался для сообщений в определенных папках, установите флажок Ответ или пересылка сообщений в папки. Щелкните значок папки, чтобы выбрать одну или несколько папок из списка существующих папок или создайте новую папку. Введите запятую или точку с запятой для разделения папок в поле.

  7. Нажмите кнопку Сохранить, чтобы сохранить образ.  

Если нужно создать подпись, откройте вкладку «Подпись» после нажатия кнопки «Сохранить». См. раздел Использование подписи.
