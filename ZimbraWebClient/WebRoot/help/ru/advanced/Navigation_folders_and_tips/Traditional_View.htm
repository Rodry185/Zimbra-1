Просмотр сообщений на виде «Сообщение»

На обычном виде «Сообщение» отдельно отображается каждое полученное сообщение

  • При наличии функции вида Разговоры по умолчанию включен вид разговоров. Чтобы сменить вид для данного сеанса, щелкните на панели задач стрелку, расположенную рядом с пунктом Вид. Чтобы сменить вид по умолчанию, откройте вкладку Настройки->Почтовый ящик

    .  

Смена видов с помощью меню «Вид»

  1. Нажмите кнопку Вид и выберите пункт По разговорам или По сообщениям.

  2. Выбор области чтения включает и выключает нижнюю область.
