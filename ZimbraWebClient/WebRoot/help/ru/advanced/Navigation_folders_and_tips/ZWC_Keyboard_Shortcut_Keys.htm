Горячие клавиши ZWC

Горячие клавиши используются для перемещения в веб-клиенте Zimbra (ZWC). Кроме того, пользователь может создать собственные комбинации горячих клавиш. Откройте вкладку Настройки>Горячие клавиши, чтобы просмотреть список горячих клавиш или создать собственные комбинации горячих клавиш. На четырех вкладках можно увидеть следующее.

  • Список горячих клавиш — список горячих клавиш для всех приложений.

  • Горячие клавиши папок почты — здесь можно создавать и назначать горячие клавиши для доступа к папкам почты.

  • Горячие клавиши для сохраненных запросов поиска — здесь можно создавать и назначать горячие клавиши для сохраненных запросов поиска.

  • Горячие клавиши для меток — здесь можно создавать и назначать клавиши для созданных вами меток.
