Настройка исходной страницы почты

По умолчанию почтовый ящик открывается на виде Входящие. Все сообщения в папке «Входящие» отображаются в области содержимого. Первоначальный вид можно поменять на любой другой — папки, профили поиска, теги — в области обзора. Для внесения этих изменений следует использовать Настройки > Почта > Отображение сообщений > Поиск в сообщениях по умолчанию.  

Ниже приведены примеры отображения определенных сообщений при открытии почтового ящика.

  • Чтобы при открытии почтового ящика отображались только непрочитанные сообщения, введите :непрочитанное в поле Поиск в сообщениях по умолчанию.

  • Чтобы отобразить сообщения с определенными тегами, введите «tag:< имя_тега> в поле Поиск в сообщениях по умолчанию. Например, чтобы отображались только сообщения с тегом «Задачи», введите «tag: Задачи».

  • Чтобы отобразить сообщения, отправленные в определенную папку при помощи фильтра почты, введите «in:< имя_папки> в поле Поиск в сообщениях по умолчанию. Например, чтобы отобразить сообщения в папке «Рабочая», введите in:рабочая.

Примечание.  Если тег или имя папки содержит более одного слова, необходимо использовать кавычки. Например, in:"Непрочитанные сообщения".

Настройка параметров получения почты для почтового ящика

В разделе Настройки > Почта > Отображение сообщений при помощи одного из параметров в области При нажатии кнопки «Получить почту» можно указать, будет ли меняться вид почтового ящика после нажатия кнопки «Получить почту».

  • Если выбран параметр Запустить поиск по стандартным критериям, то при нажатии кнопки Получить почту будет выполнен почтовый поиск, описанный в поле «Поиск в сообщениях по умолчанию».  Если поиск по умолчанию задан в папке, отличающейся от просматриваемой, вид будет автоматически изменен на вид папки по умолчанию. Обычно это папка «Входящие».  

  • Если выбран параметр Обновить текущий вид, то при нажатии кнопки Получить почту из папки при выполнении запроса «Получить почту» осуществляется поиск обновлений текущего вида папок и новых почтовых сообщений для других папок. Текущий вид папок при этом не меняется.

Примечание.  Если в другие папки поступила новая почта, имена этих папок выделяются жирным шрифтом, а также обновляется количество непрочитанных сообщений.
