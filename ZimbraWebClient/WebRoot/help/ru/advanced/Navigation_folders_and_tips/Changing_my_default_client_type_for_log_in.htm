Изменение типа клиента по умолчанию для входа

При входе в учетную запись можно выбрать, какой версией веб-клиента пользоваться: используемой по умолчанию, расширенной или стандартной.

  • Тип клиента По умолчанию настраивается на странице «Настройки > Общие». Там можно установить расширенный или стандартный клиент.

  • Расширенный веб-клиент предлагает полный набор функций групповой работы в сети (Ajax). Расширенная версия клиента работает с новейшими браузерами и быстрым Интернет-подключением.

  • Стандартный веб-клиент — хороший вариант для использования с медленным Интернет-подключением или на тот случай, когда пользователи предпочитают формат HTML в почтовых сообщениях.

Тип клиента по умолчанию можно сменить после входа в почтовый ящик.

1. Откройте вкладку Настройки.

2.  В области обзора выберите Общие и в поле Использовать для входа выберите Расширенный или Стандартный.

3. Нажмите кнопку Сохранить.

При следующем входе в почтовый ящик выбранный параметр будет принят по умолчанию.
