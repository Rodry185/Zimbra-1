Импорт ежедневников iCal

Пользователи могут импортировать ежедневники iCal, сохраненные в формате .ics.  Ежедневники iCal являются стандартным средством обмена данными ежедневников в Интернете, а формат .ics является стандартным форматом для данных ежедневников iCal.

Ежедневник iCal можно импортировать в существующий ежедневник, или можно создать новый ежедневник и импортировать в него файл .ics.

Как импортировать ежедневник iCal

  1. Перейдите на вкладку «Ежедневник».

  2. В области обзора ежедневника щелкните пункт Изменить.

  3. В области обзора ежедневника выберите ежедневник в списке или создайте новый ежедневник, в который будет импортирован файл .ics.

  4. Щелкните пункт Создать ежедневник. В поле Импорт из  .ICS укажите файл .ics для импорта.

  5. Нажмите кнопку Импорт.

  6. Нажмите кнопку Закрыть.
