Создание и сохранение запроса поиска

В случае частого выполнения какого-либо запроса поиска можно создать запрос и сохранить его в папке «Поиск». Для повторного выполнения поиска нужно просто щелкнуть папку с сохраненным запросом поиска. Это действие запускает поиск, результаты которого отображаются в области содержимого.

Как создать и сохранить запрос поиска

  1. В области просмотра>в заголовке «Поиск» нажмите кнопку Изменить.  В результате в области просмотра будут отображены все папки в алфавитном порядке, а папка «Входящие» отобразится в области содержимого.

  2. На панели инструментов выберите пункт Новый поиск. В области содержимого появится новая форма поиска.

  3. В поле Имя укажите имя для запроса поиска.

  4. В поле Родительская папка укажите, где нужно сохранить этот запрос поиска.  По умолчанию он сохраняется в корневой папке. Папка будет сохранена под заголовком «Поиск» в области обзора.  Если запрос поиска нужно сохранить в другой папке, выберите нужную папку в раскрывающемся списке. Папку поиска можно узнать по значку с изображением лупы.

  5. В поле Запрос поиска создайте запрос поиска. Сведения о синтаксисе языка запросов см. в разделе Описание языка запросов.

  6. В диалоговом окне Сохранить профиль поиска укажите имя и каталог для создания папки поиска.  Если сохранить папку в корневой папке, запрос поиска будет отображаться в разделе «Поиск». Если выбрать другую папку, запрос поиска сохраняется в выбранной папке.

  7. Нажмите кнопку ОК.

Для выполнения поиска щелкните папку поиска в области обзора. Результаты поиска будут показаны в области содержимого.
