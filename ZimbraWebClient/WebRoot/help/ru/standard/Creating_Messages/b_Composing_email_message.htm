Создание нового сообщения

  1. Для создания нового сообщения электронной почты нажмите кнопку Написать письмо на панели инструментов «Почта». Появится пустая область создания сообщения.  

  2. Введите адрес электронной почты пользователя или пользователей, которым должно быть отправлено сообщение, в поле Кому. Также можно ввести адреса в поле Копия или Скрытая копия.
    Для поиска адреса электронной почты на панели инструментов создания сообщения нажмите кнопку . В списках контактов можно осуществлять поиск конкретных имен. Выберите, в какое поле необходимо ввести имя – «Кому», «Копия» или «Скрытая копия».

  3. В поле Тема введите тему сообщения.

  4. Введите текст сообщения в текстовое поле, расположенное ниже поля «Тема». См. раздел Использование редактора форматированного текста.

При наличии настроенной подписи она будет автоматически добавлена в сообщение.

  1. Чтобы добавить вложение, щелкните Добавить вложения.

  2. Если необходимо пометить сообщение как имеющее высокий приоритет, измените выбранный приоритет.  Это можно сделать в правой части строки темы.

  3. Чтобы отправить сообщение, нажмите кнопку Отправить. Если включена функция Сохранять копии сообщений в папке Отправленные, сообщение будет сохранено в папке «Отправленные».

Если отправлять сообщение в данный момент не нужно, нажмите кнопку Сохранить черновик. Сообщение будет сохранено в папке «Черновики».
