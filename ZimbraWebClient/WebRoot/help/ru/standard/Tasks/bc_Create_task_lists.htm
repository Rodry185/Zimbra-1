Создание списка задач

  1. Откройте вкладку Задачи.

  2. В заголовке «Задача» нажмите кнопку Изменить.

  3. Выберите пункт Новый список задач.

  4. В поле Новый список задач введите имя списка задач и выберите цвет для отображения задачи.

  5. Выберите пункт Создать список задач. Список добавлен к задачам в области просмотра.

  6. Нажмите кнопку Закрыть.
