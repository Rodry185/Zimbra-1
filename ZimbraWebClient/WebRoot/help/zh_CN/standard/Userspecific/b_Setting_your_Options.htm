设置首选项

默认的用户首选项是在帐户创建时配置的，并确定了邮件、地址簿和书写等功能的使用方式。

您可在“首选项”选项卡中更改这些首选项设置。如果此处描述的首选项在您的视图中看不到，则说明该首选项对于您的帐户不适用。

常规

  • 您登入的是标准网络客户端。要将此设置为登录时使用的默认客户端类型，请选择标准。

  • 如果 Internet 连接速度较慢、所用浏览器为早期版本或为了访问更简便，可采用标准。

  • 高级网络客户端具有全套的网上协同办公功能。此种网络客户端最适合于较新版的浏览器和更快的互联网连接。

  • 在主题处选择邮箱的背景色。

  • 选择时区后可设置日期和时间，同时也可调整行事历的约会时间。这可能与您计算机上配置的时区不同。

  • 如果显示变更密码，您可修改您的密码。如果在变更密码时遇到问题，请向您的管理员询问密码规则。

  • 在搜索文件夹中，您可选择在所执行的任何搜索中自动包括“垃圾邮件”和“已删除邮件”文件夹。默认情况下，这两个文件夹没有包括在内。

  • 在搜索语言中，勾选总显示搜索字符串以便在搜索字框中显示搜索字符串的细节。

邮件

  • 选择每页显示多少条目（邮件或会话）。默认设定值为每页显示 25 个条目，但您可选择 10、50 或 100。

  • 选择邮件如何分组，按会话或按邮件。如果有“会话”功能，则“会话”是默认的。

  • 对于已经格式化为 HTML 的邮件，可指定是否以 HTML 方式查看。若关闭此项，邮件则以纯文本形式显示，即只显示 HTML 标记，但不显示标记语言版本。

  • 阅读窗格可用于预览您收件箱和其他文件夹中的邮件。启用此窗格后，邮件显示在预览窗格中。阅读窗格可以显示在邮件列表的下方，也可以显示在三窗格视图的右侧。

  • 若启用了邮件预览，则邮件文本开头的文字会显示在“主题”之后。

  • 在默认邮件搜索字段中定义的搜索可确定登录时应执行的搜索目标。默认的搜索目标为您的收件箱。当您登录时，显示的是您收件箱中的结果。您可修改此字段，以使在刚打开网络客户端时显示的是其他的文件夹、标记或保存的搜索。有关句法，请见查询语言说明。

  • 启用并指定一个邮件转发地址。勾选此复选框后，原始邮件将从您的邮箱中删除。

  • 启用并指定一个地址，以便系统发送关于收到新邮件的通知。

  • 启用发送自动回复讯息，以便在收到邮件时自动回复一则您不在办公室的讯息。在文本框中键入此邮件的文本，然后设置使用邮件的起止日期。

  • 配置我发的邮件。用以处理以下两种情况：您发送的邮件，但您自己的地址是邮件的收件人之一；或所前往的地址为包括您自己在内的邮件列表。

  • 如果您设置使用一个 POP3 客户端，您可配置在邮件下载到 ZWC 后 POP 的访问行为。

写信

  • 在写信中：选择是使用 HTML 格式还是以纯文本形式编写邮件。使用 HTML 格式编写邮件时，您可以使用文本格式选项，包括字体、字号和颜色。

  • 在回复/全部回复中，选择是否在您的回复邮件正文中包含原始邮件文本。

    • 直接插入完整文本选项可使邮件正文中包括原始邮件，但不含特殊的标记。

    • 前缀后直接插入完整文本选项可使原始邮件的每一行加上一个前缀。前缀字符可以是 > 或 |。

    • 附件选项可使原始邮件作为附件附加到您的回复中。

    • 不包含选项可使回复中不包含原始邮件。

  • 在转发部分，您可选择在转发邮件时如何包含原始邮件文本，即插入邮件正文，插入带前缀的邮件正文，或作为附件附加在邮件上。

  • 前缀字符可用来设置回复邮件或转发邮件中使用的前缀。

  • 已发送邮件，选择保存已发送邮件的副本以将您发送的每封邮件的副本保存到您的“已发送邮件”文件夹中。

签名

  • 可为您的邮件创建各种签名。签名中可包括姓名和其他结束语句，帐户设定中对语句的长短和字符数量上限均有限制。如果您有多个身份，则可创建不同的签名，并分别指派给具体的地址。

  • 选择签名的位置。这是回复邮件的签名位置。您希望签名出现在回复邮件的上方还是邮件的末尾？

地址簿

  • 选择每页显示的联系人的人数。默认值为 25。

  • 启用此功能后，您可在发送邮件时自动将地址添加到地址簿。如果启用了此选项，地址会保存到“电子邮件联系人地址簿”中。

  • 管理地址簿链接可将您转到“地址簿编辑联系人”页面，在该页面中，您可以创建新的地址簿并链接到共享地址簿、导出和导入联系人列表，还可以删除地址簿中的联系人。

行事历

  • 选择如何查看行事历。默认值为按照工作周。

  • 指定在行事历中把哪一天作为一周的第一天。

  • 在天/周视图部分，可设置日历天的开始时间和结束时间。

  • 如果经常往返于不同的时区，应该选择在约会视图中显示时区列表。

  • 管理行事历链接用于打开行事历编辑页面，您可在这里编辑现有的行事历、添加新的行事历、订阅行事历或链接到一个共享行事历。

帐户

当发送新邮件、回复或转发邮件时，邮件总会与一个身份关联。这是作为帐户名称创建的默认身份。您可更改您的电子邮件“收件人”字段中显示的昵称，还可选择要在您的电子邮件中自动添加的签名。

邮件过滤器

过滤功能可将一系列的匹配规则应用于发来的邮件并执行特定的动作。使用此选项卡定义邮件过滤规则，详细说明请参见过滤您的邮件。
