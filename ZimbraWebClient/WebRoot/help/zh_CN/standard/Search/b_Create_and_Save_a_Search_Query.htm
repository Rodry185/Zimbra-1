创建并保存搜索查询

如果您频繁地执行同样的搜索条件，可创建该搜索条件并将其保存到“搜索”文件夹。当需要再次执行该搜索时，只需单击已保存的搜索文件夹即可启动搜索，并在内容窗格中显示结果。

创建并保存搜索查询

  1. 在“概述窗格 > 搜索”标题中，单击编辑。 概述窗格被刷新，按字母顺序显示所有文件夹，且内容窗格显示收件箱。

  2. 在工具栏中，选择新搜索条件。一个新搜索条件窗口即显示在内容窗格中。

  3. 在名称字段中，输入搜索的名称。

  4. 在父文件夹字段中，选择将此搜索保存到何处。 默认位置为根文件夹。文件夹将保存在“概述”窗格的“搜索”标题下。 如果要将搜索保存到另一个文件夹，请从下拉列表中选择该文件夹。可用放大镜图标标识搜索文件夹。

  5. 在搜索条件字段中，创建您的搜索条件。请参见查询语言说明以获得您可使用的搜索语法的信息。

  6. 在保存搜索对话框中，为保存搜索结果的文件夹命名，并选择在何处创建搜索文件夹。 如果保存在根文件夹中，此搜索将显示在“搜索”区。如果选择其他文件夹，此搜索即被保存到该文件夹。

  7. 单击确定。

运行搜索时，单击概述窗格中的搜索文件夹。搜索结果将在内容窗格中显示。
