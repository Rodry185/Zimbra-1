创建新的行事历

只为您创建了一个行事历，您可以创建更多的行事历以便跟踪不同类型的活动。 比如，您可以为工作、个人、公司的各项活动创建分开的行事历，也可订用互联网上的行事历，还可以链接到共享的行事历。

可使用日程安排视图 对照查看两个或更多行事历。

若需创建行事历

  1. 选择“行事历”标签。

  2. 在行事历概述窗格中，单击编辑。

  3. 在工具栏中，单击新建行事历。

  4. 键入行事历的名称。

  5. 选择一种显示行事历活动的颜色。此操作可使您为邮箱中的行事历设定颜色，从而使行事历的活动总是以选定的颜色显示。

  6. 如果不希望其他人看到行事历中的活动，请 勾选报告空闲/忙碌时间时排除此行事历。

  7. 默认情况下，已在 UI 中勾选处于启用状态。打开“行事历”视图时，将显示此行事历中的活动。 如果不希望随同别的行事历一起查看此行事历，请取消勾选此框。在这种情况下，要查看此行事历的活动，须选择该行事历并将其打开。

  8. 单击创建行事历。

若需订阅其他行事历

  1. 选择行事历标签。

  2. 在行事历概述窗格中，单击编辑。默认的行事历信息随即显示。  

  3. 在工具栏中，单击订阅行事历。  

  4. 键入该行事历的名称，输入该远程行事历的有效 iCal 文件 URL，选择一种显示该行事历中的活动的颜色。

  5. 如果不希望其他人看到行事历中的活动，请 勾选报告空闲/忙碌时间时排除此行事历。

  6. 默认情况下，已在 UI 中勾选处于启用状态。打开“行事历”视图时，将显示此行事历中的活动。 如果不希望随同别的行事历一起查看此行事历，请取消勾选此框。在这种情况下，要查看此行事历的活动，须选择该行事历并将其打开。

  7. 单击创建行事历。
