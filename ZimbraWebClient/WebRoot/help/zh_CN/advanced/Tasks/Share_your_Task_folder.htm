共享您的任务列表

您可与他人共享您的任务列表，还可为共享列表定义以下各种权限：

  • 内部用户或组。您可从以下两种权限类型中选择要授予的权限：

    • 浏览者。“共享者”可以看到任务列表的内容，但是不能对任务列表的内容进行任何修改。

    • 管理者。此类共享者有权创建任务、编辑任务和从任务列表中删除任务。

    • 管理员。此类共享者有权创建任务、编辑任务、从任务列表中删除任务以及与其他人共享您的任务列表。

    • 无 选项用于临时禁止访问授权者的任务列表但不撤销共享权限。“共享者”的任务列表视图中仍有该列表，但不能查看或管理。

  • 外来客。您须创建一个用于访问共享任务列表的密码。来客必须输入该密码才能查看任务列表。来客不能对任务进行做任何改变。

  • 公开。任何知道此共享任务列表 URL 的人都可以查看其内容。但不能做任何改动。

若需与人共享您的“任务”列表

右键单击要共享的任务列表，然后选择“共享任务列表”。

在“共享属性”对话框中，输入共享人人选并设定适当的信息。

 

与内部用户或组共享1. 选择内部用户或组作为共享者

2. 在电子邮件字段中键入共享者的电子邮件地址。在键入地址时，与您所键入字母匹配的姓名会自动列出。可在该列表中选择姓名。

3. 选择角色（浏览者或管理者）

4. 在邮件部分，选择要发送给共享者的通知电子邮件的类型。

5. 单击确定。

与外来客共享

1. 在共享用户部分，选择外来客。选择此项后，屏幕会出现“密码”字段。

2. 输入该外来客人的邮件地址。

3. 输入密码，该外来客人今后便可将用该密码访问此任务列表。

4. 选择要发送给他们的通知邮件的类型。此邮件将提供他们访问此列表所需的信息。

5. 单击确定。

 

与公众共享

1. 在共享目标部分，选择公众。选取后，共享属性对话框随即显示您的任务列表的 URL。

2. 注意此 URL，并将此电子邮件地址告知“公众”用户。他们单击此 URL 时，便可阅读此任务列表的内容，但是不能做任何修改。

编辑或撤销权限

您可在任何时候撤销权限。

  1. 右键单击一个共享任务列表。

  2. 单击编辑属性。共享该文件夹的用户随即列出。

  3. 若需改变权限，可单击编辑；若需移除权限，
    可单击撤销。

  4. 选择是否发送一条描述这一变更的消息。

  5. 单击确定。
