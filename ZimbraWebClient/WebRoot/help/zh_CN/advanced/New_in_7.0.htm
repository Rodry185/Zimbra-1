New @ClientNameShort@ Features in 7.0

The following are major new enchantments for 7.0 Advanced @ClientNameLong@.

Email

  • Users can schedule a message to be sent at a future date and time.


 

  • Distribution lists can have an "expand" link which lets you see who is in the DL. You can then select individual addresses from the list.

 
 
 

  • You can recover items deleted from your email Trash folder within the last 30 days. Right-click on the Trash folder and select Recover Deleted Items.

 
 

  • Mail and contact actions can be undone using the "undo" dialog that displays at the top of the screen when an action is performed.

 		  

 

  • An expanded color palette allows you to create custom colors for tags and folders

 		  

 

 

  • The Add Attachments dialog includes tabs to add email messages, contacts, and briefcase files.

 

Address Book

  • Forward contact information as vCards in an email message.

 

Calendar

  • When creating a new appointment all the information displays on one page:

    1. As you enter attendees, the scheduling wizard suggest rooms and times based on attendees availability. This information is displayed in the Overview pane.

    2. You can view the free/busy Scheduler.

    3. Select Send Email to have a reminder message sent to an email address you configure in the Notification preferences page.

  • In Calendar Preference, work week and hours can be set. You can customize your work schedule for each day of the week as well.

  • Copy and paste an existing meeting to easily create an appointment similar to an existing appointment.

  • Select a date range of appointments to display in the List view.

 

Tasks

  • Reminder date and time can be configured for a task and you can select to have the reminder email message sent to an email address you configure in the Notification preferences page.

  • View a Task list with preview pane enabled either on the right or at the bottom of the page.

  • Tasks open as a tab, similar to Compose tab in Mail. You can switch between a tabs.

 

Briefcase

  • Version control to manage changes to files in Briefcase.

  • Document check-in/check-out to control changes to the source.

  • View a Briefcase folder with preview pane enabled either on the right or at the bottom of the page.

 

Calendar, Tasks and Briefcase each have a Trash folder.  Only items specific to the selected application are displayed in an application's Trash folder when it is opened.

Preferences

  • Configure different new-email signatures and reply to/forward-email signatures and automatically include your contact information as a vCard attachment.

  • Configure the font size for messages that are printed from your Mail Preferences page. The default si 12 points.

  • Notifications page shows the email address and SMS notification number where reminder messages about tasks and calendar appointments are sent.

  • Filters can be created on outgoing messages.

  • Create a list of trusted addresses or domain so that images sent in email messages from those addresses are automatically displayed in your message. Trusted Addresses page shows the trusted sites list of addresses. Addresses can be added or deleted on this page. You can also add an email address to the Trusted Address page  from the email message.
