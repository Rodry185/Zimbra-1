Link zu freigegebenem Kalender einrichten

Wenn Sie per Mail informiert werden, dass Ihnen der Zugriff auf den Kalender einer anderen Person gewährt wird, müssen Sie einen Link zu dem freigegebenen Kalender einrichten, damit dieser in Ihrer Kalenderliste angezeigt wird. Sie benötigen hierfür die vollständige Mail-Adresse des Kalenderbesitzers und den genauen Namen des freigegebenen Kalenders.

So richten Sie einen Link zu einem freigegebenen Kalender ein

  1. Klicken Sie in der Registerkarte "Kalender" auf Bearbeiten.

  2. Wählen Sie in der Symbolleiste die Option Link zu freigegebenem Kalender.

  3. Im Fenster "Neuer Kalender" geben Sie dem freigegebenen Kalender den Namen, der in Ihrer Liste angezeigt werden soll.

  4. Um auf den Kalender zugreifen zu können, geben Sie die Mail-Adresse und den Kalendernamen des Besitzers ein.

  5. Wählen Sie für diesen freigegebenen Kalender eine Farbe aus.

  6. Klicken Sie auf Kalender erstellen.
