Kontaktliste exportieren

Sie können Kontaktlisten exportieren. Diese werden im Format *.csv gespeichert.

So exportieren Sie eine Kontaktliste:

  1. Wählen Sie die Registerkarte "Adressbuch".

  2. Klicken Sie im Übersichtsfenster des Adressbuchs auf Bearbeiten.

  3. Wählen Sie das Adressbuch, das exportiert werden soll.

  4. Klicken Sie auf den Link im Feld In .CSV-Datei exportieren. Das Dialogfeld <Adressbuchname> öffnen wird angezeigt.

  5. Wählen Sie Auf Datenträger speichern und bestätigen Sie mit OK.

  6. Geben Sie den Speicherort für die Datei an.

  7. Klicken Sie auf Speichern.
