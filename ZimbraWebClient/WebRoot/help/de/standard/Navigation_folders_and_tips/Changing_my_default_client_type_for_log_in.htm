Voreingestellten Client-Typ für die Anmeldung ändern

Wenn Sie sich anmelden, können Sie wählen, welche Version des Web-Clients Sie nutzen möchten: "Voreingestellt", "Erweitert" oder "Standard".

  • Voreingestellt ist der auf der Seite "Einstellungen>Allgemein" konfigurierte Client-Typ. Dies kann die erweiterte Version oder die Standardversion sein.

  • Der erweiterte Web-Client bietet den vollständigen Satz Ajax-basierter Webkommunikationsfunktionen. Der erweiterte Web-Client funktioniert mit neueren Browsern und schnelleren Internetverbindungen.

  • Der Standard-Web-Client ist eine gute Wahl, wenn Sie eine langsame Internetverbindung haben oder HTML-basiertes Messaging für die Navigation in der Mailbox bevorzugen.

Wenn der von Ihnen bevorzugte Client-Typ nicht als Standard eingerichtet ist, können Sie die Voreinstellung nach dem Anmelden ändern.

1. Klicken Sie auf die Registerkarte Einstellungen.

2.  Klicken Sie im Übersichtsfenster auf Allgemein und wählen Sie für das Feld Anmeldeoptionen entweder Erweitert oder Standard aus.

3. Klicken Sie auf Speichern.

Bei der nächsten Anmeldung wird der ausgewählte Typ als Voreinstellung verwendet.
