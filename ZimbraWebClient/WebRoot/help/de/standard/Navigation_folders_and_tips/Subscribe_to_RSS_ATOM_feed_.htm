RSS/ATOM-Feeds abonnieren

Sie können Websites abonnieren, die von Ihnen bevorzugte RSS-Feeds (Really Simple Syndication) und Podcast-Feeds bieten und aktuelle Informationen direkt an Ihre Mailbox senden.

Für jeden RSS-Feed legen Sie einen Ordner an.

Bevor Sie fortfahren, benötigen Sie die URL für den RSS-Feed.

  1. Klicken Sie im Übersichtsfenster unter "Ordner" auf Bearbeiten.

  2. Klicken Sie in der Symbolleiste auf Neuer RSS-/ATOM-Feed.

  3. Geben Sie einen Namen für den Ordner ein, wählen Sie den Speicherort des Ordners aus und geben Sie die URL für den  RSS-Feed der Website in das URL-Feld ein.

  4. Klicken Sie auf OK.

Der Ordner wird im Übersichtsfenster angelegt. Öffnen Sie den Ordner, um den Link anzuzeigen. Um den Inhalt zu aktualisieren, klicken Sie mit der rechten Maustaste und wählen Sie im Ordner Neu laden.
