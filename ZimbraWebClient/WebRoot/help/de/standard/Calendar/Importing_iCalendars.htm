iCalendars importieren

Sie können iCalendars-Kalender (iCal), die im Format .ics gespeichert wurden, importieren. iCalendar ist das Standardformat für den Austausch von Kalenderdaten über das Internet, das Dateiformat lautet .ics.

Sie können Ihren iCal-Kalender in einen vorhandenen Kalender importieren oder einen neuen Kalender erstellen und die .ics-Datei in diese Datei importieren.

So importieren Sie einen iCal-Kalender

  1. Öffnen Sie die Registerkarte "Kalender".

  2. Klicken Sie in der Kalenderübersicht auf Bearbeiten.

  3. Wählen Sie in der Übersicht des Kalenders einen Kalender in der Liste aus oder legen Sie einen neuen Kalender an, um die .ics-Datei zu importieren.

  4. Klicken Sie auf Kalender erstellen. Navigieren Sie im Feld Aus  .ICS importieren zur .ics-Datei, um diese zu importieren.

  5. Klicken Sie auf Importieren.

  6. Klicken Sie auf Schließen.
