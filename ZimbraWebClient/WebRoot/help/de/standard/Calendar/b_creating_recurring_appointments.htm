Wiederkehrende Besprechungen anlegen

Wiederkehrende Besprechungen sind Termine, die mit gewisser Regelmäßigkeit stattfinden. Sie können wiederkehrende Besprechungen für die tägliche, wöchentliche, monatliche oder jährliche Wiederholung planen. Außerdem können Sie wiederkehrende Termine benutzerdefiniert planen.

  1. Wenn Sie einen neuen Termin anlegen, klicken Sie in der Symbolleiste auf Wiederholung, um die Seite zum Einrichten wiederkehrender Termine aufzurufen.

  2. Klicken Sie unter EinfacheWiederholung auf das Optionsfeld, um auszuwählen, wie oft die Besprechung wiederholt werden soll - jeden Tag, jede Woche, jeden Monat oder jedes Jahr.

  3. Wenn es für die wiederkehrende Besprechung ein Enddatum gibt, wählen Sie unter Ende im Abschnitt Erweitert aus, ob die Besprechung nach einer bestimmten Anzahl von Besprechungen oder zu einem bestimmten Datum enden soll.

  4. Im Abschnitt "Erweitert" können Sie benutzerspezifische Wiederholungsmuster planen.

    • Täglich: Für tägliche Wiederholungen können Sie angeben, ob die Besprechung jeden Tag oder in einer anderen Folge, z. B. jeden zweiten oder vierten Tag, stattfinden soll.

    • Wöchentlich: Für wöchentliche Wiederholungen können Sie angeben, ob die Besprechung jede Woche oder in einem anderen Muster, z. B. jede zweite Woche, stattfinden soll.  Sie können auswählen, an welchem Wochentag die Besprechung stattfinden soll.

    • Monatlich: Für monatliche Wiederholungen können Sie angeben, ob der Termin jeden Monat oder in einem anderen Muster, z. B. jeden zweiten Monat, stattfinden soll.  Es wird ein Kalender angezeigt, damit Sie auswählen, an welchem Tag bzw. an welchen Tagen die Besprechung stattfinden soll.

    • Jährlich: Für jährliche Wiederholungen wählen Sie den Monat und das Datum aus.

  1. Klicken Sie auf Fertig, um zur Terminseite zurückzukehren.
