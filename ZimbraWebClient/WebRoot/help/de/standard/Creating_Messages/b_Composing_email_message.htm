Neue Mail verfassen

  1. Um eine neue Mail in der Mail-Symbolleiste zu verfassen, klicken Sie auf die Registerkarte Verfassen. Es wird ein leeres Fenster zum Verfassen einer Mail angezeigt.  

  2. Geben Sie in das Feld An: die Mail-Adressen der Personen ein, an die Sie die Mail senden möchten . Sie können auch in die Felder CC oder BCC Adressen eingeben.
    Um die Mail-Adresse einer Person nachzuschlagen, klicken Sie in der Symbolleiste „Verfassen“ auf . Sie können in den Adressbüchern nach bestimmten Namen suchen. Achten Sie darauf, ob der Name unter "An", "CC" oder "BCC" eingefügt werden soll.

  3. Geben Sie den Betreff der Mail in die Zeile Betreff ein.

  4. Geben Sie den Text der Mail in das Textfeld unterhalb der Betreffzeile ein. Siehe Im Rich-Text-Editor arbeiten

Wenn Sie eine Signatur eingerichtet haben, wird Ihre Signatur automatisch in Ihre Mail eingefügt.

  1. Um einen Anhang hinzuzufügen, klicken Sie auf Anhänge hinzufügen.

  2. Wenn die Mail eine hohe Priorität erhalten und entsprechend gekennzeichnet werden soll, ändern Sie die ausgewählte Priorität.  Diese ist rechts von der Betreffzeile zu finden.

  3. Klicken Sie auf Senden, um die Mail zu senden. Wenn Sie Kopien der Mails im Ordner "Gesendet" speichern aktiviert haben, wird eine Kopie Ihrer Mail in Ihrem Ordner "Gesendet" abgelegt.

Wenn Sie die Mail nicht sofort senden möchten, klicken Sie auf Entwurf speichern. Die Mail wird dann im Ordner "Entwürfe" gespeichert.
