IM-Chat-Optionen zur individuellen Gestaltung Ihres Chats

Ihnen stehen zahlreiche Optionen zur individuellen Gestaltung und Verbesserung Ihres Chats in @ClientNameShort@ zur Verfügung.

Textformat ändern

So ändern Sie Ihre Textoptionen:

  1. Klicken Sie auf Format ändern. Eine Formatierungssymbolleiste wird im Chat-Fenster angezeigt.

  2. Wählen Sie die Schriftart und -größe, den Textstil (fett gedruckt, kursiv oder mit Unterstrich) sowie die Textfarbe entsprechend Ihren Wünschen aus.

  3. Klicken Sie erneut auf Format ändern, um die Formatierungssymbolleiste zu schließen. Sie können sie ggf. auch geöffnet lassen. Der Text wird nun entsprechend Ihren Anforderungen angezeigt.

Chat-Transkript per Mail senden

Sie können ein Chat-Transkript über das IM-Fenster per Mail an Ihre Kontakte senden. So senden Sie einen Chat per Mail:

  1. Klicken Sie auf Per Mail senden. Es wird ein neues Fenster zum Verfassen einer Mail geöffnet, wobei das Chat-Transkript im Textfeld der Mail angezeigt wird.

  2. Füllen Sie alle notwendigen Felder aus. Sie können auch Text in das Textfeld der Mail eingeben, um auf einen Punkt zu verweisen.

  3. Klicken Sie auf Senden, um das Chat-Transkript per Mail an den oder die Kontakt(e) zu senden.

Alternativ können Sie einen gespeicherten Chat aus dem Ordner „Chats“ im Navigationsfenster unter der Registerkarte „Mail“ weiterleiten. Wählen Sie das Chat-Transkript, das Sie per Mail versenden wollen, und klicken Sie zum Senden der Mail auf Weiterleiten.

Chat als „Wichtig“ markieren

Sie können einen Chat als „Wichtig“ markieren, indem Sie auf den Stern im IM-Fenster klicken. Der Stern wird grün. Klicken Sie erneut auf den Stern, um die Markierung zu entfernen.

Fenster immer im Vordergrund anzeigen

Wird ein IM-Fenster immer im Vordergrund angezeigt, können Sie das Chat-Fenster auch dann sehen, wenn Sie sich außerhalb der IM-Registerkarte befinden.

  • Um ein Fenster immer im Vordergrund anzuzeigen, klicken Sie oben rechts im IM-Fenster auf das Symbol der Pinnwandnadel, sodass sie farbig und horizontal dargestellt wird.

Sie können nun auf andere Registerkarten zugreifen und dennoch das IM-Chat-Fenster im Vordergrund auf der @ClientNameShort@-Benutzeroberfläche sehen.

  • Um das Fenster nicht länger im Vordergrund anzuzeigen, klicken Sie erneut auf das Symbol der Pinnwandnadel. Dabei ist es egal, auf welcher Registerkarte Sie sich gerade befinden. Das IM-Fenster wird anschließend wieder nur auf der IM-Registerkarte angezeigt.
