Auf den Instant Messenger zugreifen

Sie können über die IM-Leiste am unteren Rand des Inhaltsfensters @ClientNameShort@ auf den Instant Messenger zugreifen. Zum Starten einer Instant Messaging (IM)-Sitzung klicken Sie auf die Favoritenliste und wählen Sie einen Freund aus.

Bestätigen Sie Ihren Status in der IM-Leiste, um sicherzustellen, dass Sie für andere als „Online“ angezeigt werden. Wenn für den Status angezeigt wird, können Sie beginnen, den IM zu verwenden. Bei Anzeige eines anderen Status klicken Sie auf das Symbol, um das Menü zu öffnen und Ihren Status in „Online“ oder „Verfügbar für Chat“ zu ändern.

Nun können Sie damit beginnen, Freunde hinzuzufügen.

Freunde hinzufügen

Um den Instant Messenger zu verwenden, müssen Sie mindestens einen Freund zu Ihrer Favoritenliste hinzufügen. Freunde können über Ihr Adressbuch hinzugefügt werden oder Sie können sie manuell eingeben.

So fügen Sie einen Freund hinzu:

  1. Klicken Sie in der IM-Leiste auf . Das Dialogfeld Neuen Freund erstellen wird angezeigt.

  2. Geben Sie die Mail-Adresse des Kontaktes ein, den Sie als Freund hinzufügen wollen.

  3. Geben Sie den Namen des Freundes so ein, wie er in Ihrer Liste angezeigt werden soll. Wenn Sie keinen Namen für Ihren Freund eingeben, wird die Mail-Adresse des Kontakts als Name in der Liste aufgeführt.

Der Freund wird automatisch benachrichtigt und hat so die Möglichkeit, Ihre Anfrage zum Hinzufügen zu akzeptieren oder abzulehnen.

  1. Falls Sie Ihre Freunde in Gruppen einteilen möchten, wählen Sie eine Gruppe für Ihren neuen Freund aus. Sie können auch eine Gruppe über den Dropdown-Pfeil auswählen. Die Standardgruppe ist Favoriten. Um eine neue Gruppe zuzuordnen, geben Sie den Namen der neuen Gruppe ein und klicken Sie auf OK.

 

  • Um Ihre IM- und Anmeldeeinstellungen zu ändern, gehen Sie zum Ordner Einstellungen > Instant Messaging.
