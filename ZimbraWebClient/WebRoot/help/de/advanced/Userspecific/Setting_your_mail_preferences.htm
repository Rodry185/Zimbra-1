Maileinstellungen festlegen

Unter "Mail" können Sie die folgenden Einstellungen ändern.

  • Abfrage auf neue Mails alle x. Wählen Sie das Abrufintervall aus, das angibt, wie häufig Ihr Computer neue Mails abruft. Standardmäßig wird alle 5 Minuten auf neue Mails geprüft. Es ist nicht empfehlenswert, diesen Zeitraum auf weniger als 5 Minuten einzustellen, weil der Mailserver durch eine häufige Mailabfrage stark ausgelastet wird und dies die Leistungsfähigkeit herabsetzt. Wenn Sie eine Mail erwarten, können Sie jederzeit auf Mails empfangen klicken, um neue Mails sofort zu empfangen. Wenn Sie Nie einstellen, müssen Sie neue Mails über die Schaltfläche Mails empfangen in der Symbolleiste abrufen.

  • Wenn ich auf „Mails empfangen“ klicke. Dadurch wird festgelegt, ob sich Ihre Mailbox-Ansicht ändert, wenn Sie auf „Mails empfangen“ klicken.

    • Wenn Standardsuche durchführen ausgewählt ist, wenn Sie auf Mails empfangen klicken, wird die im Feld „Standard-Mail-Suche“ festgelegte Einstellung für die Mail-Suche durchgeführt. Wenn Ihre Standard-Suche nicht der Ordner ist, den Sie sehen, nachdem Sie auf „Mails empfangen“ geklickt haben, wird Ihre Ansicht automatisch zur Standard-Ordneransicht umgeleitet. In den meisten Fällen ist dies Ihr Posteingang.  

    • Wenn Aktuelle Ansicht aktualisieren ausgewählt ist, wenn Sie in einer beliebigen Ordneransicht auf Mails empfangen klicken, sucht die Anfrage auf Abruf neuer Mails nach Aktualisierungen an Ihrer aktuellen Ordneransicht sowie nach neuen Mails in anderen Ordnern.Ihre aktuelle Ordneransicht ändert sich nicht.  Wenn neue Mails in anderen Ordnern empfangen werden, werden diese Ordner fett dargestellt und die Anzahl an ungelesenen Mails wird aktualisiert.

  • Mails anzeigen. Wählen Sie "Als HTML (wenn möglich)", um die Mails im HTML-Format anzuzeigen, z. B. mit fett, farbig oder anderweitig formatiertem Text. Wählen Sie Als Plain-Text, um die Mails im reinen Textformat anzuzeigen. Im Plain-Text-Format werden HTML-Tags angezeigt und nicht als Formatierung umgesetzt.

  • Optionen für die Mail-Vorschau

    • Mailauszüge in Mailliste anzeigen. Ist diese Option eingeschaltet, wird die erste Zeile der Mail angezeigt, wenn Sie den Mauszeiger auf dem Betreff einer Mail positionieren. Ist die Option ausgeschaltet, werden nur die ersten Wörter der Mail angezeigt.

    • Ist Durch Doppelklick die Mail in einem neuen Fenster öffnen aktiviert, wird die Mail durch Doppelklicken in einem neuen Fenster geöffnet.

  • Aktivieren Sie Bilder, um Bilder in HTML-Mails automatisch herunterzuladen. Wenn diese Option aktiviert ist, müssen Sie nicht extra klicken, um das Bild in der Mail anzuzeigen.

  • Unter Beim Lesen im Lesefenster legen Sie das Verhalten zum Markieren von Mails als gelesen oder ungelesen fest, wenn Sie die Mails im Lesefenster anzeigen. Sie können bestimmen, ob Mails sofort oder nach einer festgelegten Anzahl von Sekunden der Anzeige im Lesefenster als gelesen markiert werden, oder trotz Anzeige im Lesefenster als ungelesen markiert bleiben.

  • Nach dem Verschieben oder Löschen einer Mail legt das Navigationsverhalten im Inhaltsfenster beim Lesen von Mails fest.

  • Die Standardmailsuche definiert die Suche, die bei Ihrer Anmeldung durchgeführt werden soll. Die Ergebnisse dieser Suche werden im Inhaltsfenster angezeigt, wenn Sie sich an Ihrer Mailbox anmelden. Standardmäßig wird nach Ihrem Posteingang gesucht. Wenn Sie sich anmelden, wird deshalb der Inhalt Ihres Posteingangs angezeigt. Sie können dieses Feld ändern und einen anderen Ordner, einen Tag oder eine gespeicherte Suche angeben, damit diese Suche durchgeführt wird, wenn Sie sich beim Zimbra-Web-Client anmelden. Informationen zur Syntax finden Sie unter Beschreibung der Sprache für Suchabfragen.

  • Mails empfangen. Für den Empfang von Mails können Sie angeben, wie Sie benachrichtigt werden möchten: Sound abspielen, Mail-Register blinkend darstellen, Browser-Titel blinkend darstellen. Siehe Desktop-Benachrichtigungen einrichten

  • Eine Kopie weiterleiten an. Geben Sie eine Adresse an, an die Ihre Mails weitergeleitet werden. Die ursprüngliche Mail kann dabei aus Ihrer Zimbra-Mailbox gelöscht werden.

  • Geben Sie an, ob auf eingehende Mails automatische Antwort senden festgelegt werden soll. Richten Sie hierfür ein Anfangs- und Enddatum ein.  Diese Funktion wird häufig als Abwesenheitsmeldung genutzt. Sie können den Text selbst eingeben. Sie können auch ein Anfangs- und Enddatum für das Senden dieser Abwesenheitsmails festlegen. Siehe Abwesenheitsmeldung.

  • Lesebestätigung. Geben Sie an, wie bei Mails vorgegangen werden soll, bei denen der Absender der Mail eine Lesebestätigung angefordert hat.

  • Unter „Mails von mir ...“ legen Sie fest, wie mit Mails verfahren werden soll, die von Ihnen gesendet werden, die Ihre Adresse als einen der Empfänger der Mail enthalten oder die an eine Adresse wie einen Verteiler gehen, in dem auch Sie enthalten sind.

  • Optionen für Spam-Mail. Sie können maximal 100 Adressen angeben, deren Mails immer blockiert und nicht in Ihrem Posteingang zugestellt werden sollen. Außerdem können Sie 100 Adressen angeben, für die Sie sicherstellen möchten, dass deren Mails immer in Ihrem Posteingang zugestellt und nicht im Spam-Ordner abgelegt werden.

  • Wenn Sie einen POP3-Client nutzen, können Sie das POP-Zugriffsverhalten beim Laden von Mails in ZWC einrichten. Folgende Optionen für den POP-Zugriff sind möglich:

    • Herunterladen aller Mails, einschließlich alter Mails, zulassen

    • Herunterladen der Mails ab jetzt zulassen.
