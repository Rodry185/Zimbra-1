Eine Liste der vertrauenswürdigen Adressen verwalten

Auf der Seite Vertrauenswürdige Adressen unter Einstellungen können Sie eine Liste vertrauenswürdiger Adressen oder Domänen anlegen, die integrierte Bilder in den Mails anzeigen können, die sie Ihnen zusenden.  Auf dieser Seite können Adressen hinzugefügt oder gelöscht werden. Sie können auch der Seite "Vertrauenswürdige Adressen" eine Mail-Adresse aus der Mail hinzufügen.

Sie können dieser Liste bis zu 500 Adressen hinzufügen.
