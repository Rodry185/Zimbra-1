Externe Accounts hinzufügen

Statt sich bei jedem Ihrer externen Mail-Accounts anzumelden, können Sie diese Accounts direkt in diesem Zimbra-Account auf Mails abfragen. Siehe Auf externe Accounts zugreifen.

So fügen Sie externe Accounts zu Ihrem Web-Client-Account hinzu

  1. Öffnen Sie die Registerkarte Einstellungen>Accounts und klicken Sie auf Externen Account hinzufügen. Ein Accountname Neuer externer Account-(Nummer) wird zu Ihrer Account-Liste hinzugefügt. Nun können Sie diesen Account einrichten.

  2. Geben Sie im Bereich Einstellungen für externen Account im Feld Accountname einen Namen für den Account ein. Neuer externer Account wird in diesen Namen geändert. Beim Verfassen neuer Mails können Sie diesen Accountnamen aus der Liste der Absenderadressen unter "Von" auswählen.

  3. Geben Sie im Feld Mail-Adresse die genaue Mail-Adresse des externen Accounts ein.

  4. Konfigurieren Sie nun die Account-Einstellungen. Diese Daten müssen Sie ggf. bei Ihrem Mail-Provider erfragen.

    • Wählen Sie den Account-Typ, POP oder IMAP.

    • Nutzername des Accounts. Geben Sie den Namen dieses Mail-Accounts ein.

    • Mail-Server. Geben Sie den Hostnamen des Mail-Servers ein.

    • Passwort: Geben Sie das Passwort für die Anmeldung an diesem externen Account ein. Um zu überprüfen, ob das eingegebene Passwort korrekt ist, aktivieren Sie Passwort anzeigen.

    • Erweiterte Optionen Wenn Ihr Mail-Anbieter einen anderen Port als den Standardport nutzt, passen Sie ihn nun an. Wenn Sie eine sichere Verbindung nutzen, aktivieren Sie "Für den Zugriff auf diesen Server eine verschlüsselte Verbindung ( SSL) nutzen".

  5. Wählen Sie unter Mails herunterladen auf aus, in welchem Ordner die Mails gespeichert werden sollen. Standardmäßig legen Sie einen neuen Ordner an und geben diesen zum Ablegen der Mails an. Sollen die Mails in Ihrem Posteingang gespeichert werden, wählen Sie "Posteingang".

  6. Aktivieren Sie Mails nach dem Herunterladen auf dem Server löschen, wenn die Mails in Ihrem externen Account gelöscht werden sollen, sobald sie an Ihren @ClientNameShort@-Account gesendet werden.

  7. Geben Sie in das Textfeld unter dem Feld  Von: den Namen ein, der in Ihren Mails, die Sie senden, im Feld „Von“ angezeigt werden soll (z. B. Peter Meier). Dies ist der ausgeschriebene Name, der vor Ihrer Mail-Adresse angezeigt wird. Wählen Sie die richtige Mail-Adresse, die zusammen mit diesem Namen verwendet werden soll.

  8. Wenn Sie möchten, dass Antworten an einen anderen Namen und an eine andere Adresse und nicht an die im Feld "Von" angezeigte Adresse gesendet werden sollen, aktivieren Sie Antwortadresse. Geben Sie den Namen und die Adresse ein, an die die Antworten für diese Identität gesendet werden sollen. Beispiel: Antworten auf Schulungsmails, die Peter Meier sendet, sollen an Mitarbeiterschulung unter training@example.com gesendet werden.

  9. Klicken Sie auf Signaturen verwalten..., um eine Signatur anzulegen, die mit dieser Persönlichkeit verwendet wird.

  10. Klicken Sie auf Speichern, um diesen externen Account einzurichten.
