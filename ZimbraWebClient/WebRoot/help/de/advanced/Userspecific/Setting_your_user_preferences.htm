Benutzereinstellungen einrichten

Wenn Ihr Account angelegt wird, werden die Standardeinstellungen konfiguriert. Diese Einstellungen legen fest, wie die @ClientNameLong@-Features funktionieren.

Sie können diese Einstellungen auf der Registerkarte Einstellungen ändern. Die Einstellungsseiten werden im Übersichtsfenster angezeigt.  Wenn eine hier beschriebene Option in Ihrer Ansicht nicht verfügbar ist, ist die Funktion für Ihren Account nicht eingerichtet.

Sie können die Einstellungen in folgenden Ordnern festlegen. Klicken Sie für eine Beschreibung des Inhalts auf den Ordnernamen.

  • Allgemein

    Auf der Seite „Allgemein“ können Sie

    • die ZWC-Standardversion, die verwendet werden soll – "Erweitert" oder "Standard" – konfigurieren und Ihren Mailbox-Stil, die Sprache und die Standard-Zeitzone festlegen,

    • die Sucheinstellungen festlegen, um den Spam-Ordner bzw. den Papierkorb in Ihre Suche(n) aufzunehmen und

    • Kontrollkästchen im Inhaltsfenster anzeigen

    Weitere Informationen finden Sie unter Allgemeine Einstellungen einrichten

  • Mail

    Auf der Seite „Mail“ können Sie:

    • Konfigurieren, wie oft kontrolliert werden soll, ob neue Mails vorhanden sind. Der Standardwert ist 5 Minuten.

    • Das Verhalten der Schaltfläche "Mails empfangen" festlegen, um entweder Ihre Standardsuche auszuführen oder Ihre aktuelle Ansicht zu aktualisieren.

    • Festlegen, wie Mails angezeigt werden sollen, in HTML oder als Plain-Text.

    • Einrichten, wie Ihre Mailbox empfangene Mails handhabt, einschließlich den Mail-Vorschau-Optionen, dem Lesefenster und den Verschiebungsoptionen.

    • Mail-Benachrichtigungen einrichten.

    • Eine automatische Abwesenheitsnachricht einrichten.

    • Die Lesebestätigung aktivieren.

    • Wenn Sie einen POP3-Client nutzen, können Sie das POP-Zugriffsverhalten beim Laden von Mails in @ClientNameShort@ konfigurieren.

    • Spam-Mail-Optionen festlegen, um Mails zu blockieren und Nachrichten von speziell festgelegten Adressen zuzulassen.

    Weitere Informationen finden Sie unter Maileinstellungen einrichten

  • Verfassen

    Auf der Seite „Verfassen“ können Sie

    • Den Texteditor festlegen, den Sie standardmäßig nutzen möchten: HTML oder Plain-Text.

    • Auswählen, ob der ursprüngliche Text im Textkörper Ihrer Antwort-Mail enthalten sein soll.

    • Die Rechtschreibprüfung als obligatorisch festlegen.

    • Festlegen, wie weitergeleitete oder beantwortete Mails gekennzeichnet werden sollen.

    • Standardmäßig ist die Option Kopie im Ordner „Gesendet“ speichern aktiviert. Diese Eingabe können Sie ändern.

    Weitere Einstellungen finden Sie unter Einstellungen für das Fenster zum Verfassen von Mails einrichten

  • Signaturen

    Sie können für Ihre Mails Signaturen anlegen.  Wenn Sie mehrere Identitäten anlegen, können Sie verschiedene Signaturen anlegen und diese den jeweiligen Adressen zuordnen.

  • Adressbuch

    Auf der Seite „Adressbuch“ können Sie

    • Die Funktion aktivieren, dass beim Senden von Mails die Adressen automatisch zu Ihrem Mail-Empfänger-Adressbuch hinzugefügt werden sollen.

    • Aktivieren, dass die "Globale Adressliste" bei der Anzeige von Kontakten in der Kontaktauswahl genutzt wird.

    • Die "Globale Adressliste" für dieAutovervollständigung von Adressen aufnehmen.

    • Adressen bei der Autovervollständigung von Adressen in freigegebene Adressbücher aufnehmen. 

    Weitere Informationen finden Sie unter Einstellungen für das Adressbuch einrichten

  • Accounts

    Auf der Seite „Accounts“ erstellen und verwalten Sie Ihre Persönlichkeiten. Eine Persönlichkeit ist die Mail-Identität, die in von Ihnen gesendeten Mails im Feld "Von" verwendet wird. Sie können Ihre Accounts  und POP-Accounts einrichten

  • Mailfilter

    Im Ordner „Mailfilter“ definieren Sie die Regeln für Mailfilter. Weitere Informationen dazu, wie Sie Regeln für Mailfilter einrichten, finden Sie unter Funktionsweise von Filtern.

  • Kalender

    Auf der Seite „Kalender“ können Sie

    • Ihre Standardansicht der Kalender festlegen

    • Parameter festlegen, die bei der Erstellung eines Termins verwendet werden.

    • Auswählen, ob der  Mini-Kalender angezeigt werden soll.

    • Eine Adresse eingeben, an die Ihre Einladungen von einem freigegebenen Kalender gesendet werden

    • Erinnerungen für Besprechungen einrichten.

    • Berechtigungen für die Ansichten Frei/Beschäftigt festlegen und dafür, wer Sie zu Besprechungen einladen kann

    Weitere Informationen finden Sie unter Kalendereinstellungen einrichten

  • Freigeben

    Die Seite „Freigeben“ enthält Objekte, die für Sie freigegeben sind, sowie Objekte, die von Ihnen freigegeben werden. Weitere Informationen erhalten Sie unter Freigegebene Objekte verwalten

  • Benachrichtigungen

    Sie können eine Mail-Adresse oder eine SMS-Benachrichtigung für Ihr Mobilgerät konfigurieren, um eine Erinnerungs-Mail für eine Aufgabe oder eine in Ihrem Kalender festgelegte Besprechung zu erhalten.

  • Mobilgeräte

    Die Mobilgeräte, die Sie für die Synchronisierung mit dem ZWC konfiguriert haben, sind hier aufgeführt.  Sie können Ihr Gerät auch von dieser Seite löschen. Weitere Informationen erhalten Sie unter Mobilgeräte verwalten

  • Importieren/Exportieren

    Über die Seite Importieren/Exportieren können Sie Ihre vollständigen Account-Daten, bestimmte Ordner, Kalender und Kontaktlisten importieren bzw. exportieren. Dies ist beim lokalen Sichern Ihres Accounts oder eines bestimmten Ordners hilfreich.

  • Tastenkürzel

    Auf der Seite „Tastenkürzel“ können Sie die Liste der Tastenkürzel anzeigen und die Liste drucken.

  • Zimlets

    Sie können verwalten, welche Zimlets aus dem Ordner „Zimlets“ Sie verwenden wollen. Zimlets sind zusätzliche Anwendungen, die die Funktionalität Ihres Clients erweitern.  Zimlets werden vom Systemverwalter aktiviert. Die Zimlets, die Sie verwenden können, werden im Übersichtsfenster angezeigt.
