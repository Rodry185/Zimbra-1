Aufgabenliste freigeben

Sie können Ihre Aufgabenlisten für andere Nutzer freigeben. Dabei haben Sie folgende Möglichkeiten für die Berechtigungen:

  • Interne Nutzer oder Gruppen: Sie wählen die zu gewährenden Rechte aus:

    • Leseberechtigung: Der Nutzer kann den Inhalt der Aufgabenliste lesen, er kann jedoch keine Änderungen am Inhalt der Aufgabenliste vornehmen.

    • Manager. Der Nutzer hat die Berechtigung, Aufgaben in Ihrer Aufgabenliste anzulegen, zu ändern oder zu löschen.

    • Admin. Der Nutzer hat die Berechtigung, Aufgaben in Ihrer Aufgabenliste anzulegen, zu ändern oder zu löschen und Ihre Aufgabenlisten für andere Nutzer freizugeben.

    • Keine: Diese Option dient dazu, den Zugriff auf eine freigegebene Aufgabenliste für kurze Zeit zu deaktivieren, ohne die Freigabeberechtigungen zu widerrufen. Der Nutzer, dem die Freigabe erteilt wurde, hat die Aufgabenliste noch in seiner Liste, er kann jedoch keine Aufgaben anzeigen oder verwalten.

  • Externe Nutzer: Sie richten ein Passwort für den Zugriff auf die freigegebene Aufgabenliste ein. Die Nutzer müssen dieses Passwort eingeben, um die Aufgabenliste aufzurufen. Sie können keine Änderungen an den Aufgaben vornehmen.

  • Öffentlich: Jeder, der die URL der freigegebenen Aufgabenliste kennt, kann die Liste anzeigen. Änderungen sind nicht möglich.

Ihre Aufgabenliste freigeben

Klicken Sie mit der rechten Maustaste auf die freizugebende Aufgabenliste und wählen Sie "Aufgabenliste freigeben" aus.

Geben Sie im Dialogfeld "Eigenschaften der Freigabe" an, für wen Sie die Aufgabenliste freigeben möchten. Nehmen Sie die gewünschten Einstellungen vor.

 

Freigabe für interne Nutzer oder Gruppen1. Wählen Sie Interne Nutzer oder Gruppen, für die die Freigabe erfolgen soll

2. Geben Sie in das Feld Mail die Mailadressen der Nutzer ein. Während Sie eine Adresse eingeben, werden mit Ihrer Eingabe übereinstimmende Namen angezeigt. Sie können die Namen aus dieser Liste auswählen.

3. Wählen Sie die Rolle aus - Leseberechtigung oder Manager

4. Wählen Sie im Bereich Mail die Art der Mail aus, die zur Benachrichtigung an den Nutzer gesendet werden soll.

5. Klicken Sie auf OK.

Freigabe für externe Nutzer

1. Wählen Sie unter Freigabe für die Option Externe Nutzer. Wenn Sie diese Option wählen, wird das Feld "Passwort" angezeigt.

2. Geben Sie die Mail-Adressen der externen Nutzer ein.

3. Geben Sie das Passwort ein, mit dem die Nutzer Zugriff auf die Aufgabenliste erhalten.

4. Wählen Sie aus, welche Art der Benachrichtigung den Nutzern gesendet werden soll. Diese Mail teilt den Nutzern die Informationen mit, die sie für den Zugriff auf die Aufgabenliste benötigen.

5. Klicken Sie auf OK.

 

Öffentliche Freigabe

1. Wählen Sie unter Freigabe für die Option Öffentlich. Wenn Sie diese Option auswählen, wird im Dialogfeld "Eigenschaften der Freigabe" die URL für Ihre Aufgabenliste angezeigt.

2. Notieren Sie sich diese URL und teilen Sie "öffentlichen" Nutzern diese Adresse mit. Durch Aufruf dieser URL können die Nutzer den Inhalt Ihrer Aufgabenliste lesen, sie können jedoch keine Änderungen vornehmen.

Rechte ändern oder entziehen

Sie können die Rechte jederzeit entziehen.

  1. Klicken Sie mit der rechten Maustaste auf eine freigegebene Aufgabenliste.

  2. Klicken Sie auf Eigenschaften bearbeiten. Nutzer, für die der Ordner freigegeben wurde, werden aufgeführt.

  3. Klicken Sie auf Bearbeiten, um Rechte zu ändern, oder
    klicken Sie auf Entziehen, um Rechte zu löschen.

  4. Geben Sie an, ob eine Mail mit einem Hinweis zu der Änderung gesendet werden soll.

  5. Klicken Sie auf OK.
