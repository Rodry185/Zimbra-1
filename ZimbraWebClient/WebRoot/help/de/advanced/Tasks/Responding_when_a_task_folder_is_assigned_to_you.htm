Zugriff auf freigegebene Aufgabenlisten annehmen

Wenn Sie per Mail informiert werden, dass Ihnen der Zugriff auf die Aufgabenliste einer anderen Person gewährt wird, können Sie diese Freigabe annehmen oder ablehnen.

Wenn Sie die Freigabe annehmen, wird die freigegebene Aufgabenliste zusammen mit Ihren Aufgabenlisten im Übersichtsfenster angezeigt.

So nehmen Sie an

  1. Klicken Sie in der Mail auf Freigabe annehmen. Das Dialogfeld "Freigabe annehmen" wird geöffnet. Es beschreibt die Ihnen zugeordnete Rolle.

  2. Bevor Sie annehmen, können Sie den Namen der Aufgabenliste ändern und eine Farbe für die Anzeige der Liste in Ihrem Übersichtsfenster auswählen.

  3. (Optional) Geben Sie an, ob Sie eine Bestätigung an den freigebenden Nutzer senden möchten.

  4. Klicken Sie auf Ja. Die freigegebene Aufgabenliste wird im Übersichtsfenster zusammen mit Ihren Aufgabenlisten angezeigt. Die Mail wird in Ihren Papierkorb verschoben.
