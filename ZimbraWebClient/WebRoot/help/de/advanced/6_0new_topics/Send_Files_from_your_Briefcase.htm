Dateien aus Ihrem Ordner "Dateien" senden

Während Sie im Ordner "Dateien" arbeiten, können Sie per Mail einen Link auf die Dateien in diesem Ordner, aber auch eine Datei als Mailanhang versenden. Die Empfänger benötigen die entsprechende Software, um die Dateien zu öffnen.

Link auf eine Datei per Mail senden

Sie können Mails senden, die Links auf Dateien im Ordner "Dateien" enthalten. Der Empfänger benötigt die entsprechende Freigabeberechtigung, um mit diesen Dateien zu arbeiten. Alle Änderungen, die der Empfänger an einer Datei vornimmt, werden in Ihrer Datei im Ordner "Dateien" übernommen.

  1. Öffnen Sie den Ordner "Dateien" und klicken Sie mit der rechten Maustaste auf die Dateien, die Sie als Link senden möchten.

  2. Klicken Sie in der Symbolleiste auf Senden und wählen Sie Link(s) senden aus.
    Daraufhin wird die Seite zum Verfassen einer Mail geöffnet, wobei die Browser-Adresse für jeden Link im Textkörper der Mail angezeigt wird.

  3. Fügen Sie die Mail-Adresse des Empfängers ein und vervollständigen Sie den Text der Mail.

  1. Klicken Sie auf Senden.

 

Datei aus dem Ordner "Dateien" per Mail senden

Sie können Dateien direkt aus dem Ordner "Dateien" heraus als Mail-Anhänge senden.  

  1. Öffnen Sie den Ordner "Dateien" und klicken Sie mit der rechten Maustaste auf die Dateien, die Sie senden möchten.

  2. Klicken Sie in der Symbolleiste auf Senden und wählen Sie Als Anhang senden aus.
    Daraufhin wird die Seite zum Verfassen einer Mail geöffnet, wobei die Browser-Adresse des Links im Textkörper der Mail angezeigt wird.

  3. Fügen Sie die Mail-Adresse des Empfängers ein und vervollständigen Sie den Text der Mail.

  1. Klicken Sie auf Senden.
