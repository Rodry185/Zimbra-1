Ansichten benutzerspezifisch einrichten

Sie können Ihre Account-Ansicht benutzerspezifisch einrichten, indem Sie die Hintergrundfarbe festlegen, die angezeigt wird, wenn Sie sich an Ihrem Account anmelden. Außerdem können Sie das Inhaltsfenster Ihrer Mailbox-Ansicht, Ihrer Kalenderansicht und Ihrer Aufgabenansicht so einrichten, dass die Informationen in der von Ihnen gewünschten Reihenfolge dargestellt werden.

Hintergrundfarbe ändern

Sie können möglicherweise über Einstellungen > Allgemein die Hintergrundfarbe Ihrer Mailbox ändern. Im Abschnitt "Anmeldeoptionen" werden im Feld "Stile" die verfügbaren Stile aufgelistet.

Mailansicht ändern

Standardmäßig wird beim Öffnen Ihrer Mailbox der Posteingang angezeigt. Dies können Sie auf eine der folgenden Arten benutzerspezifisch einrichten:

  • Sie können den Ordner ändern, der beim ersten Anmelden angezeigt wird. Standardmäßig wird der Posteingang mit allen darin enthaltenen Mails angezeigt. Sie können jeden beliebigen Mailordner für die erste Ansicht nach dem Anmelden auswählen, Sie können auch bestimmte Suchkriterien festlegen, z. B. ungelesene Mails. Dann werden nach dem Anmelden nur die ungelesenen Mails angezeigt. Siehe Startseite Ihrer Mail benutzerspezifisch einrichten.

  • Wählen Sie aus, ob das Lesefenster angezeigt werden soll oder nicht, und wenn ja, ob das Lesefenster unten oder rechts dargestellt werden soll. Diese Änderungen nehmen Sie über den Link "Ansicht" in der Mail-Symbolleiste vor.

  • Wenn Ihr Lesefenster ausgeschaltet ist oder im unteren Bereich angezeigt wird, können Sie die Spalten auswählen, die in der Mailliste im Inhaltsfenster dargestellt werden sollen. Unter Umständen wollen Sie im Inhaltsfenster nicht alle verfügbaren Details zu einer Mail angezeigt bekommen. Klicken Sie mit der rechten Maustaste auf den Spaltenkopf, um ein Menü mit den Spaltenüberschriften aufzurufen. Um eine Spalte aus dem Inhaltsfenster zu entfernen, klicken Sie auf den Spaltennamen. Dieser Befehl dient zum Umschalten, wenn Sie also die Spalte wieder anzeigen möchten, klicken Sie erneut auf den Spaltennamen.

Kalenderansicht ändern

Sie können festlegen, welche Ansicht beim Öffnen des Kalenders angezeigt wird: Tag, Arbeitswoche, 7-Tage-Woche, Monat, Planung oder Liste. Sie können außerdem festlegen, mit welchem Wochentag Ihre Kalenderwoche beginnen soll. Siehe Kalenderansichten.

Aufgabenansicht ändern

Sie können auswählen, welche Spalten im Inhaltsfenster dargestellt werden, wenn Sie Ihre Aufgaben anzeigen. Klicken Sie mit der rechten Maustaste auf den Spaltenkopf, um ein Menü mit den Spaltenüberschriften aufzurufen. Um eine Spalte aus dem Inhaltsfenster zu entfernen, klicken Sie auf die Spalte. Dieser Befehl dient zum Umschalten; wenn Sie also die Spalte wieder anzeigen möchten, klicken Sie erneut auf den Spaltennamen.
