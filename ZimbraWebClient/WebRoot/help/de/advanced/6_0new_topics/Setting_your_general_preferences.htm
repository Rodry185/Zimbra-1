Allgemeine Einstellungen einrichten

Unter "Allgemein" können Sie die folgenden Einstellungen ändern.

  • Sie sind am erweiterten Web-Client angemeldet, doch Sie können Ihren Standard-Client-Typ hier festlegen.

    • Die Einstellung Erweitert bietet die vollständigen Webkommunikationsfunktionen. Dieser Web-Client funktioniert am besten mit neueren Browsern und schnelleren Internetverbindungen.

    • Die Einstellung Standard wird empfohlen, wenn Sie mit einer langsamen Internetverbindung oder einem älteren Browser arbeiten oder einen barrierefreien Zugang wünschen.  

  • Der Stil ist die Hintergrundfarbe Ihrer Mailbox-Oberfläche. Auf der Registerkarte "Allgemein" wird unter "Stile" eine Liste der Hintergrundfarben angezeigt, die zur Auswahl stehen.

  • Sie können die Sprache auswählen, in der der Zimbra-Web-Client angezeigt wird.

  • Unter Standardzeitzone legen Sie das Datum und die Uhrzeit für die von Ihnen gesendeten und empfangenen Mails sowie für Termine im Kalender fest. Diese Zeitzone kann sich von der Zeitzone, die für Ihren Computer eingerichtet ist, unterscheiden. Wenn die angezeigte Zeitzone nicht korrekt ist, können Sie sie hier ändern.  

  • Wenn Passwort ändern angezeigt wird, können Sie Ihr Passwort ändern. Wenn Sie beim Ändern Ihres Passworts Schwierigkeiten haben, erfragen Sie die Passwortregeln bei Ihrem Administrator. Wenn diese Option nicht verfügbar ist und Sie Ihr Passwort ändern möchten, wenden Sie sich an Ihren Systemadministrator.

  • Unter Sucheinstellungen können Sie angeben, dass bei jeder Suche automatisch auch im Papierkorb und im Spam-Ordner gesucht wird. Standardmäßig werden diese Ordner nicht durchsucht.

  • Unter Sprachsuche können Sie die Option Suchzeichenfolge immer anzeigen aktivieren, wenn die Details der Suchabfrage im Suchtextfeld oben im ZWC-Fenster angezeigt werden sollen.  

  • Aktivieren Sie unter Einstellungen die Option Kontrollkästchen für die schnelle Auswahl von Objekten in Listen anzeigen (Aktualisierung erforderlich), damit für jedes Objekt im Inhaltsfenster ein Kontrollkästchen angezeigt wird. Wenn diese Option aktiviert ist, können Sie über die Kontrollkästchen eines oder mehrere Objekte für einen Bearbeitungsschritt auswählen, z. B. für das Löschen, das Verschieben oder das Markieren als gelesen oder ungelesen.
