Desktop-Benachrichtigungen und Erinnerungen einrichten

Sie können zur Benachrichtigung bei neuen Mails, Erinnerungen an Termine und Benachrichtigungen bei Instant Messages Desktop-Benachrichtigungen erstellen. Um Benachrichtigungen zu erhalten, muss Ihre Mailbox geöffnet sein.

  • Mail-Benachrichtigungen. Sie richten Ihre Einstellungen unter "Einstellungen > Mail" ein. Folgende Benachrichtigungsoptionen sind verfügbar:

    • Sound abspielen. Sie werden durch einen Signalton benachrichtigt, wenn in Ihrem Posteingang eine neue Mail eingeht. Wenn Sie die Lautstärke stumm geschaltet haben, hören Sie keinen Signalton. Sie müssen QuickTime oder Windows Media installiert haben.

    • Mail-Register hervorheben. Die Registerkarte "Mail" wird hervorgehoben, wenn im Posteingang eine neue Mail eingeht und Sie nicht auf der Registerkarte "Mail" arbeiten.

    • Browser-Titel blinkend darstellen. Wenn im Posteingang eine neue Mail eingeht, blinkt der Browser, bis Sie die Registerkarte "Mail" öffnen.

    • Popup-Benachrichtigung anzeigen. Wenn eine neue Mail eingeht, wird eine Popup-Meldung angezeigt. Sie müssen Yahoo! BrowserPlus installiert haben.

  • Erinnerungen an Termine. Sie richten Ihre Einstellungen unter "Einstellungen > Kalender" ein. Bevor Sie eine Benachrichtigung einstellen können, müssen Sie die Uhrzeit festlegen, zu der Sie an Ihren Termin erinnert werden möchten. Dies kann bis zu 60 Minuten vor dem Termin sein. Eine Erinnerungsmeldung wird zu der festgelegten Uhrzeit vor dem Termin angezeigt. Wenn Sie für Erinnerungen die Option Nie auswählen, werden Sie auch dann nicht an den Termin erinnert, wenn Sie die Erinnerungsoptionen aktiviert haben.

Die ZWC-Erinnerung wird nicht im Vordergrund der Seite eingeblendet, wenn Sie sich nicht im ZWC-Kalender befinden. Wenn Sie eine der folgenden Optionen aktivieren, nehmen Sie die Erinnerung besser wahr:

    • Sound abspielen. Sie werden durch einen Signalton benachrichtigt, wenn eine Erinnerung an einen Termin eingeblendet wird. Wenn Sie die Lautstärke stumm geschaltet haben, hören Sie keinen Signalton.

    • Browser-Titel blinkend darstellen. Wenn eine Erinnerung an einen Termin eingeblendet wird, blinkt der Browser, bis Sie das Popup-Fenster schließen.

  • IM-Benachrichtigungen. Sie richten Ihre Einstellungen unter "Einstellungen > IM" ein. Sie müssen unmittelbare Benachrichtigungen aktivieren, damit die Benachrichtigung angezeigt wird. Folgende Benachrichtigungsoptionen sind verfügbar:

    • Sound abspielen. Sie werden durch einen Signalton benachrichtigt, wenn eine IM eingeht. Wenn Sie die Lautstärke stumm geschaltet haben, hören Sie keinen Signalton.

    • Popup-Benachrichtigung anzeigen. Die Registerkarte "IM" wird hervorgehoben, wenn eine neue IM eingeht.

    • Browser-Titel blinkend darstellen. Wenn eine IM eingeht, blinkt der Browser, bis Sie die Registerkarte "IM" öffnen.
