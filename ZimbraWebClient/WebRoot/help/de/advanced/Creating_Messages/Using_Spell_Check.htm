Rechtschreibprüfung nutzen

Sie können die Rechtschreibung Ihrer Mail prüfen, bevor Sie sie versenden. Klicken Sie in der  Verfassungssymbolleiste auf Rechtschreibprüfung.

Wenn Sie die Rechtschreibung aller Mail-Nachrichten automatisch prüfen möchten, bevor sie versendet werden, aktivieren Sie die Option Obligatorische Rechtschreibprüfung vor dem Senden einer Mail auf Ihrer Seite Einstellungen >Verfassen.

  1. Klicken Sie auf Rechtschreibprüfung. Wörter, die falsch geschrieben oder unbekannt sind, werden hervorgehoben.

  2. Klicken Sie auf ein hervorgehobenes Wort. Ein Popup zeigt Korrekturvorschläge an.

  3. Wählen Sie das korrekte Wort aus. Das Wort wird in einer anderen Farbe hervorgehoben.

  4. Um Ihre Änderungen zu übernehmen und die Rechtschreibprüfung zu schließen, klicken Sie auf Bearbeitung fortsetzen.

Wörter zum Wörterbuch für die Rechtschreibprüfung hinzufügen

Sie können Wörter zu Ihrem Wörterbuch für die Rechtschreibprüfung hinzufügen.

  1. Klicken Sie auf Rechtschreibprüfung. Wörter, die falsch geschrieben sind, werden hervorgehoben.

  2. Klicken Sie auf ein hervorgehobenes Wort. Ein Popup zeigt Korrekturvorschläge an.

  3. Falls das als falsch markierte Wort korrekt ist, klicken Sie auf das Wort und anschließend auf Hinzufügen. Dieses Wort wird Ihrem Wörterbuch hinzugefügt.

Sie können einmal im Wörterbuch eingefügte Wörter nicht wieder löschen.
