Mit Dateien im Ordner "Dateien" arbeiten

Dateien im Ordner "Dateien" öffnen

Dateien, die sich in Ihrem Ordner "Dateien" befinden, können Sie auf jedem Computer öffnen. Sie benötigen die entsprechende Softwareanwendung, um die Datei zu öffnen, zu lesen und zu ändern. Gängige Dateiformate wie Textdateien, HTML-Dateien und Bilder in den Formaten .gif oder .jpg können mit einer Vielzahl von Programmen geöffnet werden.

Um eine Datei zu öffnen, klicken Sie mit der rechten Maustaste auf die Datei und wählen Öffnen aus.

Hinweis: Die Datei kann auch durch Doppelklicken darauf geöffnet werden.

  • Wenn der Dateityp unbekannt ist, wird ein Dialogfeld angezeigt und Sie werden gefragt, wie Sie vorgehen möchten: Ob Sie sie diese Datei auf die Festplatte speichern oder in einer bestimmten Anwendung öffnen möchten.

  • Wenn Sie die Dateien für Andere freigegeben haben, sollten Sie die Datei lieber auschecken, damit keine anderen Benutzer die Datei im Ordner "Dateien" bearbeiten können.

    • Klicken Sie mit der rechten Maustaste auf die Datei und wählen Sie dann Datei auschecken. Sie können die Datei entweder öffnen oder sie auf Ihrem Computer speichern.

    • Nachdem Sie Ihre Änderungen vorgenommen haben, laden Sie die Datei hoch. Wenn die Datei, die hochgeladen wird, denselben Namen hat wie die Datei im Ordner "Dateien" wird ein Dialogfeld "Konflikt beim Hochladen" angezeigt. Wenn Sie "Ja" wählen, wird die Datei hochgeladen und es wird eine neue Version der Datei erstellt. Wenn Sie "Nein" wählen, wird die Datei nicht hochgeladen. Sie müssen den Dateinamen ändern oder ihn in einen anderen Ordner "Dateien" hochladen.

  • Wenn der Dateityp ein Zimbra-Dokument ist, öffnen Sie die Datei mit einem Doppelklick auf einer Browserseite. Sie können in dieser Ansicht kein Zimbra-Dokument bearbeiten.  

Zimbra-Dokumentendatei bearbeiten

  • Klicken Sie mit der rechten Maustaste auf die Zimbra-Dokumententdatei und wählen Sie Bearbeiten. Die Datei wird auf einer neuen Browserseite geöffnet.

  • Bearbeiten Sie die Seite und und klicken Sie zum Schließen der Seite auf Speichern und Schließen. Wenn Sie Ihre Arbeit nicht speichern wollen, schließen Sie die Browser-Registerkarte.

Hinweis: Wenn Sie auf Speichern oder Speichern und Schließen klicken, wird eine neue Version der Datei erstellt

Dateien verschieben

Sie können Dateien von einem Ordner "Dateien" in einen anderen verschieben.

  1. Wählen Sie die Datei(en), die verschoben werden soll(en), aus.

  2. Klicken Sie in der Symbolleiste auf .

  3. Wählen Sie den Zielordner oder erstellen Sie einen neuen Ordner "Dateien".

  4. Klicken Sie auf OK.

Dateien mit Tags markieren

Sie können Dateien mit Tags markieren.

  1. Klicken Sie auf die gewünschte Datei und dann in der Symbolleiste auf .

  2. Wählen Sie den Tag aus. Tags werden nur in der Detailansicht angezeigt.

  3. Klicken Sie auf OK.
