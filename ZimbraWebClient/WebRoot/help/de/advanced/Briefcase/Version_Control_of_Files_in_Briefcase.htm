Versionskontrolle der Dateien im Ordner "Dateien"

Mithilfe der Versionskontrolle können Sie Änderungen an ihren Dokumenten im Ordner "Dateien" vornehmen. Jedes Mal, wenn Sie eine Änderung an einer Datei vornehmen und diese Datei dann speichern, wird die Versionsnummer aktualisiert. Die neueste Versionsnummer wird in der Dokumentenliste angezeigt.  

Klicken Sie auf , um den Versionsverlauf und die Verfasser einzublenden. Zur Wiederherstellung einer älteren Version klicken Sie mit der rechten Maustaste auf eine andere Version. Dann wählen Sie Als aktuelle Version wiederherstellen.  Zum Löschen einer Version, die Sie nicht länger benötigen, klicken Sie mit der rechten Maustaste auf die Datei und wählen Sie Löschen.

Hinweis: Jede Version einer Datei wird gegen Ihr Accountkontingent gewertet.  

Dateien, die Sie hochladen, können ausgecheckt werden, um zu verhindern, dass mehrere Personen an derselben Datei im Ordner "Dateien" arbeiten. Diese Dateien werden auf Ihren Desktop heruntergeladen. Wenn Sie die aktualisierte Datei hochladen, wird diese eingecheckt und die Version wird geändert.
