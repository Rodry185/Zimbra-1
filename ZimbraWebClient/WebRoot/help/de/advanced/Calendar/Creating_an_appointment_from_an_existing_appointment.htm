Termin anhand eines vorhandenen Termins anlegen

Sie können einen Termin aufgrund eines bereits vorhandenen ähnlichen Termins anlegen. Die bereits festgelegten Teilnehmer, Orte, Hinweise und Wiederholungsintervalle können auf diese Weise schnell für neue Daten und Uhrzeiten konfiguriert werden. Sie können die Angaben auch ändern, bevor Sie die Einladungen senden.

  1. Klicken Sie mit der rechten Maustaste auf den Termin, der kopiert werden soll, und wählen Sie Eine Kopie erstellen. Die Seite "Termin erstellen" wird geöffnet.

  2. Wenn Sie einen Termin anlegen möchten, der sich von dem kopierten Termin nur durch das Datum unterscheidet, brauchen Sie nur das Anfangs- und Enddatum zu ändern.

  3. Nehmen Sie alle weiteren gewünschten Änderungen an den Teilnehmern, Orten, am Text usw. vor.

  4. Klicken Sie auf Senden, um eine Einladung an die Teilnehmer zu senden. Der Termin wird in Ihren Kalender eingetragen.
