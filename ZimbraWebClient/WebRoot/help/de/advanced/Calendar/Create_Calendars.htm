Kalender erstellen

Ein Kalender ist bereits vorhanden. Sie können weitere Kalender erstellen, um einen Überblick über verschiedene Arten von Ereignissen zu haben.  Sie können z. B. einen Kalender für die Arbeit, einen privaten Kalender und einen für Aktivitäten der Firma erstellen. In der Planungsansicht werden Ihnen die Kalender nebeneinander angezeigt.

So erstellen Sie einen Kalender

  1. Öffnen Sie die Registerkarte "Kalender".

  2. Im Fenster Kalenderübersicht klicken Sie auf oder klicken Sie in der Symbolleiste  auf den Pfeil und wählen Sie Kalender.

  3. Geben Sie im Dialogfeld Neuen Kalender erstellen den Namen für den Kalender ein und wählen Sie eine Farbe aus, in der die Aktivitäten auf diesem Kalender angezeigt werden sollen.

  4. Wenn die Aktivitäten dieses Kalenders nicht von anderen Nutzern eingesehen werden sollen, aktivieren Sie Diesen Kalender bei Frei-/Belegt-Zeiten ignorieren.

  5. Zum Erstellen eines Kalenders und zum Synchronisieren der Termine mit einem externen Kalender markieren Sie das Kontrollkästchen Termine von externem Kalender synchronisieren. Geben Sie die gültige URL einer iCal-Datei eines externen Kalenders ein.

  6. Klicken Sie auf OK. Der neue Kalender wird in der Kalenderliste angezeigt.

Externe Kalender werden regelmäßig entsprechend den vom Systemadministrator festgelegten Abfrageintervallen für dezentrale Kalender aktualisiert.
