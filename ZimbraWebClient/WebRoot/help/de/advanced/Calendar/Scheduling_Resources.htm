Ressourcen planen

Als Ressourcen werden ein Ort (Tagungsraum, Gebäude oder anderer Ort) oder ein Gerät (audio/visuell) angesehen, die für eine Besprechung geplant werden können. Beim Einrichten einer Besprechung wählen Sie zum Planen einer Ressource die entsprechende Ressource aus und "laden" Sie zur Besprechung ein.  Die Ressource erhält die Einladung und nimmt sie an, sofern sie frei ist.  

Administratoren richten Ressourcen-Accounts ein und konfigurieren die Planungsregeln der Ressource. Eine Besprechung kann abgelehnt werden, wenn die Ressource bereits über eine geplante Besprechung für diesen Zeitraum verfügt.  In einigen Fällen wird die Ressourcenplanungsregel so eingerichtet, dass eine Ressource Datumsangaben in wiederkehrenden Besprechungen annehmen kann, selbst wenn es zu Konflikten kommt.  Wenn Sie eine Ressource zu einer wiederkehrenden Besprechung einladen und einige der Daten bereits geplant sind, wird ein Dialogfeld Ressourcenkonflikt mit dem sich überschneidenden Datum und der sich überschneidenden Uhrzeit eingeblendet.  Sie erhalten außerdem eine Mail mit den sich überschneidenden Daten.
