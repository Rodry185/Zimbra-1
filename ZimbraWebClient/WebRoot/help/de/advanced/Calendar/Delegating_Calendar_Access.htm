Kalender freigeben

Sie können Ihre Kalenderinformationen anderen Menschen freigeben und Ihnen folgende Zugangsberechtigungen gewähren:

  • Interne Nutzer oder Gruppen:  Sie wählen die zu gewährenden Rechte aus:

    • Leseberechtigung: Der Nutzer sieht die in Ihrem Kalender eingetragenen Aktivitäten und den Status von Besprechungsanfragen, er kann jedoch keine Änderungen an Ihrem Kalender vornehmen.

    • Manager. Der Nutzer hat die Berechtigung, Besprechungen anzulegen, Einladungen anzunehmen oder abzulehnen und Aktivitäten in Ihrem Kalender zu ändern oder zu löschen.

    • Admin. Der Nutzer hat die Berechtigung, Besprechungen anzulegen, Einladungen anzunehmen oder abzulehnen, Aktivitäten in Ihrem Kalender zu ändern oder zu löschen, Ihren Kalender für andere Nutzer freizugeben und andere Nutzer von Ihrem freigegebenen Kalender zu entfernen.

    • Keine: Diese Option dient dazu, den Zugriff auf einen freigegebenen Kalender vorübergehend zu deaktivieren, ohne die Freigabeberechtigungen zu widerrufen. Der Nutzer, dem die Freigabe erteilt wurde, hat den Kalender noch in seiner Kalenderliste, er kann jedoch keine Aktivitäten im freigegebenen Kalender anzeigen oder verwalten.

  • Externe Nutzer: Sie richten ein Passwort für den Zugriff auf Ihren freigegebenen Kalender ein. Die Nutzer müssen dieses Passwort eingeben, um den Kalender aufzurufen. Sie können keine Änderungen am Kalender vornehmen.

  • Öffentlich: Jeder, der die URL des freigegebenen Kalenders kennt, kann den Kalender anzeigen. Sie können keine Änderungen am Kalender vornehmen.

Private Termindetails freigeben

Wenn Sie Ihren Kalender freigeben, können Sie das Kontrollkästchen Die Anzeige privater Termine durch andere Nutzer zulassen markieren, damit interne Benutzer oder Gruppen und externe Gäste Ihre privaten Termine sehen können. Sie werden Details von Terminen sehen können, die als privat gekennzeichnet sind.  Wenn Sie das Kontrollkästchen nicht markieren, zeigt der Kalender während einer privaten Termindauer nur an, dass Sie beschäftigt sind.

So geben Sie Ihren Kalender frei

  1. Klicken Sie mit der rechten Maustaste auf den Kalender, den Sie freigeben möchten, und wählen Sie "Kalender freigeben" aus.

  2. Wählen Sie im Dialogfeld "Eigenschaften der Freigabe" aus, für wen Sie den Ordner freigeben möchten.

Interne Nutzer oder Gruppen

  1. Wenn Sie "Interne Nutzer oder Gruppen" wählen, können Sie aus der beim Eingeben angezeigten Liste Adressen auswählen oder Sie können Adressen eingeben, die nicht in der Liste stehen. Bei dem Nutzer bzw. der Gruppe muss es sich um eine gültige Adresse auf dem Mailserver Ihres Unternehmens handeln.

  2. Im Bereich Rolle wählen Sie die Zugriffsrechte aus.

  3. Wenn Sie Informationen zu Ihren privaten Terminen freigeben möchten, aktivieren Sie die Option Die Anzeige privater Termine durch andere Nutzer zulassen.

  1. Wählen Sie im Bereich Mail aus, welche Art von Mail gesendet werden soll.  Es stehen folgende Optionen zur Auswahl:

  • Standardmail senden. Die Standardmail mit der Freigabebenachrichtigung wird an die Mail-Adresse gesendet.  

  • Notiz zur Standardmail hinzufügen. Sie können die Standardmail um zusätzliche Informationen erweitern.

  1. Klicken Sie auf OK.

 

Externe Nutzer

  1. Wenn Sie externe Nutzer auswählen, geben Sie die Mail-Adressen der Nutzer ein.   Sie können mehrere Adressen in das Feld „Mail“ eingeben. Trennen Sie die Adressen durch Semikolons (;).

  2. Geben Sie das Passwort ein, das die Nutzer eingeben müssen, um Ihren Kalender anzuzeigen.

  3. Wenn Sie Informationen zu Ihren privaten Terminen freigeben möchten, aktivieren Sie die Option Die Anzeige privater Termine durch andere Nutzer zulassen.

  1. Wählen Sie im Bereich Mail aus, welche Art von Mail gesendet werden soll.  Die URL für Ihren Kalender und das Passwort werden in die Mail eingefügt.

    • Standardmail senden. Die Standardmail mit der Freigabebenachrichtigung wird an die Mail-Adresse gesendet.

    • Notiz zur Standardmail hinzufügen. Sie können die Standardmail um zusätzliche Informationen erweitern.

  2. Klicken Sie auf OK.

Öffentlich

Wenn Sie "Öffentlich" auswählen, kann jeder, der die URL kennt, Ihren Kalender anzeigen.  Es ist kein Passwort erforderlich.

Rechte ändern oder entziehen

Sie können die Rechte jederzeit ändern, um Ihren Kalender aufzurufen.

  1. Klicken Sie mit der rechten Maustaste auf einen freigegebenen Kalender.

  2. Klicken Sie auf Eigenschaften bearbeiten. Nutzer, für die der Ordner freigegeben wurde, werden aufgeführt.

  3. Klicken Sie auf Bearbeiten, um Rechte zu ändern, oder
    klicken Sie auf Entziehen, um Rechte zu löschen.

  4. Geben Sie an, ob eine Mail mit einem Hinweis zu der Änderung gesendet werden soll.

  5. Klicken Sie auf OK.

Alternativ können Sie die Berechtigungen von Ihrer Seite Einstellungen>Freigeben bearbeiten oder widerrufen.
