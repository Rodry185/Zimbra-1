Termin löschen

Termine können vom Organisator gelöscht werden und der Teilnehmer kann eine Besprechung von seinem Kalender löschen.

Organisator storniert die Besprechung

Als Organisator einer Besprechungseinladung stornieren Sie eine Besprechung folgendermaßen:

  1. Klicken Sie auf den Termin, der storniert werden soll.

  2. Klicken Sie mit der rechten Maustaste und wählen Sie Abbrechen. Wenn es sich bei dem Termin um eine Folge wiederkehrender Termine handelt, können Sie einen Einzeltermin der Folge oder die gesamte Terminserie löschen.

  3. Ein Bestätigungsdialogfeld wird eingeblendet, in dem Sie gefragt werden, ob Sie die Terminstornierungsmail bearbeiten wollen.

    • Wenn Sie auf "Nein" klicken, werden die Teilnehmer durch eine automatisch generierte Mail informiert, und der Termin wird aus den Kalendern der Teilnehmer gelöscht.

    • Wenn Sie auf "Ja" klicken, öffnet sich das Fenster zum Verfassen einer Mail und Sie können die Mail vor dem Versenden bearbeiten.

    • Wenn Sie auf Abbrechen klicken, wird die Mail nicht gelöscht.

Teilnehmer storniert die Besprechung

Als Besprechungsteilnehmer können Sie Meldungen aus Ihrem Kalender löschen. Wenn Sie dies tun, erhalten Sie die Gelegenheit, den Organisator zu benachrichtigen.

  1. Klicken Sie auf die Besprechung, das gelöscht werden soll.

  2. Wählen Sie im Bestätigungsdialogfeld, ob der Organisator benachrichtigt werden soll. Klicken Sie dann auf Ja, um diese Besprechung zu löschen.
