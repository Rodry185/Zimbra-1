Mails im HTML-Format anzeigen

Sie können Mails in zwei Formaten anzeigen.

  • HTML. HTML-Mails können formatierten Text, Nummerierungen, Aufzählungszeichen, farbigen Hintergrund und Links enthalten, wodurch der Text leichter lesbar wird. Einige Mail-Clients können keine HTML-Mails anzeigen.

  • Text. Mails im Plain-Text-Format enthalten keine HTML-Formatierungscodes. Plain-Text-Mails können in allen anderen Mail-Clients angezeigt werden. Sie können sich Mails auch dann im Plain-Text-Format anzeigen lassen, wenn sie HTML-Code enthalten.

Um Ihre Mails immer im HTML-Format anzuzeigen, wählen Sie Einstellungen>Mail die Option Als HTML (wenn möglich).  Die Mails werden formatiert angezeigt, sofern sie als HTML-Mails gesendet wurden.

Siehe auch Benutzereinstellungen einrichten.
