Anhänge blockieren

Je nach den Sicherheitsregeln in Ihrer Firma kann es sein, dass einige Arten von Dateianhängen nicht in Ihrer Mailbox angezeigt werden. Es wird möglicherweise die Büroklammer angezeigt, die darauf hinweist, dass Dateianhänge vorhanden sind, doch Sie können die Anhänge nicht öffnen.

Ihr Systemadministrator kann die folgenden Optionen zum Blockieren von Anhängen einrichten:

  • Er kann auswählen, welche Arten von Dateien sich öffnen lassen. Sie können z. B. Anhänge anzeigen, wenn diese in HTML umgewandelt werden können (wie Word-, PowerPoint-, PDF-, Excel- oder Textdokumente), aber Sie können keine Bilddateien, ZIP-Dateien oder EXE-Dateien anzeigen.

  • Alle Anhänge können unabhängig vom Dateityp angezeigt werden.

Diese Option entspricht nicht der Option, bei der alle Anhänge abgewiesen werden, wobei typischerweise die Zustellung von Mails blockiert wird, die möglicherweise Viren oder nicht annehmbaren Inhalt enthalten.

Wenn Sie einen Mail-Anhang erhalten, den Sie nicht öffnen können, wenden Sie sich an Ihren Systemadministrator.
