Neue Notizbücher anlegen

  1. Klicken Sie im Zimbra-Web-Client auf die Registerkarte „Notizen“.

  2. Klicken Sie mit der rechten Maustaste auf die Leiste Notizbücher und wählen Sie Neues Notizbuch aus.


     

  3. Daraufhin wird das Dialogfeld Neues Notizbuch erstellen angezeigt. Geben Sie einen Namen für das neue Notizbuch ein.

Dieser Name muss innerhalb der Hierarchie Ihrer Mailbox-Ordner eindeutig sein. Notizbücher, die Sie über die Leiste Notizbücher anlegen, dürfen nicht denselben Namen haben wie ein übergeordneter Ordner in der Mail, im Kalender oder im Adressbuch von Zimbra. Wenn Sie beispielsweise einen Kalender mit dem Namen „Urlaub“ haben, dürfen Sie keinem übergeordneten Notizbuch den Namen „Urlaub“ geben. Ein Notizbuch, das sich in einem anderen Notizbuch befindet, darf allerdings „Urlaub“ heißen.

  1. Wählen Sie die Ordnerhierarchie aus. Sie können ein neues übergeordnetes Notizbuch anlegen oder das neue Notizbuch in ein vorhandenes Notizbuch einfügen.

  2. Klicken Sie auf OK. Das neue Notizbuch wird in der Übersicht angezeigt.
