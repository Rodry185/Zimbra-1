Zugriff für Ihr Notizbuch ändern oder beenden

Sie können die im Dialogfeld mit den Notizbucheigenschaften erteilten Freigaberechte entziehen und Berechtigungen ändern.

So ändern Sie die Rechte

  1. Klicken Sie unter „Notizen“ auf das freigegebene Notizbuch, das Sie ändern möchten.

  2. Wählen Sie Eigenschaften bearbeiten. Daraufhin wird das Dialogfeld „Ordnereigenschaften“ angezeigt. In diesem Dialogfeld werden die Namen der Nutzer angezeigt, für die das Notizbuch freigegeben ist.

  3. Klicken Sie unter „Freigabe für diesen Ordner“ bei dem Namen, für den Sie Rechte ändern möchten, auf Bearbeiten. Daraufhin wird das Dialogfeld „Eigenschaften für Freigabe“ angezeigt. Ändern Sie die Rolle.

    • Leseberechtigung: Der Nutzer kann den Inhalt des Notizbuchs lesen, er kann ihn jedoch nicht ändern.

    • Manager: Der Nutzer hat vollständige Zugriffsrechte und kann den Inhalt des Notizbuchs anzeigen und ändern, neue Seiten hinzufügen oder Seiten löschen.

    • Keine: Diese Option dient dazu, den Zugriff auf ein freigegebenes Notizbuch vorübergehend zu deaktivieren, ohne die Freigabeberechtigungen zu widerrufen. Der Nutzer, dem die Freigabe erteilt wurde, hat das Notizbuch noch in seiner Liste, er kann jedoch keine Aktivitäten im freigegebenen Notizbuch anzeigen oder verwalten.

  1. Dem Nutzer wird eine Mail zur Information gesendet. Sie können aber auch auswählen, keine Mail zu senden bzw. eine benutzerdefinierte Mail zu senden.

  2. Klicken Sie auf OK.

So beenden Sie die Freigabe eines Notizbuchs

  1. Klicken Sie unter „Notizen“ mit der rechten Maustaste auf das freigegebene Notizbuch.

  2. Wählen Sie Eigenschaften bearbeiten. Daraufhin wird das Dialogfeld „Ordnereigenschaften“ angezeigt. In diesem Dialogfeld werden die Namen der Nutzer angezeigt, für die das Notizbuch freigegeben ist.

  3. Klicken Sie auf Entziehen. Daraufhin wird das Dialogfeld „Freigabe entziehen“ angezeigt. Dem Nutzer wird eine Mail zur Information gesendet. Sie können aber auch auswählen, keine Mail zu senden bzw. eine benutzerdefinierte Mail zu senden.

  4. Klicken Sie auf OK. Der Nutzer hat keinen Zugriff mehr auf Ihr Notizbuch. Der Nutzer kann das Notizbuch in seiner Liste löschen.
