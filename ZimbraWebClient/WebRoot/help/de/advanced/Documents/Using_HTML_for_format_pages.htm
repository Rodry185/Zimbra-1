Seiten unter „Notizen“ mit HTML-Quellcode formatieren

Wenn Sie mit HTML-Tags vertraut sind, können Sie sie für die Erstellung von Inhalten nutzen. Alle grundlegenden HTML-Elemente können genutzt werden. Um Seiten in HTML zu erstellen, klicken Sie in der Symbolleiste auf Format und wählen HTML-Quelle aus.

Wenn Sie eine Seite bearbeiten, können Sie zwischen Rich-Text und HTML-Quellcode umschalten. Klicken Sie in der Ecke oben rechts auf Format, um den gewünschten Editor auszuwählen.
