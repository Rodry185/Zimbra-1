Zugriff auf ein Notizbuch annehmen

Wenn Sie per Mail informiert werden, dass Ihnen der Zugriff auf ein Notizbuch einer anderen Person gewährt wird, können Sie diese Freigabe annehmen oder ablehnen. Notizbücher befinden sich auf der Registerkarte „Notizen“.

Wenn Sie die Freigabe annehmen, wird das freigegebene Notizbuch in Ihrer Notizbuchliste angezeigt.

So nehmen Sie an

  1. Klicken Sie in der Mail auf Freigabe annehmen. Das Dialogfeld Freigabe annehmen wird geöffnet. Es beschreibt die Ihnen zugewiesene Rolle.

  2. Bevor Sie die Freigabe annehmen, können Sie den Namen des Notizbuchs ändern.

  3. Aktivieren Sie Standardmail senden, um eine Bestätigung an den Besitzer des Notizbuchs zu senden (optional).

  4. Klicken Sie auf Ja. Das Notizbuch wird in die Liste Ihrer Notizbücher aufgenommen. Die Mail wird in Ihren Papierkorb verschoben.
