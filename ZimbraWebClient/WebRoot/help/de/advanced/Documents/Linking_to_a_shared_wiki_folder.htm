Link zu einem freigegebenen Notizbuch einrichten

Wenn Sie die Mailbenachrichtigung annehmen, dass Ihnen der Zugriff auf ein Notizbuch gewährt wurde, können Sie auch Links auf solche freigegebenen Notizbücher einrichten. Hierfür arbeiten Sie im Dialogfeld Link zu freigegebenem Notizbuch. Dies ist z. B. dann erforderlich, wenn es in Ihrer Firma öffentliche Notizbücher gibt, auf die Sie zugreifen möchten.

Hinweis: Ihre Zugriffsrechte werden von dem Besitzer des Notizbuchs festgelegt.

  1. Klicken Sie unter „Notizen“ mit der rechten Maustaste auf die Notizbuchkopfzeile und wählen Sie Link zu freigegebenem Notizbuch. Ein Dialogfeld wird aufgerufen.

  1. Um das freigegebene Notizbuch hinzuzufügen, müssen Sie die Mail-Adresse des Notizbuchbesitzers kennen. MMustermann@Beispiel.de hat beispielsweise das Notizbuch „Projekte“ mit einem weiteren, untergeordneten Notizbuch „Projekt 1“ angelegt. MMustermann erteilt Ihnen dann die Freigabe für „Projekt 1“. Um den Zugriff auf „Projekt 1“ einzurichten, geben Sie folgende Informationen ein:

Mail: MMustermann@Beispiel.de

Pfad: Projekte/Projekt1

Name: Der Name wird automatisch durch die Mail-Adresse und den Pfad vervollständigt. Diese Eingabe können Sie ändern.

Farbe: Wählen Sie eine Farbe in der Liste aus (optional).

  1. Klicken Sie auf OK. Das Notizbuch wird in die Liste Ihrer Notizbücher aufgenommen.
