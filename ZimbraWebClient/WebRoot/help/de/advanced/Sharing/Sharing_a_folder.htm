Ordner freigeben

Sie können die folgenden Objekte für andere Nutzer Ihrer Organisation, für externe Nutzer oder öffentlich freigeben: Mail, Adressbuch, Kalender, Aufgaben und den Ordner "Dateien".  Klicken Sie mit der rechten Maustaste auf einen Ordner. Geben Sie in dem Dialogfeld, das nun angezeigt wird, die Zugriffsberechtigung für den Nutzer an.  Sie können die Freigabe internen Nutzern erteilen, die vollständigen Manager-Zugriff auf die Ordner erhalten können, Sie können die Freigabe externen Nutzern erteilen, die sich den Ordnerinhalt nach Eingabe eines Passworts ansehen können, und Sie können öffentlichen Zugriff auf die Ordner erteilen, so dass jeder, der die URL kennt, sich den Inhalt der Ordner ansehen kann.

Wenn ein Ordner für interne Nutzer freigegeben ist, wird eine Kopie des freigegebenen Ordners in der Liste der Ordner dieser Nutzer in deren Übersichtsfenster abgelegt. Bei Auswahl des Ordners wird der Inhalt im Inhaltsfenster angezeigt.

Auf der Seite Freigabe unter „Einstellungen“ finden Sie Hilfe bei der Verwaltung Ihrer freigegebenen Objekte. Diese Seite enthält die Ordner, die Sie für andere Nutzer freigegeben haben, sowie Ordner, die andere Nutzer für Sie freigegeben haben.

Zugriff auf Ihre Ordner gewähren

Sie können Ordner für interne Nutzer oder Gruppen, externe Nutzer oder für die Öffentlichkeit freigeben.

Interne Nutzer oder Gruppen

Sie können anderen Nutzern oder Gruppen in Ihrer Firma die Freigabe erteilen. Wenn Sie internen Nutzern oder Gruppen die Freigabe erteilen, geben Sie die Mail-Adressen ein und wählen die Art der Zugriffsberechtigung aus:

  • Leseberechtigung. Der Nutzer kann sich den Inhalt des Ordners ansehen, er kann den Inhalt jedoch nicht ändern.  

  • Manager. Der Nutzer hat die Berechtigung, den Inhalt anzuzeigen, zu bearbeiten und zu löschen sowie Kalendereinladungen oder Anfragen für die Freigabe von Ordnern anzunehmen oder abzulehnen.

  • Admin. Der Nutzer hat die volle Berechtigung, den Ordner zu verwalten und den Inhalt anzuzeigen, zu bearbeiten und zu löschen, Einladungen anzunehmen oder abzulehnen, Ihren Ordner für andere Nutzer oder Gruppen freizugeben sowie den Zugriff auf einen freigegebenen Ordner zu entziehen.

  • Keine. Diese Option dient dazu, den Zugriff auf den freigegebenen Ordner eines Nutzers vorübergehend zu deaktivieren, ohne die Freigabeberechtigungen zu widerrufen. Der Nutzer, dem die Freigabe erteilt wurde, hat den Ordner noch in seiner Liste, kann jedoch keine Aktivitäten im freigegebenen Ordner anzeigen oder verwalten.

Externe Nutzer

Sie können externen Nutzern die Freigabe erteilen, die sich den Ordnerinhalt nach Eingabe eines Passworts ansehen können. Externe Nutzer können keine Änderungen an Ihrem Ordner vornehmen. Sie erstellen das Passwort.

Öffentliche URL

Jeder, der die URL des Ordners kennt, kann sich den Ordnerinhalt ansehen. Änderungen am Ordner können von öffentlichen Nutzern nicht vorgenommen werden. Wenn Sie die öffentliche Freigabe auswählen, wird die URL, die hierfür im Dialogfeld „Eigenschaften für Freigabe“ angegeben wird, der Öffentlichkeit zugänglich gemacht.

Zugriff auf freigegebene Ordner entziehen

Sie können die Zugriffsrechte bearbeiten und jederzeit Berechtigungen widerrufen. Hierfür rufen Sie die Seite „Eigenschaften bearbeiten“ des Ordners auf oder wählen „Einstellungen > Freigabe“.

Auf einen für Sie freigegebenen Ordner zugreifen

Wenn für Sie ein Ordner freigegeben wurde, erhalten Sie eine Benachrichtigung per Mail. Wenn Sie in der Mail auf „Annehmen“ klicken, wird der Link auf den Ordner ins Übersichtsfenster Ihrer Mailbox eingefügt. Wenn Sie auf den Ordner klicken, wird Ihnen der Inhalt des freigegebenen Ordners angezeigt.

Sie können auch in der Übersicht auf Freigaben suchen klicken, um Ordner, die für Sie freigegeben wurden, schnell zu finden und hinzuzufügen.  Wenn sich das Dialogfeld Freigabeordner hinzufügen öffnet, werden darin Objekte, die Ihnen über Verteilerlisten freigegeben wurden, angezeigt. In das Feld Freigaben suchen können Sie eine Mail-Adresse eingeben, um festzustellen, ob Ihnen ein Objekt von einer bestimmten Adresse freigegeben wurde.
