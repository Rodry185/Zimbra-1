Getting Started with @CollaborationProductShortName@ Standard Edition

The standard Web client offers you an easy to use email account with the following features.

Note: Your account may not include all the features mentioned. Check with your system administrator to see which features are enabled for your account.

See Navigating around your mailbox.

Email

    • Compose and send new mail messages

    • Read and reply to mail messages

    • Track mail exchanges using the Conversation feature or optionally switch to a traditional message view

    • Include attachments with your message

    • Forward mail messages to one or more recipients

    • Create your own folders to organize mail

    • Create tags with which to organize your mail

    • Create filters to route incoming mail to designated folders

Address Book

    • Create and manage multiple address books

    • Import and export contact lists

Calendar

    • Create and manage multiple calendars

    • Create appointments, meetings, and events

    • Import and export other calendar programs

Tasks

    • Create multiple Tasks lists

    • Create to-do tasks

    • Add attachments to you task

    • Manage a task, set the priority and track the progress

Briefcase

    • Upload any type of file to folders in your briefcase so that you can access the file whenever you log on to your account

    • Create folders to organize files that you upload

    • Link to other briefcase

Preferences

    • Manage how your mailbox works

    • Configure your account settings including the email address that appears in the From field and whether to automatically include a signature in your email messages

    • Configure your email signatures

    • Create mail filters

    • View shortcut key combinations available for quick navigation within your mailbox

 

In addition, you can search for email messages, personal contacts, calendars, and task lists from any view.
