Sorting tasks

You can sort tasks in your task list to quickly see tasks by status. On the Tasks toolbar click Sort by and select one of the following:

  • Not started.

  • Completed

  • In Progress

  • Waiting for someone else

  • Deferred

  • All  All shows all tasks no matter the status.

  • To-Do List  To-Do List shows tasks that with the status of In Progress, Not Started or, if the task is overdue, Waiting for someone else. .
