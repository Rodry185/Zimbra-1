Searching for messages sent to specific recipients

You can search for messages by to whom the messages was sent in one of the following ways:

  • From the Sent or Drafts folders

  • Using Advanced Search, Basic Search To/Cc field

When you search within a folder, only results for messages within that folder are displayed.  When you use the Advanced Search option, you can select which folders to search. Results include messages from the folders selected for the search.

Search Sent and Drafts folders

  1. In the Sent and Drafts folders, select a message with the name of the recipient you want to find.

  2. Click To in the header. A select an Address dialog displays.

  3. Click OK. The search is executed and messages sent to that recipient display in the Content pane.

Using Advanced Search

  1. Click Advanced to open the Advanced search area.

  2. In Basic Search dialog To/Cc field enter the name to search.  The more specific the name the better the results. By default the Inbox is searched.

  3. To search other folders, click Folder, and select the folders to search.

As soon as you enter a criteria in a search pane, or as soon as you have made any selection change in any other search pane, the search executes.

Items matching the search criteria are listed in the lower right. If the Conversation feature is enabled, conversations containing messages that match the search criteria are listed.
