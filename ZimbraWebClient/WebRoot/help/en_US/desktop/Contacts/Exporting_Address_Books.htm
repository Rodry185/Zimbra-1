Exporting address books

Your  contact lists are exported and saved in a comma-separated (.csv) text file format.

To export your address book:

  1. Select Preferences and click Import / Export .

  2. In the Export section, select the type as Contacts.

  3. In the Source field, click to select the contacts list to be exported.

  1. Click Export.

  2. Select where to save the exported file and click OK. To import them into another contact manager program, see documentation for that program.
