Manage Data in Your Yahoo! Zimbra Desktop Accounts

You can manage data for each of your accounts. The Manage Data import and export functions can be used to copy all or part of your account data.

Export Data

Export can be used to export the content of your account or specific types of data.  You can set up the export to

  • Limit the data types to only include any of  email, address book, calendar, task lists, Documents and Briefcase content.

  • Export from specific folders only

  • Export specific files

Import Data

Import can be used to add the content of one account to another account or to recover information that may have been deleted. You can set up what type of information to import and how the data should be organized after being imported.

  • You can select the type of files to import: email, address book, calendar, tasks, documents, briefcase.

  • You can select to import to a specific folder in the account or to create a new folder

 

Reset  

Reset deletes all data in the account. Email messages, contact lists, calendars,  Documents Notebook pages, files saved to Briefcase, and  data archived to the Local Folder. Reset removes the files from the computer not from the server. The next time you sync you account, the data on the server is downloaded. Except for Zimbra Web Client accounts, files that you saved to the Local Folder, in Briefcase, in Documents, and in Tasks are not synced to the server.
