Version Control of Files in Briefcase

Version control lets you can manage changes to files in Briefcase. The version number is displayed in the document list. When you view by version history, you can see previous versions of the file and the author. You can restore a previous version of the file and you can delete versions you no longer need.

Note: Each version of a file counts against your account quota.  

Files that you upload can be checked out to prevent more than one person working on the same file in Briefcase. These files are downloaded to your desktop. When you upload the updated file, it is checked in and the version is changed.

How to Check Out a File  

  1. Right-click on the file and select Check Out File.

  2. In the Download files dialog, select Check out File for changes and click Download.  

The file is downloaded to your computer.
