Advanced Search

When you use Advanced Search, you can create a complex query to search for addresses by domain or server. Individual mini-search panes let you select the criteria for the search. The results of the search display in the content pane and the total number of items found are displayed on the right side of the toolbar.

The advanced search allows you to search by the following criteria:

Basic Attributes. This pane sets the specific details about the address.

  • Name is the account user, resource name, or distribution list name before the @ sign in the email address.

  • Last Name is the last name of the user.

  • Display Name is the name configured as the display name, often this is the last name

  • ID is the complete mailbox ID must be entered. Use this to find out which domain or server an account is on.

  • Account Status. You can select which status to search. If you do not select a specific status, the search includes all of the statuses - active, maintenance, locked, closed, and locked out.

  • For administrators or for delegated administrators. You can easily find all administrator and delegated administrator accounts.

Object Type. Specify which address type to search for, Accounts, Distribution Lists, Aliases, and Resources. All object types are searched by default.

Domains: Specify a specific Zimbra domain. If no domain is specified, all domains are searched. If you want to search more than one domain, but not all, open a domain pane for each specific domain you want to search.

Server. All servers are listed. The default is to search all servers. Select the servers to search.

Advanced Attributes. This pane lets you search for accounts by the last log in time or for accounts that have never been logged in.

Class of Service. Specify a COS name. When you select the COS from the list, the search field shows the COS ID

As you enter criteria in the search panes, the query syntax displays in the search field.

To use Advanced search

  1. Select Advanced Search in the Search bar. The Basic Attributes pane automatically displays.

  2. Open other mini-panes to build your query as necessary and enter your criteria.

  3. Click Search in the Search bar. The results display in the Content pane. The number of results found is displayed on the right of the content pane toolbar.

  4. Close the Advanced Search to enlarge the Content pane. If you want to save this search, before closing the Advanced Search, click the save icon in the search bar.
