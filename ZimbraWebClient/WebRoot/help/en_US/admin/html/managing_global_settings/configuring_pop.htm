Configuring POP

You can enable the POP3 service and set the following from the Global Settings, POP tab. You can also enable the POP service from a Server's POP tab. The server settings overrule the global settings.

 

Enable POP3  service

Allows POP connection to the Zimbra servers.

Enable SSL for POP service

Enables encrypted connections.  (HTTPS)

Enable clear text login

Enables connections that are not encrypted.  (HTTP)

POP3 port number

The default port number for HTTP connections is 110. If you use IMAP/POP proxy, the port number is 7110.

POP3 SSL port number

The default port number for HTTPS connections is 995. If you use IMAP/POP proxy, the port number is 7995.

POP3 proxy port number

This is configured if IMAP/POP proxy is set up. The required port number is 110.

POP3 SSL proxy port number

This is configured if IMAP/POP proxy is set up. The required port number is 995.

Number of threads

Number of threads that should be available. The global setting is not set. The server default setting is 100.

 

The following additional POP information can be configured.

Address

The local IP address the POP service should bind to.

Advertised name

The name to identify the server when a POP client connects.  The default is the local server name.
