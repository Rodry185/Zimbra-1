Configuring IMAP

You can enable the IMAP service and set the following from the Global Settings, IMAP tab. You can also enable the IMAP service from a Server's IMAP tab. The server settings overrule the global settings.

 

Enable IMAP service

Allows IMAP connection to the Zimbra servers.

Enable SSL for IMAP service

Enables encrypted connections.  (HTTPS)

Enable clear text login

Enables connections that are not encrypted.  (HTTP)

 

IMAP port number

The default port number for HTTP connections is 143. If you use IMAP/POP proxy, the port number is 7143.

 

IMAP SSL port number

The default port number for HTTPS connections is 993. If you use IMAP/POP proxy, the port number is 7993.

 

IMAP proxy port number

This is configured if IMAP/POP proxy is set up. The required port number is 143.

 

IMAP SSL proxy port number

This is configured if IMAP/POP proxy is set up. The required port number is 993.

 

Number of threads

Number of threads that should be available. The global setting is not set. The server default setting is 200.
