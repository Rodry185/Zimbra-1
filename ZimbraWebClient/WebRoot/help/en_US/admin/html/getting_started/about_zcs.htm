About Zimbra Collaboration Server

The Zimbra Collaboration Server is a full-featured messaging and collaboration server. ZCS offers reliable, high-performance service along with advanced mail features, including advanced search capabilities, ability to sort mail by conversations, and tags to organize mail, shared and delegated calendar that supports subscribing to external calendars, tasks lists, and document management, as well as standard mail features such as address books, user-defined folders, and user-defined mail filtering.
