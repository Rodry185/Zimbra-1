How to Configure GAL

The Global Address List (GAL) is configured on a per-domain basis. The GAL Configuration Wizard directs you through the steps to configure GAL.

GAL is configured when a new domain is created and can be edited from the Domain toolbar.

To configure the GAL setting

  1. To edit the GAL for a domain, select the domain, and on the toolbar click Configure GAL.

  2. Select the GAL mode, Internal, External, or  Both.

  3. Enter the value for the maximum results that should be returned from a GAL search. The default is 100.

    • The maximum results settings can also be set in Global Settings. The value entered in Global Settings becomes the default for domains if you do not set it here.

  4. To create a GAL sync account, enter an account name for the GAL sync account name; the recommended default is galsync.   If the account does not exist, a new account is created.  This account is used only to sync the LDAP contacts to the GAL sync account's address book.

  5. Server type can be ether LDAP or Active Directory.

    • If LDAP is selected, enter the LDAP URL and enter the port as 389.

    • If Active Directory is selected, enter the Active Directory LDAP URL and enter the port as 3268.

  6. If you check Use SSL, the server port is 636.

  7. To add additional LDAP servers, click Add. A new LDAP URL field displays.

  8. In the LDAP filter: field, enter the search expression that is used.  See Configure the Global Address List.

  9. The LDAP search base is configured by default. Click Next.

  10. To use DN to bind to the external server, check Use DN/Password to bind to external server.

    Enter the Bind DN and Bind Password. Click Next. The GAL Sync Settings dialog displays. You can set GAL search settings and GAL sync settings separately.

  11. If you want to set your sync settings different from your GAL search settings, uncheck the box Use GAL search settings for GAL sync, and continue to complete the GAL sync settings in this and the next dialog.

    If the GAL Search and GAL sync settings are the same, on the Gal Sync Settings dialog, click Next.

  12. The GAL Settings Summary page displays the information you entered. If the information is correct, to test the GAL settings, enter a name from the directory in the Please provide a search term field. Click Test.

  13. The GAL Test Results page displays the test results. If the results failed, click Previous, to resolve the issues described in the Details text box. Click Finish, if the test past.
