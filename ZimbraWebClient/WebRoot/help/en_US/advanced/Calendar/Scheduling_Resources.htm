Scheduling resources

Resources are considered a location (meeting room, building, or other location) or equipment (audio/visual) that can be scheduled for a meeting. When you set up a meeting, to schedule a resource, you select the resource and "invite" it to the meeting.  The resource receives the invite and if it is free, accepts the meeting.  

Administrators set up resource accounts and configure the resource's scheduling policy. A meeting can be declined if the resource already has a meeting scheduled for that time.  In some cases, the resource scheduling policy is set up to allow a resource to accept dates in recurring meeting even if there are conflicts.  If you invite a resource to a recurring meeting and some of the dates are already scheduled, a Conflicting Resources dialog opens showing the conflicting dates and times.  You also receive an email showing which dates conflict.
