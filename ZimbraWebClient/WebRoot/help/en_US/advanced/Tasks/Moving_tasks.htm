Moving tasks

You can move a task to another list in one of the following ways:

  • Right-click the task, select Move and in the Move Task dialog, select the destination task list or create a new list. Click OK.

  • Select the task to move, hold down the left mouse button and drag the task to the lists in the Overview pane.

  • Open the task page and in Task Lists in the Details section select the destination task list.
