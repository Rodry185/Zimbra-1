Trabalhando em documentos Zimbra

O aplicativo Documentos Zimbra é um programa de colaboração e compartilhamento de documentos que proporciona ao usuário um ponto central no Cliente Zimbra para Web  para desenvolver, organizar e compartilhar informações. Imagens, planilhas e outros objetos de conteúdo web podem ser incorporados às páginas do Documento. Documentos externos também podem ser carregados. As páginas são exibidas em seu navegador.

Documentos Zimbra é composto por notebooks com páginas individuais que são organizadas em um índice. Um notebook, chamado Notebook, já foi criado para você. Você pode criar outros notebooks e também criar notebooks dentro de um outro para organizar suas informações.  Você cria páginas em um notebook e desenvolve o conteúdo usando o editor de texto com formatação usando recursos HTML.

Os notebooks do Documentos podem ser compartilhados com outras pessoas em sua organização e fora dela. Você pode conceder permissões a outra pessoa   para criar e editar páginas em um notebook ou apenas visualizar as páginas.

O que você pode fazer no Documentos

  • Criar vários notebooks

  • Criar notebooks dentro de outros

  • Criar páginas usando um editor de texto com formatação ou HTML

  • Carregar e criar links para arquivos e imagens para páginas diferentes

  • Compartilhar notebooks, dando permissão somente para visualização ou para visualizar, adicionar, editar e excluir

  • Buscar em todos os notebooks, páginas e anexos

  • Enviar um link para notebook por e-mail rapidamente a partir do índice
