Teclas de atalho de teclado do ZWC

As teclas de atalho podem ser usadas para navegar no Cliente Zimbra para Web (ZWC) e você pode criar suas próprias combinações de tecla de atalho. Vá para Preferências>Atalhos para ver a lista de teclas de atalho e para criar suas próprias teclas de atalho. As quatro abas exibem o seguinte:

  • A lista de atalhos inclui uma lista de atalhos para todos os aplicativos

  • Os atalhos da pasta E-mail possibilitam que você crie e atribua atalhos para acessar suas pastas de e-mail

  • Os atalhos de busca salva possibilitam que você crie e atribua atalhos às buscas salvas

  • Os atalhos para marcadores possibilitam que você crie e atribua atalhos aos marcadores criados
