Salvando o conteúdo de sua conta

É possível exportar todos os dados de sua conta, inclusive mensagens de e-mail e anexos, contatos, agenda, tarefas, notebooks com documentos e pastas Porta-arquivos, ou é possível exportar itens específicos da conta. Depois, você pode salvar os dados em seu computador ou em outro local para backup. Os dados da conta são salvos em um arquivo Tar-GZip (tgz), possibilitando que ele seja importado para restaurar sua conta.

  • Ao exportar pastas individuais de contatos, elas são salvas como arquivos .csv; já pastas individuais de agenda são salvas no formato .ics.

Os dados são copiados, e não removidos, de sua conta.

Você exporta o arquivo em Preferências > aba Importar/Exportar.

Para exportar dados

  1. Na aba Preferências  , clique na pasta Importar/Exportar.

  2. É possível exportar todos os dados da conta ou selecionar o tipo de pasta para exportar.

    • Para exportar todos os dados da conta, selecione Conta. Observação: as pastas Lixeira e Spam não são exportadas.

    • Para exportar uma lista de contatos ou uma agenda específica, selecione essa opção. Depois de selecionar uma agenda ou um contato para exportar, selecione o tipo de formado da exportação. Consulte Exportando itens de sua conta.

    • Se você quiser selecionar vários tipos específicos de itens para exportar de uma só vez, marque Configurações avançadas para expandir a exibição. Selecione as opções dos itens de dados a exportar. Todos os itens do tipo escolhido dentro de todas as pastas dentro da pasta selecionada são incluídos. Se você quiser dados específicos, poderá usar o campo de filtro de busca para especificar o tipo de dados a exportar. Consulte Descrição da linguagem de critérios e Exemplos de busca.

  1. Clique em Exportar e selecione o local para salvar o arquivo.

  2. Clique em OK.

  3. O arquivo com os dados exportados da conta pode ser visualizado com um programa de compactação como o WinRAR.

 
Para importar dados salvos da conta

Uma conta é restaurada com base no arquivo que você exportou.  

  1. Na aba Preferências  , clique na pasta Importar/Exportar.

  2. Na seção Tipo de importação, selecione o tipo do item a importar, seja Conta, Agenda ou Contatos. Se você importar agendas ou contatos, especifique o formato da importação. Consulte Importando contatos. Quando Conta é selecionado, apenas o arquivo tgz que você já exportou pode ser importado para a conta.  

  1. Clique em Procurar para encontrar o arquivo que deve ser importado.

  2. Selecione o destino.

  3. Se você estiver importando os dados da conta, selecione a opção apropriada nos campos Resolver duplicatas.

    • Ignorar. Os arquivos duplicados não serão resolvidos. Todos os arquivos são incluídos de volta, e a versão existente de um item existente é mantida.

    • Substituir. Os itens que sofreram modificação desde a exportação dos dados da conta são revertidos para a versão do arquivo exportado. Itens antigos são incluídos de volta. Itens novos, que foram adicionados a sua conta desde a criação do arquivo exportado, são mantidos.

    • Restaurar. Quando selecionado, todos os itens do tipo escolhido dentro da pasta selecionada são removidos antes de importar os novos itens.

  1. Se você quiser selecionar tipos específicos de dados do arquivo tgz, marque Configurações avançadas para expandir a exibição e selecionar os tipos de itens para restaurar.

Selecione os tipos específicos de itens do arquivo tgz salvo para importar na seção Tipos de dados. Ao clicar em Importar, apenas os tipos de itens selecionados são importados.

  1. Clique em Importar.
