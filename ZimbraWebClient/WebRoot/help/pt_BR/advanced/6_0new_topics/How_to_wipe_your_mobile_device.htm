Como limpar seu dispositivo móvel

Caso seu dispositivo móvel seja perdido ou roubado, você pode remover todo o conteúdo dele através do link Limpar dispositivo na página Dispositivos móveis de Preferências. Este comando apaga todos os dados do dispositivo e restaura-o para as configurações de fábrica.

  1. Clique em Preferências e selecione a pasta Dispositivo móvel.

  2. Selecione o dispositivo para limpar e clique em Limpar dispositivo.  

O status apresentará Limpeza pendente. O servidor está aguardando que o dispositivo seja conectado a ele para iniciar a solicitação. Quando o dispositivo é conectado ao servidor, ele é limpo. O status muda para Limpeza concluída.

  • Observação: você pode cancelar a solicitação de limpeza do dispositivo contanto que o status seja Limpeza pendente.

Além de iniciar a limpeza do dispositivo no servidor, o ZCS comporta configurações de políticas para celular de modo a impor políticas de segurança a um dispositivo. Uma dessas políticas estabelece os requisitos de bloqueio do nome de identificação pessoal (PIN). A configuração de bloqueio do PIN determina quantas vezes alguém pode tentar acessar um dispositivo sem conseguir inserir a senha correta antes que seja realizada uma limpeza iniciada localmente do dispositivo. Isso apaga todos os dados do dispositivo. Esta configuração de política só pode ser gerenciada pelo administrador e não pode ser modificada por usuários.
