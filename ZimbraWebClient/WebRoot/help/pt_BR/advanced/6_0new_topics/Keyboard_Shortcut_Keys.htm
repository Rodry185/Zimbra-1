Teclas de atalho do teclado

As teclas de atalho podem ser usadas para navegar no Zimbra Web Client (ZWC).

Para ver rapidamente uma lista de teclas de atalho, digite Ctrl Q. A lista de atalhos de teclado exibem as teclas de atalho por aplicativo.  A lista de atalhos para todos os aplicativos são teclas de atalho que podem ser usadas para qualquer aplicativo.  Por exemplo, é possível digitar gc para abrir rapidamente sua agenda ou sua lista de tarefas.

Também é possível imprimir a lista de atalhos. Vá para Preferências>Pasta de atalhos .
