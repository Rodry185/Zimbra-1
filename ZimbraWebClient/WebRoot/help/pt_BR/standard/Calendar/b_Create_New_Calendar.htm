Criando novas agendas

Depois de criar uma agenda, você pode criar agendas adicionais para ter controle dos diferentes tipos de funções.  Por exemplo, é possível criar uma agenda para trabalho, para uso pessoal ou para atividades da empresa, além de assinar o serviço de agenda que está disponível na Internet e vincular-se à agenda que está sendo compartilhada.

Para ver duas ou mais das suas agendas lado a lado, selecione Visualização de programação.

Para criar uma agenda

  1. Selecione a aba Agenda.

  2. No painel Visão geral da agenda, clique em editar.

  3. Na barra de ferramentas, clique em Nova agenda.

  4. Digite o nome da agenda.

  5. Selecione uma cor para exibir as atividades da Agenda. Isso permite codificar as agendas por cor na caixa de e-mails para que essas atividades sempre sejam exibidas na cor escolhida.

  6. Se você não quiser que as atividades da agenda sejam visualizadas por outras pessoas, marque Excluir esta agenda ao informar horários livres/ocupados.

  7. Por padrão Verificado na UI está ativado. As atividades dessa agenda serão exibidas quando você abrir a visualização Agenda.  Para não ver esta agenda com outra, desmarque a caixa. Nesse caso, para ver as atividades dessa agenda, selecione a agenda e abra-a.

  8. Clique em Criar agenda.

Para assinar outro serviço de agenda

  1. Selecione a aba Agenda.

  2. No painel Visão geral da agenda, clique em editar. As informações da agenda padrão são exibidas.  

  3. Na barra de ferramentas, clique em Assinar o serviço de agenda.  

  4. Digite o nome da agenda, insira uma URL de arquivo iCal válida para a agenda remota e selecione uma cor para exibir as atividades da Agenda.

  5. Se você não quiser que as atividades da agenda sejam visualizadas por outras pessoas, marque Excluir esta agenda ao informar horários livres/ocupados.

  6. Por padrão Verificado na UI está ativado. As atividades dessa agenda serão exibidas quando você abrir a visualização Agenda.  Para não ver esta agenda com outra, desmarque a caixa. Nesse caso, para ver as atividades dessa agenda, selecione a agenda e abra-a.

  7. Clique em Criar agenda.
