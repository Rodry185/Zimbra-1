Criar e salvar um critério de busca

Se você usar o mesmo critério de busca com freqüência, convém salvá-lo na pasta Busca. Quando você quiser fazer a busca novamente, basta clicar na pasta de busca salva para executá-la, exibindo os resultados no painel Conteúdo.

Para criar e salvar um critério de busca

  1. No painel Visão geral > cabeçalho Busca, clique em editar. O painel Visão geral é atualizado de modo a mostrar todas as suas pastas em ordem alfabética, e o painel Conteúdo mostra a Caixa de entrada.

  2. Na barra de ferramentas, selecione Nova busca. O formulário Nova busca aparece no painel Conteúdo.

  3. No campo Nome, dê um nome para sua busca.

  4. No campo Pasta principal, selecione onde salvar a busca. A pasta padrão é a raiz. A pasta é salva no cabeçalho Busca, painel Visão geral. Se você quiser salvar a busca em outra pasta, selecione a pasta em questão na lista. É possível identificar a pasta de busca pelo ícone de aumento.

  5. No campo Critério de busca, crie o seu critério. Consulte Descrição da linguagem de critérios para obter informações sobre a sintaxe que pode ser usada.

  6. Na caixa de diálogo Salvar busca, dê um nome para a pasta de busca e selecione o local onde criá-la. Se salvá-la na pasta raiz, a busca será exibida na seção Buscar. Se você selecionar outra pasta, a busca será salva naquela pasta.

  7. Clique em OK.

Para executar sua busca, clique na pasta desejada no painel Visão geral. Os resultados da busca são exibidos no painel Conteúdo.
