Importando iCalendars

Você pode importar agendas iCalendar (iCal) que são salvas no formato .ics. iCalendar é o padrão de troca de dados de agenda na Internet e .ics é o formato padrão das informações iCalendar.

Você pode importar sua agenda iCal para uma agenda existente ou criar uma nova e importar o arquivo .ics para ela.

Para importar uma agenda iCal:

  1. Selecione a aba Agenda.

  2. No painel Visão geral da agenda, clique em Editar.

  3. No painel Visão geral da agenda, selecione uma agenda da lista ou crie uma nova para receber o arquivo .ics importado.

  4. No campo Importar de arquivo .ICS, localize o arquivo .ics para importar.

  5. Clique em Importar.

  6. Clique em OK.
