Classification des mails

Plusieurs méthodes de classification des mails sont disponibles. Vous pouvez les utiliser, ainsi que la fonction de recherche, pour localiser et récupérer rapidement vos mails. Les méthodes disponibles sont les suivantes :

  • Filtres. Dans Préférences, >Filtres de >mails, vous pouvez créer des règles pour classer vos mails entrants selon les éléments de la ligne d'objet, l'expéditeur, le destinataire et d'autres caractéristiques.

  • Dossiers. Vous pouvez créer des dossiers afin d'organiser vos messages et transférer les mails entre les dossiers,  par exemple, en retirant un mail du dossier "À faire" et en le plaçant dans le dossier "Terminé".

  • Tags. Vous pouvez utiliser les tags comme libellés sur les mails, puis utiliser la fonction de recherche pour afficher tous les mails avec un tag spécifique.

  • Conversations. Vous pouvez regrouper les mails par conversation. Cette fonction intégrée regroupe automatiquement tous les mails envoyés et reçus avec le même objet dans une seule conversation.
