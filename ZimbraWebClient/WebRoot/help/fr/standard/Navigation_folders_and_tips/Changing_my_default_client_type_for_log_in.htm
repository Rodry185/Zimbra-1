Modification du type de client par défaut pour la connexion

Chaque fois que vous vous connectez, vous pouvez choisir la version du client Web que vous voulez utiliser, la version avancée ou standard par défaut.

  • Le type de client par défaut est configuré dans l'onglet Général, >Préférences. Il peut être Avancé ou Standard.

  • Le client Web avancé offre un ensemble complet de fonctions de collaboration Web sous Ajax. Le client Web avancé fonctionne avec les navigateurs les plus récents et les connexions Internet les plus rapides.

  • Le client Web standard est l'option idéale lorsque les connexions Internet sont lentes ou que les utilisateurs préfèrent utiliser une messagerie HTML pour naviguer dans leur boîte aux lettres.

Si votre type de client préféré n'est pas défini par défaut, vous pouvez le modifier après vous être connecté.

1. Activez l'onglet Préférences .

2.  Dans le Panneau d'aperçu, cliquez sur Général et dans le champ Options de connexion, sélectionnez Avancé ou Standard.

3. Cliquez sur Enregistrer.

Lors de votre prochaine connexion, votre sélection sera définie par défaut.
