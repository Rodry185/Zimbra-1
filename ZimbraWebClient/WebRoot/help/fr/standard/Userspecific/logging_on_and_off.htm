Connexion et déconnexion

Vous devez vous connecter et être authentifié pour pouvoir utiliser @ClientNameLong@.  Deux versions du @ClientNameLong@ sont disponibles : avancée et standard.

  • Le client Web avancé offre un ensemble complet de fonctions de collaboration Web sous Ajax. Le client Web avancé fonctionne avec les navigateurs les plus récents et les connexions Internet les plus rapides.

  • Le client Web standard est l'option idéale lorsque les connexions Internet sont lentes ou que les utilisateurs préfèrent utiliser une messagerie HTML pour naviguer dans leur boîte aux lettres.

Lorsque la boîte de dialogue de connexion apparaît, vous pouvez choisir la version à laquelle vous voulez vous connecter.

Remarque : Dans la plupart des cas, le client Web avancé est le client par défaut et il s'ouvre automatiquement lorsque vous vous connectez.  Si vous voulez que le client Web standard soit votre client par défaut, affichez la page Préférences >générales après vous être connecté.

Pour se connecter

  1. Ouvrez une fenêtre du navigateur, puis saisissez l'URL fournie par votre administrateur Zimbra.

    Par exemple :

    http:// mail.domain.com/ zimbra/mail

  2. Lorsque l'écran de connexion apparaît, saisissez votre nom d'utilisateur. Un message peut vous demander de saisir votre adresse mail complète comme nom d'utilisateur et votre mot de passe.  

Exemple d'adresse mail complète, tapez :

myra@exemple.com
myra123

Votre mot de passe apparaît sous la forme ******, afin de protéger votre confidentialité.

  1. Pour que l'ordinateur garde en mémoire votre session jusqu'à son expiration ou votre déconnexion, cochez Se rappeler de moi.

Si vous cochez cette option, vous n'aurez pas à vous reconnecter chaque fois que vous relancez le navigateur pendant la journée.  Lorsque vous sélectionnez cette option et que vous ne vous déconnectez pas, votre session reste active jusqu’à l’expiration de la session. L'administrateur système détermine la durée d'activité d'une session.

Remarque : Si vous laissez votre client Web ouvert dans votre navigateur et que vous n'accédez pas à votre boîte aux lettre pendant un certain temps, votre session peut expirer et vous devez donc vous reconnecter.

  1. Si vous ne voulez pas utiliser le type de client configuré par défaut, modifiez la version affichée.

  2. Cliquez sur Connexion.

 

Pour se déconnecter :

Pour empêcher les autres utilisateurs de se connecter à votre compte de messagerie, utilisez toujours le lien Déconnexion pour fermer votre session Zimbra.
