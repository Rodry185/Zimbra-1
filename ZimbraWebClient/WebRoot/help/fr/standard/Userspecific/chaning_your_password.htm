Modifier votre mot de passe

L’administrateur attribue le mot de passe initial. Selon votre configuration, il est possible que vous puissiez changer votre mot de passe. Si c’est le cas, l’option Changer le mot de passe apparaît dans les Préférences, dans l’onglet Général.

Pour changer votre mot de passe :

  1. Cliquez sur l’onglet Préférences .

  2. Sélectionnez l’onglet Général puis dans Options de connexion, cliquez sur Changer le mot de passe.

  3. Saisissez votre ancien mot de passe, votre nouveau mot de passe, puis saisissez à nouveau votre nouveau mot de passe pour le confirmer. Remarque : Votre mot de passe ne peut pas contenir de caractères accentués, comme par exemple : ñ, é, ó, etc.

  4. Cliquez sur Changer le mot de passe. Votre mot de passe a été changé.

Il est possible que votre société dispose de règles spécifiques pour les mots de passe, par exemple : le nombre de caractères compris dans le mot de passe, la fréquence de changement de votre mot de passe, et la fréquence avec laquelle vous pouvez réutiliser le même mot de passe. Contactez votre administrateur système si vous avez des difficultés pour changer votre mot de passe.
