Créer et enregistrer une requête de recherche

Si vous exécutez régulièrement la même requête, vous pouvez l'enregistrer dans votre dossier Recherche. Par la suite, vous pouvez lancer cette requête plus rapidement en cliquant sur le nom du dossier qui la contient : les résultats s'affichent dans le Panneau de contenu.

Pour créer et enregistrer une recherche

  1. Dans le Panneau d'aperçu, sous l'en-tête >Recherche, cliquez sur modifier.  Le Panneau d'aperçu affiche les dossiers disponibles (présentés dans l'ordre alphabétique) et le Panneau de contenu affiche la liste des mails présents dans la Boîte de réception.

  2. Dans la barre d'outils, sélectionnez Nouvelle recherche : le formulaire de recherche apparaît dans le panneau "Contenu".

  3. Dans le champ Nom, entrez le nom que vous souhaitez affecter à cette recherche.

  4. Dans le champ Dossier parent, sélectionnez le dossier dans lequel cette recherche sera enregistrée  (par défaut, le dossier racine). Le nom de ce dossier apparaît sous l'en-tête Recherche du Panneau d'aperçu.  Si vous préférez enregistrer cette recherche dans un dossier autre que root/racine, sélectionnez ce dossier dans la liste. Le dossier de recherche est identifié par l'icône de loupe.

  5. Dans le champ Requête, tapez les termes de la requête. Pour plus de détails sur la syntaxe des requêtes, voir cette page.

  6. Dans la boîte de dialogue Enregistrer la recherche, tapez le nom de votre choix pour le dossier qui recevra cette recherche et spécifiez ou sélectionnez l'emplacement dans lequel ce dossier sera créé.  Si vous enregistrez dans le dossier racine, la recherche apparaît dans la section Recherche. si vous sélectionnez un autre dossier, la recherche est enregistrée dans ce dossier.

  7. Cliquez sur OK.

Pour lancer (ou relancer) la recherche, cliquez sur son dossier dans le panneau "Aperçu". Les résultats de la recherche apparaissent dans le Panneau de contenu.
