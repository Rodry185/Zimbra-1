Filtrer vos mails

Le filtre applique un ensemble de règles de correspondance au courrier entrant, puis exécute une action spécifiée.

Vous pouvez filtrer vos mails entrants pour les trier dans des dossiers, les marquer automatiquement, les faire suivre, ou les supprimer. Par exemple, vous pouvez créer une règle de la manière suivante :

  • Tous les mails envoyés par votre directeur sont rangés dans le dossier "Directives commerciales".

  • Tous les mails contenus dans la liste de diffusion "événements de la société" sont marqués avec le tag "Événements".

Pour créer ou modifier une règle de filtre :

  1. Cliquez sur l'onglet Préférences .

  2. Activez le dossier Filtres de mails, puis cliquez sur Nouveau filtre.

  3. Dans Nom du filtre, saisissez le nom de la règle.

  4. Dans la zone Si --- les conditions ci-dessous sont satisfaites, sélectionnez une condition.  Vous pouvez maintenant créer la condition.

Vous pouvez répéter les étapes suivantes afin de configurer plusieurs conditions et actions dans un seul filtre.

  • Dans la première liste déroulante, choisissez une condition à utiliser.  

  • Choisissez une méthode de comparaison . Les options indiquées dépendent de la sélection effectuée dans la zone déroulante précédente. Par exemple, "est" ou "n'est pas" peuvent s'afficher.

  • Saisissez un mot ou un groupe de mots à comparer dans la zone du texte.

Cliquez sur ajouter pour ajouter des conditions. Vous pouvez continuer d'ajouter des conditions ou passer à l'étape suivante, à savoir ajouter une ou plusieurs actions.

  1. Dans le champ Exécuter les actions suivantes, choisissez une action dans le menu déroulant.

  • Spécifiez un dossier ou le nom d'un tag, si nécessaire. Ce que vous spécifiez dépend de l'action que vous choisissez.

  • Cliquez sur ajouter pour ajouter des actions. Vous pouvez continuer d'ajouter des actions ou cliquer sur OK pur arrêter.

  1. Cochez la case actif à côté de Nom du filtre pour activer la règle de filtre.

Le filtre sera appliqué de manière automatique dès la réception de nouveaux mails.

Pour plus d'informations sur les conditions et les actions, voir à la section Conditions et actions de filtre prises en charge.
