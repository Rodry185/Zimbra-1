Création d'un mail

  1. Pour créer un mail, cliquez sur Écrire dans la barre d'outils des mails. Un nouveau panneau de création de mail s'affiche.  

  2. Saisissez l'adresse mail de la personne ou des personnes à qui vous envoyez le mail dans le champ De : . Vous pouvez également ajouter des adresses dans les champs Cc ou Cci.
    Pour rechercher l'adresse mail d'un contact, affichez la barre d'outils de création de mail et cliquez sur . Vous pouvez rechercher des noms spécifiques dans les liste de contacts. Décidez si le nom doit être inséré dans le champ À, Cc ou Cci.

  3. Saisissez l'objet du mail dans la ligne Objet .

  4. Saisissez le corps du mail dans la zone de texte située au-dessous de la ligne Objet. Voir Utilisation de l'éditeur RTF

Si vous avez configuré une signature, elle sera automatiquement ajoutée à votre mail.

  1. Pour ajouter une pièce jointe, cliquez sur Ajouter pièces jointes.

  2. Si vous souhaitez que votre mail soit affecté d'une haute priorité, modifiez la priorité sélectionnée.  Cet élément se trouve à droite de la ligne d'objet.

  3. Cliquez sur Envoyer pour envoyer le mail. Si vous avez activé l'option Garder une copie des mails dans le dossier Envoyés, une copie de votre mail sera sauvegardée dans ce dossier.

Si vous ne souhaitez pas envoyer immédiatement le mail, cliquez sur Enregistrer le brouillon. Le mail est enregistré dans le dossier Brouillons.
