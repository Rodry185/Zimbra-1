Exporter une liste de contacts

Lorsque vous exportez une liste de contacts (carnet d’adresses), celle-ci est enregistrée au format .csv.

Pour exporter une liste de contacts :

  1. Activez l’onglet "Carnet d’adresses".

  2. Dans le panneau "Aperçu du carnet d’adresses", cliquez sur Modifier.

  3. Sélectionnez le carnet d’adresses concerné.

  4. Cliquez sur le lien du champ Exporter vers .CSV. La boîte de dialogue Ouverture de <nom du carnet d’adresses> apparaît.

  5. Cliquez sur Enregistrer sur disque, puis sur OK.

  6. Sélectionnez le dossier de destination du fichier.

  7. Cliquez sur Enregistrer.
