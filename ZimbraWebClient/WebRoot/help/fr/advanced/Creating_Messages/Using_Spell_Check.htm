Utilisation du vérificateur d'orthographe

Vous pouvez vérifier l'orthographe de votre mail avant de l'envoyer. Cliquez sur Vérification orthographique dans la  barre d'outils de création.

Si vous souhaitez que tous les messages soient vérifiés automatiquement avant d'être envoyés, activez la fonction Vérification orthographique obligatoire avant tout envoi de mail dans l'onglet Préférences de >création.

  1. Cliquez sur Vérification orthographique. Les mots qui ne sont pas correctement orthographiés sont mis en évidence.

  2. Cliquez sur un mot mis en évidence. Un menu contextuel affiche les suggestions de correction.

  3. Choisissez le mot correct. Le mot est mis en évidence dans une autre couleur.

  4. Pour accepter les modifications et fermer le vérificateur orthographique, cliquez sur Reprendre la modification.

Ajout de mots à votre dictionnaire orthographique

Vous pouvez ajouter des mots à votre dictionnaire orthographique

  1. Cliquez sur Vérification orthographique. Les mots qui ne sont pas correctement orthographiés sont mis en évidence.

  2. Cliquez sur un mot mis en évidence. Un menu contextuel affiche les suggestions de correction.

  3. Si le mot est correctement orthographié, cliquez sur le mot, puis sur Ajouter. Ce mot est ajouté dans votre dictionnaire.

Vous ne pouvez pas supprimer les mots du dictionnaire une fois qu'ils ont été ajoutés.
