Affichage dans le panneau d'aperçu

À gauche de l'interface du  client Web de Zimbra, se trouvent vos dossiers, recherches, tags et Zimlets (pour le client Web avancé).

Dossiers

  • Boîte de réception : emplacement par défaut pour la réception de vos mails.

  • Envoyés : contient les messages que vous avez envoyés.

  • Brouillons : contient les mails que vous avez écrits, mais pas encore envoyés.

  • Spams : contient les messages qui ont été marqués comme spam par l'utilisateur (vous-même) ou par le système.

  • Corbeille : contient les éléments que vous avez supprimés.

  • Les Dossiers que vous avez créés sont affichés au-dessous. Si votre compte mail a été migré depuis un autre système de messagerie tel que Microsoft® Outlook, il est possible que certains dossiers présents dans l'interface Zimbra soient des dossiers du système de messagerie précédent.

Nombre de mails non lus. Le nombre entre parenthèses indiqué à côté de chaque dossier correspond au nombre de mails non lus contenus dans ce dossier. Les dossiers Tags indiquent également le nombre de mails non lus. Tout dossier comportant des mails non lus apparaît en gras.

Pour connaître le nombre de mails et la taille totale du dossier, placez le curseur sur un dossier.

Rechercher les éléments partagés est un lien qui permet de trouver et d'ajouter rapidement des dossiers qui ont été partagés avec vous.  Lorsque la boîte de dialogue Ajouter dossier partagé s'ouvre, les éléments qui ont été partagés avec vous via des listes de distribution sont indiqués. Dans le champ Rechercher les éléments partagés, vous pouvez saisir une adresse mail pour déterminer si un élément a été partagé avec vous depuis une adresse spécifique.

Recherches

Ce champ contient les demandes de recherche que vous avez créées et enregistrées pour les utiliser ultérieurement.

Tags

Cette zone indique les tags que vous avez créés. Cliquez sur un tag pour afficher rapidement tous les mails marqués avec ce tag. Pour plus d'informations, voir la section Utiliser des tags pour classer les mails.

Zimlets

Les Zimlets sont créés comme un mécanisme servant à intégrer des informations et un contenu tiers dans les fonctions @ClientNameLong@. Les Zimlets vous permettent d'interagir avec différents types de contenus dans vos mails. Selon la configuration de votre interface, il est possible que vous n'ayez pas accès aux Zimlets depuis le panneau d'aperçu.
