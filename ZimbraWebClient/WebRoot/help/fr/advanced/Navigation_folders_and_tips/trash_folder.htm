Vider votre dossier Corbeille

Chaque application dans @ClientNameShort@ dispose d'un dossier Corbeille.  Lorsque vous ouvrez la corbeille d'une application, seuls les éléments supprimés de l'application en question sont affichés. Toutefois, quand vous sélectionnez Vider la corbeille dans une application, les éléments de toutes les applications, placés dans la corbeille sont supprimés.

Avant de vider le dossier Corbeille, vous pouvez récupérer des éléments qui s'y trouvent en les faisant glisser et en les déposant dans un autre dossier.

Si Restaurer objets supprimés s'affiche quand vous cliquez avec le bouton droit de la souris sur le dossier Corbeille, vous pouvez récupérer les mails que vous aviez supprimés de la corbeille. Il est possible de récupérer les mails supprimés de la corbeille dans un délai de 30 jours après la suppression de la corbeille. Passés ces 30 jours, les élémentts sont supprimés définitivement.

Les messages laissés dans la corbeille peuvent êttre supprimés automatiuqyement et ils sont alors irrécupérables.
