Personnalisation de la page de messagerie initiale

Par défaut, votre boîte aux lettres s'ouvre avec la vue Boîte de réception. Tous les mails de votre boîte de réception s'affichent dans le panneau de contenu. Vous pouvez modifier l'affichage initial de tous les dossiers, recherches ou tags dans le panneau d'aperçu. Effectuez ces modifications à partir de Préférences>Mail,>Affichage des mails, >Recherche Mail par défaut.  

Voici des exemples de méthodes pour afficher uniquement des messages spécifiques quand vous ouvrez votre boîte aux lettres :

  • Pour afficher seulement les messages non lus quand vous ouvrez votre boîte aux lettres, tapez is :unread dans le champ Recherche mail par défaut.

  • Pour afficher les mails qui sont affectés de tags spécifiques, tapez tag:< nom de tag> dans le champ Recherche Mail par défaut. Par exemple, pour afficher seulement les messages marqués avec le tageToDo , tapez tag: ToDo.

  • Pour afficher les messages que vous avez envoyés vers un dossier spécifique via un filtre mail, tapez in:< nom de dossier> dans le champ Recherche Mail par défaut. Par exemple, pour afficher les messages dans le dossier Travail, tapez in:travail.

Remarque :  Utilisez des guillemets si le nom du dossier ou du tag comporte plusieurs mots. Par exemple, in:"Mail non lu."

Configuration du comportement de la boîte aux lettres pour actualiser

Dans la section Préférences>Mail,> Affichage des mails, l'activation de l'une des options dans Lorsque je clique sur "Actualiser" indique si la vue de votre boîte aux lettres change ou quand vous cliquez sur Actualiser.

  • Si Exécuter ma recherche par défaut est sélectionné quand vous cliquez sur Actualiser, la recherche mail décrite dans le champ Recherche Mail par défaut est exécutée.  Si votre recherche par défaut n'est pas le  dossier que vous affichez, votre affichage est automatiquement redirigé vers l'affichage de dossier par défaut. Dans la plupart des cas, il s'agit de votre boîte de réception.  

  • Si Actualiser la vue actuelle est sélectionné, quand vous cliquez sur Actualiser à partir d'un dossier, Actualiser demande des recherches de mises à jour pour l'affichage de votre dossier actuel et pour un autre nouveau mail. L'affichage de votre dossier actuel ne change pas.

Remarque :  Si un nouveau mail est reçu dans d'autres dossiers, ces dossiers apparaissent en gras et le nombre de messages non lus est mis à jour.
