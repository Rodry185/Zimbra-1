Navigation @ClientNameLong@

Lorsque vous vous connectez à @ClientNameLong@, un écran similaire à celui ci-dessous s'affiche.

 

La page @ClientNameLong@ inclut les zones suivantes :

  1. En-tête. Cette zone comprend Recherche de personnes qui vous permet d'effectuer une recherche dans l'annuaire de votre société, votre nom, ainsi que les liens Aide et Se déconnecter.

  2. Onglet d'applications. Les applications accessibles sont répertoriées dans les onglets de cette ligne.

  3. Barre de recherche. Les options Rechercher et Recherche avancée sont affichées dans cette zone. Vous pouvez également sauvegarder des recherches.

  4. Barre d'outils. Affiche les actions disponibles pour l'application que vous utilisez actuellement. Dans cet exemple, la barre d'outils de l'application Mail s'affiche.

  5. Panneau d'aperçu.Affiche vos Dossiers, y compris les dossiers système Boîte de réception, Envoyés, Brouillons, Spams, et Corbeille, ainsi que tous les dossiers personnalisés que vous avez créés, les Recherches que vous avez sauvegardées et les Tags que vous avez créés, les liens Zimlet qui ont été créés pour intégrer des applications tierces à votre boîte aux lettres. Lorsque vous utilisez l'application Calendrier, la fenêtre d'aperçu affiche votre liste de calendriers et les Zimlets.

  6. Panneau de contenu. Le contenu de cette zone change selon l'application utilisée. Dans l'affichage Boîte de réception, le panneau affiche tous les mails ou conversations dans votre boîte de réception.

  7. Sous-fenêtre de lecture. La sous-fenêtre de lecture permet d'afficher un aperçu des messages. Vous pouvez remplacer votre affichage par une vue comportant trois panneaux contenant le panneau de lecture sur le droite ou vous pouvez désactiver complètement ce panneau dans Affichage dans la barre d'outils Mail.

  8. Un mini-calendrier peut s'afficher au-dessous de la fenêtre d'aperçu. L'affichage du mini-calendrier est facultatif et peut être activé sur la page Préférences>Calendrier.
