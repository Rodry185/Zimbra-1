Marquer un rendez-vous comme étant d’ordre privé

Vous pouvez marquer un rendez-vous comme étant d’ordre privé pour masquer les détails du rendez-vous sur votre calendrier. Une icône en forme de verrou apparaît à côté du message sur votre calendrier afin d’indiquer qu’il s’agit d’un rendez-vous d’ordre privé. Les calendriers qui sont partagés avec d’autres utilisateurs indiquent le rendez-vous sur votre calendrier, mais sans afficher de détails.   

Par défaut, lorsque vous créez ou recevez un rendez-vous, celui-ci est marqué comme étant Public. Si vous souhaitez que les rendez-vous que vous créez soient toujours marqués comme étant d’ordre privé, vous pouvez modifier vos préférences de Calendrier, Visibilité par défaut des rendez-vous sur Privé.

Si vous configurez le paramètre sur Privé, toutes les invitations de calendrier entrantes sont marquées comme étant d’ordre privé sur votre calendrier.

Si votre configuration par défaut est définie sur Public et que vous recevez une invitation marquée comme étant d'ordre privé par l'organisateur, le rendez-vous apparaît comme d’étant d’ordre privé sur votre calendrier.
