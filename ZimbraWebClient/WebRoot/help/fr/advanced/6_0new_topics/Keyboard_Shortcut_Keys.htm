Touches de raccourcis clavier

Les raccourcis clavier facilitent la navigation dans le Zimbra Web Client (ZWC)

Saisissez Ctrl Q pour consulter rapidement la liste des touches de raccourcis. Cette liste contient les raccourcis clavier par application.  La liste "Toutes les applications" contient les raccourcis clavier qui fonctionnent pour n’importe quelle application. Par exemple, vous pouvez saisir gc afin d’ouvrir votre Calendrier ou gt pour ouvrir votre liste de Tâches.

Vous pouvez également imprimer cette liste de raccourcis. Affichez votre dossier Préférences>Raccourcis> .
