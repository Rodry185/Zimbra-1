Créer une présentation dans le Porte-documents (version bêta)

Lorsque vous travaillez dans le Porte-documents, vous pouvez créer une présentation (diapositives) sans avoir à faire appel à un autre logiciel. Actions possibles :

  • Sélectionner une couleur dans le menu Thèmes.

  • Insérer des éléments graphiques disponibles dans votre ordinateur.

  • Utiliser l'option "Insérer un bloc de texte" pour définir la mise en page d'une diapositive.

  • Sélectionner des polices et des styles différents.

 

  1. Dans le Porte-documents, sélectionnez Nouvelle présentation . Dans le champ qui apparaît, entrez le nom de votre choix pour cette présentation et cliquez sur OK. La première diapositive (vide) apparaît.

  2. Cliquez sur le lien Thèmes et sélectionnez la couleur de fond des diapositives.

  3. Ajoutez le contenu : Cliquez dans le bloc de texte Titre pour ajouter un titre et dans le bloc de texte Contenu pour ajouter du texte dans le bloc de texte par défaut. Autres actions possibles :

    • Redimensionner les blocs de texte et insérer d'autres blocs de texte.

    • Modifier la police, le style et la taille des caractères (barre d’outils grise affichée dans la partie supérieure des diapositives).

    • Pour ajouter un élément graphique, cliquez sur Insérer une image (les éléments graphiques insérés dans les diapositives sont également enregistrés dans le dossier Porte-documents).

    • Pour ajouter une diapositive, cliquez sur Insérer une diapo.

  4. Pour enregistrer la diapositive affichée, cliquez sur Enregistrer. Pour fermer la diapositive affichée, cliquez sur la croix (X) affichée dans l'angle supérieur droit de la fenêtre.
