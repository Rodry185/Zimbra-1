Personnaliser la vue de votre compte

Vous pouvez personnaliser la vue de votre compte : modifier la couleur de fond affichée par défaut, réagencer selon besoin les éléments affichés dans le panneau "Contenu" (vue Boîte aux lettres, vue Calendrier et vue Tâches).

Changer la couleur de fond

Vous pouvez changer la couleur de fond de votre boîte aux lettres ("Préférences", onglet "Généralités"). Dans la section "Options de connexion", le champ "Thème" affiche la liste des thèmes disponibles.

Modifier l'affichage des mails

Par défaut, votre boîte aux lettres affiche les mails présents dans la Boîte de réception, mais vous pouvez modifier cette configuration :

  • Changer le dossier qui est affiché à chaque connexion. Par défaut, votre boîte aux lettres affiche les mails présents dans la Boîte de réception. Vous pouvez sélectionner tout autre dossier de mails. Vous pouvez également sélectionner des critères de recherche – par exemple, de manière à afficher les mails non lus (dans ce cas, seuls les mails non lus seront affichés à chaque connexion). Pour plus de détails, consultez cette page.

  • Précisez si vous souhaitez afficher le panneau de lecture, et si celui-ci doit être affiché au bas de la page ou sur la droite de la page. Pour effectuer ces modifications, cliquez sur le lien Vue affiché dans la barre d’outils des mails.

  • Si le panneau de lecture est désactivé ou affiché au bas de la page, vous pouvez sélectionner les colonnes qui décrivent les mails dans le panneau "Contenu". Vous n'êtes pas obligé d'afficher tous les détails relatifs aux mails dans le panneau "Contenu". Pour afficher la liste des colonnes disponibles, cliquez avec le bouton droit dans les en-têtes des colonnes. Pour masquer une de ces colonnes, cliquez sur son nom dans la liste (cette fonction étant une bascule, vous pouvez réafficher la colonne en cliquant à nouveau sur son nom dans la liste).

Modifier l'affichage du calendrier

Vous pouvez spécifier la vue dont vous souhaitez disposer lorsque vous ouvrez votre calendrier (parmi les vues Jour, Semaine ouvrée, Semaine de sept jours, Mois, Programme ou Liste). Vous pouvez également sélectionner le jour de la semaine par lequel vous souhaitez que vos semaines de calendrier commencent. Pour plus de détails, consultez cette page.

Modifier l'affichage des tâches

Vous pouvez sélectionner les colonnes dont vous souhaitez disposer dans le panneau "Contenu" lorsque vous consultez vos tâches. Pour afficher la liste des colonnes disponibles, cliquez avec le bouton droit dans l'en-tête de la colonne "Tâches". Pour masquer une de ces colonnes, cliquez sur son nom (cette fonction étant une bascule, vous pouvez réafficher la colonne en cliquant à nouveau sur son nom dans la liste).
