Modifier l’état et la disponibilité

Pour utiliser la messagerie instantanée (MI), vous devez sélectionner l’état En ligne ou Disponible pour MI.

Pour spécifier votre état dans la messagerie instantanée de Zimbra :

  1. Dans la barre de messagerie instantanée, cliquez sur l’icône pour afficher la liste des états. Cette liste présente les valeurs suivantes :

    • Hors ligne

    • En ligne

    • Disponible pour le service MI

    • Ne pas déranger

    • Absent

    • Longue absence

    • Nouveau message d’état

  1. Sélectionnez l’état que vous souhaitez afficher à l’attention de vos correspondants potentiels.

Vous pouvez également cliquer sur l’état affiché à côté de votre nom dans le panneau "Aperçu" (partie gauche) et sélectionner un état dans la liste qui apparaît.

Créer un nouveau message d’état

La messagerie instantanée de Zimbra vous permet de créer plusieurs messages d’état personnalisés.

Pour créer un message d’état :

  1. Dans la barre de messagerie instantanée, cliquez sur l’icône pour afficher la liste des états.

  2. Choisissez Nouveau message d’état.

  3. Dans la boîte de dialogue, qui apparaît, tapez le texte décrivant votre nouvel état, puis cliquez sur OK. Exemple de message d’état : "Je reviens de suite". Le message que vous venez de créer apparaît dans la liste des états.

Pour afficher l'un des états par défaut, déroulez la liste "En ligne" et sélectionnez la valeur souhaitée.

  • Les messages d’état que vous créez sont mémorisés pour en permettre l'utilisation ultérieure (ils ne peuvent pas être supprimés). ZWC garde en mémoire les cinq derniers messages d’état que vous créez.
