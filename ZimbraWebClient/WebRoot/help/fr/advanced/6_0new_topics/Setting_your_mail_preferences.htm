Configurer vos préférences mails

L’onglet Mail inclut les préférences suivantes que vous pouvez modifier.

  • Afficher. Sélectionnez le nombre d’éléments (mails ou conversations) à afficher par page. Par défaut, l’affichage est de 25 objets par page, mais vous pouvez choisir d’en afficher plus ou moins dans chaque page.

  • Tester la présence de nouveau mail chaque. Choisissez l’intervalle de consultation, c’est-à-dire la fréquence selon laquelle votre ordinateur vérifie les nouveaux mails. Par défaut, la consultation des mails s’effectue toutes les 5 minutes. Il n’est pas recommandé de configurer cet intervalle en dessous de 5 minutes car la vérification fréquente des mails entraîne une forte activité sur le serveur de messagerie et diminue ses performances. Si vous attendez un mail, vous pouvez cliquer sur Actualiser à tout moment pour recevoir immédiatement vos nouveaux mails. Si vous choisissez Jamais, vous devez cliquer sur Actualiser dans la barre d’outils pour recevoir vos nouveaux mails.

  • Afficher le mail. Choisissez "Format HTML" (si possible) pour afficher les mails au formatage HTML, par exemple, afficher le texte en gras, coloré, ou tout autre style. Choisissez Format Texte pour afficher les messages au format texte simple. Au format texte simple, les balises HTML sont affichées, et non pas appliquées en tant que code au mail.

  • Options d’Aperçu du mail

    • Afficher un extrait du contenu de chaque mail. Si cette option est activée, la première ligne du mail est affichée lorsque le curseur est placé sur l’objet d'un mail. Si elle est désactivée, seuls s’affichent les premiers mots du mail.

    • Lorsque vous double-cliquez sur un mail, celui-ci s’ouvre dans une nouvelle fenêtre si l’option Double-cliquez pour ouvrir le mail est activée.

  • Activez Images pour télécharger automatiquement les images de votre mail en HTML. Lorsque cette fonction est activée, il n’est pas nécessaire de cliquer sur l’image du mail pour voir l’image.

  • Lorsque j'affiche un mail dans le panneau de lecture définit le comportement pour marquer les mails qui sont consultés dans le panneau de lecture en tant que lus ou non lus. Vous pouvez configurer vos mails pour qu’ils soient marqués comme lus immédiatement, marqués comme lus après un nombre de secondes défini de consultation dans le panneau de lecture, ou toujours conserver comme non lus les mails consultés dans le panneau de lecture.

  • Recherche Mail par défaut définit la recherche à exécuter lorsque vous êtes connecté. Les résultats de cette recherche s’affichent dans le panneau Contenu lorsque vous vous connectez à votre boîte aux lettres. La recherche par défaut est votre Boîte de réception. Lorsque vous vous connectez, les résultats de votre Boîte de réception sont alors affichés. Vous pouvez modifier ce champ pour qu'un autre dossier, tag, ou recherche enregistrée apparaissent lorsque vous ouvrez le client Web de Zimbra pour la première fois. Consultez la section "Description du langage de demande pour connaître la syntaxe".

  • Lorsqu’un mail arrive…. Lorsque vous recevez un mail, vous pouvez définir la manière dont vous souhaitez être averti : émettre un son, afficher brièvement l’onglet Mail, afficher brièvement le nom du navigateur. Voir la section Configurer des alertes dans Zimbra Desktop.

  • Faire suivre une copie à. Spécifiez une adresse vers laquelle faire suivre votre mail. Vous pouvez supprimer le mail original de votre messagerie Zimbra.

  • Spécifiez si vous souhaitez envoyer une réponse automatique par mail à un mail entrant, et définissez la date de début et d’arrêt. Cette fonction est fréquemment utilisée pour les messages d’absence. Vous pouvez également spécifier une date de début et d’arrêt pour utiliser ce message. Voir la section Messages d’absence.

  • Accusé de lecture. Spécifiez la manière de gérer les mails que vous recevez et qui sont affectés d’un drapeau pour envoyer un message d’accusé de réception de lecture à l'expéditeur.

  • Configurez la section Mails envoyés par moi-même pour traiter les mails que vous envoyez et qui contiennent votre adresse parmi les destinataires du mail, ou une adresse telle qu'une liste de diffusion à laquelle vous appartenez.

  • Options relatives au spam. Vous pouvez identifier jusqu’à 100 adresses pour lesquelles vous voulez toujours bloquer les mails – pour qu’ils ne soient pas ajoutés à votre Boîte de réception, et 100 adresses pour lesquelles vous voulez être certain que leurs mails soient ajoutés à votre Boîte de réception – et non pas envoyés dans le dossier Spam.

  • Si votre configuration vous permet d’utiliser un client POP3, vous pouvez configurer le comportement d’accès POP lorsque des mails sont téléchargés sur le ZWC. Les options d’accès POP sont :

    • Autoriser le téléchargement de tous les mails (y compris les anciens mails)

    • Autoriser uniquement le téléchargement des mails reçus à partir de maintenant
