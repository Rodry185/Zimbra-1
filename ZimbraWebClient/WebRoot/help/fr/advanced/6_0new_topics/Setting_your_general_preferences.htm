Configurer vos préférences générales

L’onglet Général inclut les paramètres suivants, que vous pouvez modifier.

  • Vous êtes connecté au client Web avancé, mais ici, vous pouvez définir votre type de client par défaut.

    • Advanced (version avancée) – Propose la gamme complète des fonctions de collaboration Web. Ce client Web exige un navigateur récent et une connexion Internet rapide.

    • Standard – Recommandé pour les connexions Internet lentes, pour des navigateurs moins récents et pour les utilisateurs à accessibilité réduite.  

  • Dans Thème choisissez la couleur de fond pour l’interface de votre messagerie. Dans l’onglet Général, Thèmes affiche une liste des différentes couleurs de fond que vous pouvez choisir.

  • Vous pouvez changer la Langue d’affichage dans le client Web de Zimbra (choisissez celle que vous comprenez le mieux).

  • Le Fuseau horaire par défaut détermine la date et l’heure des mails que vous envoyez ou recevez, et l’heure à afficher pour les rendez-vous de votre calendrier. Celui-ci peut être différent de la configuration de fuseau horaire de votre ordinateur. Si le fuseau horaire affiché dans cet onglet est incorrect, changez-le ici.  

  • Si Changer le mot de passe s’affiche, vous pouvez modifier votre mot de passe. Si vous avez des difficultés à modifier votre mot de passe, contactez votre administrateur pour connaître les règles en matière de mot de passe. Contactez votre administrateur système si cette option n’est pas disponible et si vous voulez changer votre mot de passe.

  • Dans Paramètres de recherche, vous pouvez choisir d’inclure automatiquement dans les dossiers Spams et Corbeille toutes les recherches que vous effectuez. Par défaut, ces dossiers ne sont pas pris en compte lors des recherches.

  • Dans Langue de recherche, cochez Toujours montrer la chaîne de recherche pour afficher les détails de cette chaîne dans la zone de texte de recherche dans la partie supérieure de la fenêtre ZWC.  

  • Les Paramètres àcocher Afficher les cases à cocher qui facilitent la sélection des objets dans les listes (exige un rafraîchissement de l'affichage) permettent d’afficher une case à cocher pour chaque objet du panneau de contenu. Lorsque cette option est activée, utilisez les cases à cocher pour sélectionner un ou plusieurs éléments sur lesquels exécuter la même action. Par exemple : supprimer, déplacer, marquer comme lu/non lu.
