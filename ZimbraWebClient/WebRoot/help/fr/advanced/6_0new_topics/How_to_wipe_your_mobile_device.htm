Mobile perdu ou volé – Effacer les données personnelles

Si votre mobile est perdu ou volé, vous pouvez en effacer les données personnelle à l'aide du lien "Supprimer le mobile" (onglet des préférences des mobiles) : cette fonction efface les données stockées dans le mobile et rétablit les valeurs par défaut d'origine.

  1. Activez l'onglet Préférences et cliquez sur le dossier Mobiles.

  2. Sélectionnez le mobile requis et cliquez sur Supprimer le mobile.  

Le message d'état Suppression en instance (ou texte équivalent) apparaît. Le serveur de votre opérateur attend que le mobile se connecte pour exécuter la suppression : dès que le mobile se connecte au serveur, ses données sont effacées et le message d'état devient Suppression terminée (ou texte équivalent).

  • Remarque – Tant que le message d'état est "Suppression en instance", vous pouvez annuler la demande de suppression du mobile.

En complément de la fonction de "suppression de mobile" (effacement des données à distance), ZCS supporte d'autres mécanismes de sécurité pour les mobiles – en particulier, le verrouillage du code PIN. La fonction de verrouillage du code PIN détecte le nombre de saisies d'un mot de passe incorrect ; si ce nombre atteint le maximum autorisé, elle exécute la suppression "locale" (sans attendre la connexion au serveur) du mobile, soit l'effacement de toutes les données du mobile. Notez que la configuration de ce paramètre est réservée à l’administrateur (le paramètre ne peut pas être modifié par les utilisateurs).
