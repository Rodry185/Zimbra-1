Gérer les objets partagés

Vous pouvez consulter et gérer vos dossiers partagés à partir de la page Préférences > Partage. Actions possibles :

  • Consulter les dossiers qui vous sont proposés en partage (y compris ceux que vous n’avez pas acceptés).

  • Consulter les dossiers que vous partagez, et leur appliquer des actions telles que modifier les propriétés d'un partage, annuler un partage ou relancer une invitation de partage.

  • Partager un dossier

Dossiers partagés avec moi

Dans cette section, vous pouvez basculer entre deux vues :

  • Afficher la liste des dossiers partagés avec moi dans le cadre d’une liste de distribution – Cette option est la valeur par défaut. Dans les dossiers de ce type, les champs affichent les détails des objets partagés.

  • Afficher la liste des dossiers partagés avec moi par les utilisateurs suivants – Lorsque vous sélectionnez cette option, vous devez spécifier le nom d’un utilisateur qui partage un objet avec vous. Si vous avez accepté l’invitation de partage, les détails sont affichés dans l’un des champs. Vous pouvez consulter les détails d’un seul utilisateur à la fois, et les détails ne sont pas enregistrés.

Dossiers partagés par moi

Cette section affiche la liste complète de vos dossiers partagés, avec adresse des utilisateurs impliqués dans chaque partage, emplacement de l'objet partagé, type de l'objet partagé (Mail, Carnet d’adresses, Calendrier, Carnet, Liste des tâches ou Porte-documents) et rôle/autorisations que vous affectez à chaque utilisateur.

Vous pouvez modifier les propriétés d'un partage, annuler un partage ou relancer une invitation de partage.

Partager un dossier

À partir du dossier Préférences, vous pouvez partager les dossiers suivants : mails, carnets d’adresses, calendriers, listes de tâches, carnets, Porte-documents.

  1. Sélectionnez le type de dossier requis.

  2. Cliquez sur Partager.

  3. Sélectionnez l'objet spécifique à partager et cliquez sur OK. La boîte de dialogue "Propriétés du partage" apparaît.

  4. Sélectionnez la personne avec laquelle vous souhaitez partager cet objet.

Utilisateurs ou groupes internes

  1. Si vous activez l’option "Interne", vous pouvez saisir l'adresse complète manuellement ou sélectionner l'une des adresses suggérées à mesure que vous saisissez des caractères. L’utilisateur ou le groupe doit être associé à une adresse correcte sur le serveur mail de votre société.

  2. Dans la zone Rôle, sélectionnez les autorisations d'accès à accorder.

  3. Si vous acceptez de partager vos rendez-vous privé, cochez la case Autoriser les utilisateurs à consulter les rendez-vous privés.

  1. Dans la zone Mail , sélectionnez le type de mail à envoyer. Options disponibles :

  • Envoyer un mail standard – Les destinataires recevront le mail standard de notification de partage.  

  • Ajouter une note au mail standard – Pour ajouter des informations complémentaires au mail standard (facultatif).

  1. Cliquez sur OK.

 

Invités externes

  1. Si vous activez l’option "Externe", vous devez saisir l'adresse mail de chaque destinataire. Dans le champ "Mail", vous pouvez saisir plusieurs adresses, à condition de les séparer par un point-virgule (;).

  2. Spécifiez le mot de passe que les utilisateurs devront saisir pour consulter votre calendrier.

  3. Si vous acceptez de partager vos rendez-vous privés, cochez la case Autoriser les utilisateurs à consulter les rendez-vous privés.

  1. Dans la section Mail, sélectionnez le type de mail à envoyer. L’URL de votre calendrier et le mot de passe d'accès à celui-ci sont insérés automatiquement dans le mail.

    • Envoyer un mail standard – Les destinataires recevront le mail standard de notification de partage.

    • Ajouter une note au mail standard – Pour ajouter des informations complémentaires au mail standard (facultatif).

  2. Cliquez sur OK.

Public

Si vous activez l’option "Public", toute personne disposant de l’URL peut consulter votre calendrier (avec cette configuration, le mot de passe est facultatif).
