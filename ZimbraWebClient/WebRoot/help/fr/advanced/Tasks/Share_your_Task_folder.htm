Partager votre liste de tâches

Vous pouvez partager vos listes de tâches avec d'autres utilisateurs, et vous pouvez définir les autorisations pour la liste partagée. Voici comment :

  • Les utilisateurs ou groupes internes. Vous choisissez le type de privilèges à accorder ou :

    • Visionneur – Le bénéficiaire peut consulter le contenu de la liste de tâches, mais ne peut pas apporter de modifications.

    • Gestionnaire – Le bénéficiaire peut appliquer les fonctions suivantes à votre liste de tâches : créer, modifier, supprimer des tâches.

    • Admin – Le bénéficiaire peut appliquer les fonctions suivantes à votre liste de tâches : créer, modifier, supprimer des tâches, partager votre liste de tâches avec d'autres utilisateurs.

    • Aucune est une option pour désactiver temporairement l'accès à une liste d'un cédant sans retirer les privilèges de partage. Le bénéficiaire dispose encore de la liste de tâches dans son affichage de listes de tâches, mais ne peut ni consulter ni gérer les tâches.

  • Invités externes. Vous créez un mot de passe pour accéder à la liste de tâches partagée. Les invités doivent saisir ce mot de passe pour pouvoir consulter la liste de tâches. Ils ne peuvent apporter aucune modification aux tâches.

  • Public. Tout utilisateur ayant connaissance de l'URL menant à la liste de tâches partagée peut la consulter. Ces utilisateurs ne peuvent pas effectuer de modifications.

Pour partager votre liste de tâches

Cliquez avec le bouton droit sur la liste de tâches à partager, puis choisissez Partager une liste de tâches.

Dans la boîte de dialogue Propriétés partagées, saisissez le nom de la personne avec laquelle vous souhaitez partager la liste, puis configurez les informations qui conviennent.

 

Partage avec des utilisateurs ou des groupes internes1. Sélectionnez Utilisateurs ou groupes internes avec qui partager

2. Dans le champ Mail, tapez les adresses mails des bénéficiaires. Lorsque vous saisissez une adresse, une liste de noms correspondant aux lettres que vous insérez s'affiche. Vous pouvez sélectionner le nom dans cette liste.

3. Sélectionnez le rôle, Visionneur ou Gestionnaire

4. Dans la section Message, sélectionnez le type de mails à envoyer pour informer le bénéficiaire.

5. Cliquez sur OK.

Partage avec des invités externes

1. Dans la section Partager avec, choisissez invités externes. Lorsque vous remplissez cette case, un champ de mot de passe s'affiche.

2. Saisissez les adresses mail des invités.

3. Saisissez le mot de passe que les invités devront utiliser pour accéder à cette liste de tâches.

4. Sélectionnez le type de message de notification à leur envoyer. Ce message leur fournit les informations nécessaires pour qu'ils puissent accéder à la liste.

5. Cliquez sur OK.

 

Partage avec le public

1. Dans la section Partager avec, choisissez Public. Lorsque vous effectuez ce choix, la boîte de dialogue des propriétés du partage affiche l'URL pour votre liste de tâches.

2. Notez bien cette URL et informez les utilisateurs de "public" de cette adresse mail. Lorsqu'ils cliquent sur l'URL, ils peuvent lire le contenu de la liste des tâches mais ne peuvent pas apporter de modifications.

Modification ou retrait de l'autorisation

Vous pouvez retirer les autorisations à tout moment.

  1. Cliquez avec le bouton droit sur la liste de tâches partagée

  2. Cliquez sur Propriétés d'édition. Les utilisateurs partageant le dossier sont répertoriés.

  3. Cliquez sur Modifier pour modifier les autorisations, ou
    sur Retirer pour supprimer les autorisations.

  4. Choisissez si un mail décrivant les modifications doit être envoyé.

  5. Cliquez sur OK.
