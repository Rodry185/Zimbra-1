Exporter des objets de votre compte

Le dossier Import/Export de l'onglet Préférences vous permet d'exporter différents objets de votre compte : détails du compte, dossiers, calendriers, listes de contacts.

Exporter une liste de contacts (carnet d'adresses)

Lorsque vous exportez une liste de contacts, celle-ci est enregistrée au format .csv (dans le fichier généré, les différents contacts sont séparés par une virgule).

  1. Dans l'onglet Préférences, cliquez sur le dossier Import/Export.

  2. Dans la section "Export", sélectionnez le type Contacts.

  3. Dans le champ Fichier, retrouvez et sélectionnez le dossier contenant le carnet d'adresses à exporter.

  4. Dans la liste Type, sélectionnez le format d'export requis, parmi :

    • Contacts du compte – Le fichier est exporté tel qu'il apparaît dans votre compte.

    • Contacts Yahoo!

    • Contacts Gmail

    • Contacts Outlook 2000

    • Contacts Outlook 2003

    • Contacts Outlook

    • Contacts Thunderbird

La valeur sélectionnée garantit que les détails des contacts exportés seront présentés sous le format adapté à l'application cible.

  1. Cliquez sur Exporter.

  2. Sélectionnez l'emplacement de destination du fichier exporté, puis cliquez sur OK. Pour plus de détails sur l'importation de ces contacts dans l'application cible, voir la documentation de celle-ci.
