Effacer les données de votre périphérique mobiile

Remarque : La fonction mobile est disponible avec @CollaborationProductFullName@ Network Edition seulement.

Si votre mobile est perdu ou volé, vous pouvez en effacer les données personnelle à l'aide du lien Supprimer le mobile (onglet des Préférences des mobiles) : cette fonction efface les données stockées dans le périphérique et rétablit les paramètres d'origine définis en usine.

  1. Activez l'onglet Préférences et cliquez sur le dossier Mobiles.

  2. Sélectionnez le périphérique à effacer et cliquez sur Supprimer le mobile.  

Le message d'état Suppression en instance (ou texte équivalent) apparaît.  Le serveur de votre opérateur attend que le périphérique se connecte  pour exécuter la suppression :  dès que le périphérique se connecte au serveur, ses données sont effacées  et le message d'état devient Suppression terminée.

Remarque :  tant que le message d'état est Suppression en instance, vous pouvez annuler la demande d'effacement du périphérique.
