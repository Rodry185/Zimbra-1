Changer votre fuseau horaire par défaut

La configuration du fuseau horaire par défaut qui est affichée dans le dossier Préférences>Général  sert à localiser l'heure des messages reçus et des activités du calendrier dans le client Web standard.  

En utilisant le client Web standard, le fuseau horaire sur l'ordinateur ne sert pas à définir l'heure à laquelle un message est reçu ou pour les activités du calendrier ; il s'agit de la configuration du fuseau horaire dans l'onglet Préférences>Général. Cependant, en utilisant le client Web avancé, la configuration du fuseau horaire sur l'ordinateur sert d'horodatage pour les messages reçus et pour les activités du calendrier, pas la configuration du fuseau horaire sur l'onglet Général.

 

Remarque : étant donné que le client Web avancé et le client Web standard n'utilisent pas la même source de fuseau horaire pour indiquer les messages, vous pouvez remarquer que le même message comporte une heure différente quand il est affiché sur l'un ou l'autre des clients.  Pour éviter cette situation, utilisez le même fuseau horaire sur l'ordinateur et le client Web.
