Restaurer les préférences par défaut

Sur chacun des onglets Préférences hormis l’onglet Filtres de mails, le bouton Restaurer les valeurs par défaut sur chaque page permet de réinitialiser les modifications appliquées à cette page. Il restaure les valeurs par défaut définies lors de votre première connexion avec votre nouveau compte Zimbra. Vous devez cliquer sur Enregistrer pour appliquer ces modifications.

  • Cette fonction de réinitialisation n’affecte pas les modifications apportées aux mots de passe.
