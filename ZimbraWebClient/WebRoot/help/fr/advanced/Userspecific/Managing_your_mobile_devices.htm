Gérer vos périphériques mobiles

Remarque : La fonction mobile est disponible avec @CollaborationProductFullName@ Network Edition seulement.

Si les fonctions ZCS applicables aux mobiles sont disponibles, le dossier Mobiles est affiché dans l'onglet Préférences. Ce dossier affiche des mobiles que vous avez configurés pour être synchronisés avec votre compte ZWC.

À partir de ce dossier, vous pouvez :

  • Lancer une synchronisation à partir de ZWC vers votre mobile

  • Suspendre une synchronisation

  • Supprimer les périphériques mobiles que vous n'utilisez plus.

En outre, si vous perdez votre mobile ou qu'on vous le vole, vous pouvez effacer les données personnelles.

Fonctions de sécurité du mobile

Votre administrateur peut avoir configuré des règles de sécurité de mobile qui vous obligent à créer un numéro d'identification personnelle (PIN) sur votre mobile. Si vous oubliez votre PIN, contactez votre administrateur système.

Quand vous avez un PIN, les règles peuvent être configurées avec une fonction pour bloquer les utilisateurs s'ils n'entrent pas le PIN correct.  L'administrateur système configure la fonction de verrouillage PIN afin que si quelqu'un tente de se connecter à un périphérique avec un mot de passe incorrect un nombre de fois supérieur au nombre de tentatives défini, un effacement lancé localement sur le périphérique est effectué. Cela efface toutes les données du mobile. La configuration de ce paramètre est réservée à l'administrateur et ne peut pas être modifié par les utilisateurs.  Pour plus d'informations, contactez votre administrateur système.
