Modifier vos préférences d'utilisateur

Vos préférences d'utilisateur par défaut sont configurées lors de la création de votre compte. Ces options définissent le fonctionnement de votre messagerie, de vos carnets d'adresses et de vos calendriers.

Vous pouvez modifier les paramètres dans l'onglet Préférences. Si l'option décrite ici n'est pas visible dans votre affichage, cela signifie que cette préférence n'est pas disponible pour votre compte.  

Voici les indications générales pour modifier vos préférences :

  1. Cliquez sur Préférences.

  2. Dans le Panneau d'aperçu, cliquez sur le dossier contenant la préférence à modifier. Pour plus d'informations, voir le sujet Aide pour chacun de ces dossiers.

  3. Modifier les paramètres.

  4. Cliquez sur Enregistrer.
