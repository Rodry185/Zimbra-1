Supprimer un rendez-vous

Les rendez-vous peuvent être annulés par leur organisateur et le participant peut supprimer une réunion de son calendrier.

Organisateur annulant la réunion

En tant qu'organisateur d'une réunion, vous pouvez l'annuler de la manière suivante :

  1. Cliquez sur le rendez-vous à annuler.

  2. Cliquez avec le bouton droit, puis sélectionnez Annuler. Si le rendez-vous est une série récurrente, vous pouvez supprimer une instance de la série ou la série complète.

  3. Une boîte de dialogue s'affiche alors, vous demandant si vous souhaitez modifier le message d'annulation du rendez-vous.

    • Si vous sélectionnez Non, un mail est envoyé aux participants et le rendez-vous est supprimé de leurs calendriers.

    • Si vous sélectionnez Oui, la fenêtre de composition du message s'ouvre, et vous pouvez alors le modifier avant de l'envoyer.

    • Si vous sélectionnez Annuler, la réunion ne sera pas supprimée.

Participant supprimant une réunion

En tant que participant à une réunion, vous pouvez la supprimer de votre calendrier. Vous pouvez alors prévenir l'organisateur.

  1. Cliquez sur la réunion à supprimer.

  2. Dans la boîte de dialogue de confirmation, indiquez si vous souhaitez prévenir l'organisateur, puis cliquez sur Oui pour supprimer la réunion.
