Création de calendriers

Un calendrier a été créé pour vous ; vous pouvez en créer des nouveaux pour effectuer un suivi des différents types de fonctions.  Par exemple, vous pouvez créer un calendrier pour le travail, un calendrier personnel, et un calendrier pour les activités de la société.  Vous pouvez utiliser la vue de Programme pour afficher côte à côte les calendriers.

Pour créer un calendrier

  1. Activez l'onglet "Calendrier".

  2. Dans le panneau d'aperçu du calendrier, cliquez sur ou, depuis la barre d'outils,  cliquez sur la flèche et sélectionnez Calendrier.

  3. Dans la boîte de dialogue Créer un Calendrier , saisissez le nom du calendrier puis sélectionnez une couleur pour indiquer les activités du Calendrier.

  4. Si vous ne voulez pas que d'autres utilisateurs aient accès aux activités du calendrier, cochez Exclure ce calendrier au sujet de vos disponibilités.

  5. Pour créer un calendrier et synchroniser vos rendez-vous depuis un calendrier distant, cochez Synchroniser les rendez-vous avec le calendrier distant. Saisissez une URL d'un fichier iCal correct pour le calendrier distant.

  6. Cliquez ensuite sur OK. Le nouveau calendrier s'affiche dans la liste du Calendrier.

Les calendriers distants sont mis à jour à intervalles réguliers (définis par l'administrateur système).
