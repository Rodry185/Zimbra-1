Assistant @CompanyName@

L'assistant @CompanyName@ est une fonction utilisateur avancée qui permet d'interagir rapidement avec le client Web à l'aide du clavier pour effectuer les tâches suivantes :

  • Créer des rendez-vous

  • Accéder à une date dans votre calendrier

  • Créer des contacts

  • Envoyer un mail sans modifier l'application que vous utilisez actuellement

Vous pouvez ouvrir l'assistant @CompanyName@ à partir de tout affichage, sauf l'onglet Préférences. Le raccourci clavier pour ouvrir l'assistant @CompanyName@ est ` (guillemet inversé). Cette touche se trouve en dessous de la touche Échap. ( Échap.)

La boîte dialogue de l'assistant @CompanyName@ fournit une aide détaillée. Vous trouverez ci-dessous les étapes générales à suivre pour utiliser l'assistant @CompanyName@.

  1.  Dans @ClientNameLong@, utilisez la touche de raccourci (guillemet inversé ou maj~ (tilde)). La boîte de dialogue de l'assistant @CompanyName@ s'affiche.

  2. Saisissez l'une des commandes suivantes :

    • Pour créer un rendez-vous, saisissez rendez-vous. Vous pouvez alors saisir un objet entre guillemets, l'heure, une date et un jour de la semaine et tout autre commentaire pour le rendez-vous. Cliquez sur OK.  Remarque : Si vous voulez utiliser des fonctions de rendez-vous plus avancées, cliquez sur Plus de détails sur la page de Rendez-vous.

    • Pour accéder à une date spécifique de votre calendrier, saisissez calendar et la date ou le jour de la semaine. Cliquez sur OK.  Remarque : Après la date, vous pouvez saisir l'affichage du calendrier que vous préférez : jour, semaine de travail, semaine ou mois. Le calendrier s'ouvre dans cette vue.

    • Pour créer un contact, saisissez contact et entrez le nom du contact, son adresse mail, etc. Cliquez sur OK.  Remarque : Si vous voulez ouvrir la fiche détaillée des contacts, cliquez sur Plus de détails. Vous pouvez spécifier sur cette fiche le carnet d'adresses dans lequel vous voulez ajouter le nom du contact.

    • Pour envoyer un mail sans ouvrir la fenêtre Écrire un mail, saisissez mail. Saisissez l'objet entre guillemets ; remplissez le champ à : name@address, puis tapez le corps du message.  Cliquez sur OK.
