Créer un carnet

  1. Dans le client Web de Zimbra, activez l’onglet Documents.

  2. Faites un clic droit sur la barre de Carnets, puis sélectionnez Nouveau carnet.


     

  3. La boîte de dialogue Créer un carnet s’affiche. Saisissez le nom du nouveau carnet.

Ce nom est doit être unique dans la hiérarchie des dossiers de votre boîte aux lettres. Les carnets créés dans la barre Carnets  ne peuvent pas porter le même nom qu'un dossier de niveau supérieur de votre messagerie, calendrier, ou carnet d'adresses Zimbra. Par exemple, si l'un de vos calendriers s’appelle Vacances, vous ne pouvez pas nommer un carnet de niveau supérieur du même nom. Vacances peut être le nom d’un carnet inclus dans un autre carnet.

  1. Sélectionnez la  hiérarchie des dossiers. Vous pouvez créer un carnet de niveau supérieur, ou vous pouvez l’insérer dans un carnet existant.

  2. Cliquez sur OK. Le nouveau carnet s’affiche dans le panneau d’aperçu.
