Accepter l’accès à un carnet

Si vous recevez une notification par mail indiquant que vous êtes autorisé à partager le carnet d'un autre utilisateur, vous pouvez alors accepter ou refuser le partage.

Si vous acceptez le partage, le carnet partagé s’affiche dans votre liste de Carnets.

Pour accepter

  1. Cliquez sur Accepter le partage dans le mail. La boîte de dialogue Accepter le partage s’affiche et décrit le rôle qui vous a été attribué .

  2. Avant d’accepter le partage, vous pouvez personnaliser le nom du carnet.

  3. Cochez le mail envoyé qui traite de ce partage, et envoyez une confirmation au cédant. (facultatif).

  4. Cliquez sur Oui. Le carnet est ajouté à votre liste de Carnets. Ce message est déplacé vers le dossier Corbeille.
