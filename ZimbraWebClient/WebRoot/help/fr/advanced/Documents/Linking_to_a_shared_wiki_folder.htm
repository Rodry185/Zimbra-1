Lien vers un carnet partagé

En plus d’accepter une notification par mail indiquant que vous êtes autorisé à accéder à un carnet, vous pouvez également établir des liens vers des carnets partagés – si vous avez l’autorisation d’accès, en utilisant la boîte de dialogue Lien vers un carnet partagé. Par exemple, cela pourrait être le cas si votre organisation dispose de carnets publics que vous souhaitez consulter.

Remarque : Les privilèges d’accès dont vous disposez ont été configurés par le cédant.

  1. Dans l’onglet Documents, cliquez avec le bouton droit sur l’en-tête Carnets puis choisissez Lien vers un carnet partagé. Une boîte de dialogue s’affiche.

  1. Pour ajouter le lien d'un carnet partagé, vous devez connaître l'adresse mail du propriétaire ainsi que le nom du carnet. Par exemple, Jdoe@exemple.com a créé un carnet nommé Projets, et un second carnet dans le carnet Projets qui s'appelle Projet 1. Jdoe partage alors le carnet Projet 1 avec vous. Pour accéder au carnet Projet 1, saisissez les informations de la façon suivante :

Adresse mail : jdoe@exemple.com

Chemin : projets/projet1

Nom : Le nom saisit automatiquement le nom du mail et le chemin. Vous pouvez le modifier.

Couleur : Choisissez une couleur dans la liste (facultatif)

  1. Cliquez sur OK. Le carnet est ajouté à votre liste de Carnets.
