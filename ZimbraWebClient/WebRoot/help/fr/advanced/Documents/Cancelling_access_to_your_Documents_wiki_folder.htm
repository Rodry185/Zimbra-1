Modifier ou annuler l'accès à vos Carnets

Vous pouvez annuler les privilèges de partage et modifier les autorisations accordées dans la boîte de dialogue Propriétés des carnets.

Pour modifier les autorisations d’accès

  1. Dans l’onglet Carnets, cliquez avec le bouton droit sur le carnet partagé à modifier.

  2. Sélectionnez Propriétés d'édition. La boîte de dialogue Propriétés du dossier s'affiche. Cette boîte de dialogue affiche le nom des utilisateurs qui partagent le dossier de carnet.

  3. Dans la section Partages pour ce dossier, cliquez sur Modifier par le nom à modifier. La boîte de dialogue Propriétés du partage s’affiche. Modifiez le Rôle.

    • Visionneur. Le bénéficiaire peut consulter le contenu du carnet mais ne peut pas le modifier.

    • Gestionnaire. Le bénéficiaire dispose de l’autorisation intégrale pour consulter, modifier le contenu d’un carnet, ajouter de nouvelles pages et supprimer des pages.

    • Aucune. Cette option sert à désactiver temporairement l’accès au carnet partagé d’un cédant sans retirer les privilèges de partage. Le bénéficiaire dispose toujours du carnet sur sa liste mais ne peut plus afficher ou gérer les activités dans le carnet.

  1. Un mail sera envoyé afin d’avertir l’utilisateur. Vous pouvez également choisir de ne pas envoyer de mail ou d'en envoyer un qui est personnalisé.

  2. Cliquez sur OK.

Pour annuler l’accès

  1. Dans l’onglet Carnets, cliquez avec le bouton droit sur le carnet partagé.

  2. Sélectionnez Propriétés d'édition. La boîte de dialogue Propriétés du dossier s'affiche. Cette boîte de dialogue affiche le nom des utilisateurs qui partagent le carnet.

  3. Cliquez sur Retirer. La boîte de dialogue Retirer le partage s’affiche. Un mail sera envoyé afin d’avertir l’utilisateur. Vous pouvez également choisir de ne pas envoyer de mail ou d'en envoyer un qui est personnalisé.

  4. Cliquez sur OK. L’utilisateur n’a plus accès à votre carnet. Il peut supprimer le carnet de sa liste.
