Blocage des pièces jointes

Selon la politique de sécurité associée à votre site, certains types de pièces jointes ne peuvent pas s’afficher dans votre messagerie. Il est possible que le lien symbolisé par un trombone indique une pièce jointe, mais vous ne pourrez pas ouvrir cette pièce jointe.

Votre administrateur système peut définir les options de blocage des pièces jointes suivantes :

  • Choisissez le type de fichiers que vous pouvez ouvrir. Par exemple, afficher les pièces jointes si elles peuvent être converties en fichiers HTML (par exemple, Word, PowerPoint, PDF, Excel, ou documents textes) mais pas en fichiers images, fichiers ZIP ou fichiers exécutables.

  • Afficher toutes les pièces jointes, sans considération du type de fichier.

Cette option est différente du rejet de toutes les pièces jointes, qui bloque généralement la distribution des mails contenant des virus ou un contenu inapproprié.

Si vous recevez une pièce jointe que vous ne pouvez pas ouvrir, contactez votre administrateur système.
