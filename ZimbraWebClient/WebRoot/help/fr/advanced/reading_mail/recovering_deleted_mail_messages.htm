Récupération d'objets supprimés

Le dossier Corbeille est disponible pour chaque application @CompanyName@ que vous utilisez. Seuls les éléments spécifiques à l'application sélectionnée sont affichés quand vous ouvrez le dossier Corbeille de l'application. Mais quand vous sélectionnez Vider la corbeille à partir d'une application, les éléments de toutes les applications sont supprimés.

Avant de vider le dossier Corbeille, vous pouvez récupérer les fichiers de la corbeille en les faisant glisser et en les déposant dans un autre dossier.

Si Restaurer objets supprimés s'affiche quand vous cliquez avec le bouton droit de la souris sur le dossier Corbeille, vous pouvez récupérer les mails que vous aviez supprimés de la corbeille.

  1. Cliquez avec le bouton droit de la souris sur le dossier Corbeille de messagerie, puis sélectionnez Restaurer objets supprimés.

  2. Sélectionnez un ou plusieurs messages à récupérer.

  3. Cliquez sur Restaurer à, puis sélectionnez le dossier de destination ou créez un dossier.

  4. Cliquez ensuite sur OK.

  5. Fermez.

Il est possible de récupérer les éléments supprimés de la corbeille dans un délai de 30 jours après la suppression de la corbeille. Passés ces 30 jours, les élémentts sont supprimés définitivement.

Les messages qui restent dans la corbeille peuvent être automatiquement supprimés et vous ne pouvez plus les récupérer dans ce cas.
