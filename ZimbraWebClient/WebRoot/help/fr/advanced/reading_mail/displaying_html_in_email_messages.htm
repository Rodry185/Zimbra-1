Affichage HTML dans les mails

Vous pouvez afficher les mails selon deux formats.

  • HTML. Les mails HTML peuvent comporter un formatage de texte, une numérotation, des puces, un fond coloré et des liens qui peuvent faciliter la lecture des mails. Il est possible que certains clients de messagerie n'acceptent pas les mails de format HTML.

  • Texte. Les mails en texte simple sont des messages qui ne contiennent pas de code de formatage HTML. Les mails envoyés dans le format texte simple peuvent toujours être lus par d'autres clients de messagerie. Vous pouvez consulter des mails de format texte simple, même s'ils contiennent des balises HTML.

Pour toujours afficher vos mails dans le format HTML, depuis Préférences>Mail, sélectionnezFormat HTML (si possible).  Les mails envoyés dans le format HTML ont une présentation formatée.

Voir également Configuration de vos préférences d'utilisateur.
