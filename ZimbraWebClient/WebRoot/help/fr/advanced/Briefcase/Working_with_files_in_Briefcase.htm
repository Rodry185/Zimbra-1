Travailler avec des fichiers dans un porte-documents

Ouvrir les fichiers dans le porte-documents

Vous pouvez ouvrir des fichiers de vos porte-documents depuis n'importe quel ordinateur. Vous devez disposer de l'application logicielle adéquate pour pouvoir ouvrir, lire ou mettre à jour le fichier. Les formats de fichiers courants, tels que HTML et les fichiers images, tels que . gif ou . jpg, peuvent être ouverts dans différents programmes.

Pour ouvrir un fichier, cliquez avec le bouton droit sur le fichier, puis sélectionnez Ouvrir.

Remarque : Vous pouvez également double-cliquer sur le fichier pour l'ouvrir.

  • Si le fichier est de type inconnu, une boîte de dialogue s'affiche, vous demandant ce que vous voulez faire : l'enregistrer sur l'ordinateur ou l'ouvrir avec un programme spécifique.

  • Si vous avez partagé le Porte-documents avec d'autres utilisateurs, vous pouvez extraire un fichier, ceci afin d'empêcher que d'autres utilisateurs ne le modifient dans le Porte-documents.

    • Cliquez avec le bouton droit sur le fichier, puis sélectionnez Extraire un fichier. Vous pouvez ouvrir le fichier ou l'enregistrer sur votre ordinateur.

    • Uploadez le fichier après l'avoir modifié. Si le fichier uploadé porte le même nom que le fichier de votre dossier de porte-documents, la boîte de dialogue Conflit d'upload s'affiche. Si vous sélectionnez « Oui », le fichier est uploadé et une nouvelle version des fichiers est créée. Si vous sélectionnez « Non », le fichier n'est pas uploadé. Vous devez modifier le nom du fichier ou l'uploader dans un autre dossier de porte-documents.

  • Si le fichier est au format Zimbra, double-cliquez pour ouvrir le fichier dans une page du navigateur. Vous pouvez modifier un document Zimbra depuis cette page.  

Modifier un fichier Zimbra

  • Cliquez avec le bouton droit de la souris sur le document Zimbra, puis sélectionnez Modifier. Le fichier s'ouvre dans une nouvelle page du navigateur.

  • Modifiez la page, puis cliquez sur Enregistrer et fermer pour fermer la page. Si vous ne souhaitez pas enregistrer votre travail, fermez l'onglet du navigateur.

Remarque : En cliquant sur Enregistrer ou Enregistrer et fermer, vous créez une nouvelle version du fichier.

Déplacer des fichiers

Vous pouvez transférer des fichiers d'un porte-documents vers un autre.

  1. Sélectionnez le(s)fichier(s) à transférer.

  2. Dans la barre d'outils, cliquez sur .

  3. Choisissez le dossier de destination ou créez un dossier de porte-documents.

  4. Cliquez sur OK.

Marquer des fichiers

Vous pouvez marquer les fichiers.

  1. Cliquez sur le fichier, puis sur dans la boîte de dialogue.

  2. Choisissez le tag. Vous pouvez uniquement afficher les tags dans la Vue Détails.

  3. Cliquez sur OK.
