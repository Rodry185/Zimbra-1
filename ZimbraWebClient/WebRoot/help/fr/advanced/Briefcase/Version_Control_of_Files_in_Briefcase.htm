Contrôle de la version des fichiers du porte-documents

Le contrôle de version permet de gérer les changements apportés aux documents du porte-documents. Chaque fois que vous modifiez un fichier et que vous enregistrez le fichier, le numéro de version est mis à jour. Le dernier numéro de version figure dans la liste des documents.  

Cliquez sur pour consulter l'historique de version et les auteurs. Pour restaurer une version précédente, cliquez avec le bouton droit de la souris sur une autre version et sélectionnez Restaurer comme version actuelle.  Pour supprimer une version devenue inutile, cliquez avec le bouton droit de la souris sur le fichier et sélectionnez Supprimer.

Remarque : Chaque version d'un fichier est comptabilisée dans votre quota de compte.  

Les fichiers que vous avez envoyés peuvent être extraits afin d'empêcher que plus d'une personne travaille sur le même fichier du Porte-documents. Ces fichiers sont alors téléchargés sur votre bureau. En envoyant le fichier, il est enregistré et la version est modifiée.
