使用拼字檢查

傳送郵件前，你可檢查郵件的拼字。按下工具列中的拼字檢查。

如果你希望在傳送前自動檢查所有郵件的拼字，可在偏好設定 > 寫信頁面中啟用傳送郵件前強行進行拼字檢查。

  1. 按下拼字檢查。拼字錯誤或未知的詞將會凸顯。

  2. 按凸顯的詞。此時會彈出一個有改錯建議的清單。

  3. 從中選擇正確的詞。該詞即被凸顯為其他顏色。

  4. 若接受所做的改變並關閉拼字檢查程式，按下繼續編輯。

在拼字檢查詞典中加入詞句

你可在拼字檢查詞典中加入詞句。

  1. 按下拼字檢查。拼字錯誤的詞將會凸顯。

  2. 按凸顯的詞。此時會彈出一個有改錯建議的清單。

  3. 如果詞拼字正確，按下該詞並按加入。該詞即被加入詞典。

加入詞後，即不能將其從你的詞典中移除。
