Apertura dei file allegati

I messaggi email possono includere file allegati come documenti di elaborazione testi, fogli di calcolo, file di testo, file ZIP, immagini, applicazioni eseguibili o qualsiasi altro tipo di file che risieda nella rete di computer del mittente. I messaggi che contengono allegati sono contraddistinti da una piccola graffetta accanto all'oggetto.

Alcuni tipi di file, ad esempio le immagini JPEG o i file di testo, possono essere aperti sulla maggior parte dei sistemi. Altri, ad esempio i documenti Microsoft Word, devono essere aperti nella stessa applicazione in cui sono stati creati.

Quando ricevi un messaggio contenente file allegati, puoi cliccare sul link dell'allegato a forma di graffetta nell'intestazione per andare alla fine del messaggio. Da qui puoi scegliere di aprire l'allegato, visualizzarlo come HTML o scaricarlo.

  • Se scegli di visualizzare l'allegato, è necessario che nel tuo computer sia installata l'applicazione che consente di leggere quel tipo di file.

  • Se scegli di scaricarlo, viene visualizzata una finestra con il pulsante Sfoglia che mostra il tuo file system corrente. Scegli un percorso appropriato e salva il file. Potrai aprirlo in un secondo momento anche al di fuori del tuo client di posta.

Visualizzazione di allegati come HTML

È possibile che nel tuo sistema sia disponibile la funzione che consente di visualizzare gli allegati come HTML. Ciò consente di aprire un allegato anche se l'applicazione corrispondente non è installata.  Clicca su Visualizza come html per visualizzare il contenuto del file. Questa funzione viene impostata dal tuo amministratore del sistema.

Protezione dai virus

La diffusione dei virus informatici può avvenire anche attraverso gli allegati email. Gli amministratori della posta seguono la prassi ormai consolidata di implementare i filtri anti-virus nella posta in arrivo e di eseguire la scansione anti-virus nei personal computer. Tuttavia, esiste sempre la possibilità che qualche virus riesca ad aggirare tutte le difese impostate.

Nel caso dei virus della posta, l'attivazione avviene quasi sempre al momento dell'apertura o della visualizzazione del file allegato che li contiene. Molti virus si diffondono attraverso le voci delle Rubriche degli utenti. Per questo motivo, è importante aprire sempre con grande cautela i file allegati, anche quando provengono da mittenti conosciuti e affidabili.
