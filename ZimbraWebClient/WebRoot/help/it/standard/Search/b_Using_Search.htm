Uso della funzione di ricerca

La funzione di ricerca di @CompanyName@ è uno strumento molto potente. Sebbene sia possibile imparare a creare query di testo complesse, come viene descritto in Descrizione del linguaggio di query, un metodo più rapido consiste nel digitare un nome o una parola, senza parole chiave. Ad esempio, per trovare tutti i messaggi contenenti la parola ora in qualsiasi posizione all'interno dell'oggetto, nella riga a:, da:, cc: o ccn:, nel corpo del messaggio o in un file allegato, è sufficiente digitare ora nella casella di testo Trova e cliccare su Cerca.

Puoi quindi scegliere dove eseguire la ricerca:  email, contatti,  appuntamenti, impegni,  pagine di documenti e file in Valigetta).  Inoltre, per trovare le informazioni desiderate puoi scegliere di effettuare la ricerca in tutte le cartelle.

Per eseguire una ricerca di contatti nella tua Rubrica:

Per cercare contatti presenti nelle tue Rubriche, la query deve essere una parola intera. Ad esempio, per cercare dei contatti in base al numero di telefono, devi inserire il numero completo così come appare nella voce del contatto. Specificando solo il prefisso, infatti, non viene restituito alcun risultato.

  1. Nella barra di ricerca, nel menu a tendina accanto al campo di ricerca, seleziona Contatti personali. La ricerca verrà eseguita in tutte le tue Rubriche.

  2. Inserisci un nome completo o altri dati sul contatto nel boxino di ricerca.

  3. Clicca su Cerca per eseguire la ricerca.

Per ricercare rapidamente altri tipi

Puoi inserire qualsiasi valore, ad esempio una parola, il nome, il cognome, il numero di telefono o il nome di dominio.   Per altre opzioni, puoi usare l'asterisco* come carattere jolly.  

Uso di * come carattere jolly nelle ricerche

L'asterisco* può essere usato come carattere jolly nelle ricerche per trovare contenuto che comprende parole simili fra loro.  

Puoi usare l'asterisco come carattere jolly dopo un prefisso. Ad esempio, la ricerca di do* restituisce parole come dono, donatore, domanda, ecc.

  1. Nella barra di ricerca, nel menu a tendina accanto al campo di ricerca, scegli l'elemento da cercare oppure seleziona Tutti i tipi di elementi per eseguire la ricerca in tutte le cartelle del tuo account.

  2. Inserisci un nome o altri dati relativi alla mail nel boxino di ricerca.

  3. Clicca su Cerca per eseguire la ricerca.
