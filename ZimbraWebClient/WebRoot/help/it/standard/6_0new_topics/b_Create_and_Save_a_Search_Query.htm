Creazione e salvataggio di una query di ricerca

Se ti capita di eseguire spesso la stessa query di ricerca, può essere utile crearla e salvarla nella cartella Ricerche. Per ripetere la ricerca, non dovrai fare altro che cliccare sulla cartella della ricerca salvata; la ricerca viene eseguita e i risultati vengono visualizzati nel riquadro del contenuto.

Per creare e salvare una query di ricerca

  1. Nell'intestazione Riquadro di navigazione>Ricerca, clicca su modifica. Il riquadro di navigazione si aggiorna per mostrare tutte le cartelle in ordine alfabetico e il riquadro del contenuto mostra la casella In arrivo.

  2. Nella barra degli strumenti, seleziona Nuova ricerca. Nel riquadro del contenuto viene visualizzato un modulo per una nuova ricerca.

  3. Nel campo Nome, assegna un nome alla ricerca.

  4. Nel campo Cartella principale, seleziona un percorso in cui salvare la ricerca. L'impostazione predefinita è la cartella principale. La cartella viene salvata sotto l'intestazione Ricerca nel riquadro di navigazione. Se vuoi salvare la ricerca in un'altra cartella, selezionala dall'elenco a tendina. Puoi riconoscere la cartella di ricerca dall'icona della lente di ingrandimento.

  5. Nel campo Query di ricerca, crea la tua query di ricerca. Consulta Descrizione del linguaggio di query per informazioni sulla sintassi di ricerca.

  6. Nella finestra di dialogo Salva ricerca, assegna un nome alla cartella di ricerca salvata e seleziona un percorso in cui crearla. Se decidi di salvarla nella cartella principale, la ricerca verrà visualizzata nella sezione Ricerche. Se selezioni un'altra cartella, la ricerca verrà salvata nella cartella prescelta.

  7. Clicca su OK.

Per eseguire quella ricerca, clicca sulla cartella di ricerca nel riquadro di navigazione. I risultati verranno visualizzati nel riquadro del contenuto.
