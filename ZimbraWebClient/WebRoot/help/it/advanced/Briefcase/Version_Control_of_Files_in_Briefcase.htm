Controllo delle versioni dei file nella Valigetta

Il controllo delle versioni consente di gestire le modifiche apportate ai documenti nella Valigetta. Ogni volta che si apporta una modifica a un file e lo si salva, il numero di versione viene aggiornato. Il numero di versione più recente è indicato nell'elenco dei documenti.  

Clicca su per visualizzare la cronologia e gli autori delle versioni. Per ripristinare una versione precedente, clicca con il tasto destro del mouse su un'altra versione e scegli Ripristina come versione corrente.  Per eliminare una versione non più necessaria, clicca con il tasto destro del mouse sul file e scegli Elimina.

Nota: Ogni versione di un file occupa parte della quota dell'account dell'utente.  

I file caricati possono essere estratti (checkout) per impedire che più persone lo utilizzino contemporaneamente nella Valigetta. Questi file vengono scaricati sul desktop. Quando si carica il file aggiornato, esso viene archiviato (checkin) e ne viene modificata la versione.
