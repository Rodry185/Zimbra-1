Invio di file dalla Valigetta

Quando lavori nelle cartelle della Valigetta, puoi inviare un link che rimandi ai file in essa contenuti oppure inviare i file veri e propri come allegati di messaggi email. I destinatari devono disporre del software appropriato per aprire i file.

Invio del link a un file tramite email

Puoi inviare una mail contenente link a file presenti nella cartella della Valigetta. Per poter lavorare con i file, il destinatario deve disporre dell'autorizzazione alla condivisione. Tutti i cambiamenti che il destinatario apporta al file sono riportati automaticamente nel file contenuto nella tua Valigetta.

  1. Apri la cartella della Valigetta e clicca con il pulsante destro del mouse sui file che desideri inviare come link.

  2. Clicca su Invia nella barra degli strumenti e seleziona Invia link.
    Si apre la pagina di composizione delle mail con l'indirizzo URL di ciascun link visualizzato nel corpo del messaggio.

  3. Aggiungi l'indirizzo del destinatario e completa il messaggio email.

  1. Clicca su Invia.

 

Invio di un file dalla Valigetta tramite email

Puoi inviare elementi della Valigetta come allegati di posta direttamente dalla Valigetta.  

  1. Apri la cartella della Valigetta e clicca con il pulsante destro del mouse sui file da inviare.

  2. Clicca su Invia nella barra degli strumenti e seleziona Invia come allegato.
    Si apre la pagina di composizione delle mail con l'indirizzo del link visualizzato nel corpo.

  3. Aggiungi l'indirizzo del destinatario e completa il messaggio email.

  1. Clicca su Invia.
