Importazione di elementi nell'account

L'opzione Importa/Esporta della cartella Preferenze consente di importare i dati completi dell'account, determinate cartelle, agende ed elenchi di contatti.

Importazione di contatti

Puoi importare elenchi di contatti salvati in file di testo delimitati da virgole (formato .csv). L'importazione è disponibile dalla cartella Preferenze>Importa/Esporta.

  1. Nella scheda Preferenze, clicca sulla cartella Importa/Esporta.

  2. Nella sezione Importa, seleziona il tipo Contatti.

  3. Nel menu a tendina accanto a Contatti, seleziona il tipo di file di contatti da importare:

    • Contatti Zimbra

    • Contatti Yahoo!

    • Contatti GMail

    • Contatti Outlook 2000

    • Contatti Outlook 2003

    • Contatti Outlook

    • Contatti Thunderbird

È possibile che gli elenchi di contatti dei vari account non visualizzino le informazioni sui contatti nello stesso ordine. Per far sì che le informazioni vengano visualizzate correttamente nella Rubrica, è importante selezionare il tipo giusto di file di contatti da importare.

  1. Usa il pulsante Sfoglia per individuare il file di contatti .csv da importare.

  2. Clicca su Importa. Nella finestra di dialogo Scegli Rubrica, seleziona una rubrica esistente a cui aggiungere i contatti, oppure creane una nuova in cui importare il file.

  3. Clicca su OK. Il contenuto del file dei contatti viene importato.

Importazione dell'Agenda iCal

Puoi importare agende iCalendar (iCal) salvate in formato .ics. iCalendar è il formato standard per lo scambio di dati relativi ad agende su Internet e .ics è il formato standard delle informazioni iCalendar. L'importazione è disponibile dalla cartella Preferenze>Importa/Esporta.

Puoi importare la tua agenda iCal in un'agenda esistente o creare una nuova agenda e importarvi il file .ics.

Importazione di un'agenda iCal

  1. Nella scheda Preferenze, clicca sulla scheda Importa/Esporta per aprirla.

  2. Nel campo Tipo della sezione Importa, seleziona Agenda.

  3. Nel campo File, seleziona con il pulsante Sfoglia il file .ics da importare.

  4. Clicca su Importa. Nella finestra di dialogo Seleziona cartella, indica la cartella in cui importare l'agenda, selezionando un'agenda esistente o creandone una nuova.

  5. Clicca su OK. L'agenda viene importata nel tuo account.

Le agende iCal vengono aggiornate periodicamente in base agli intervalli di ricezione automatica dell'agenda remota impostati dall'amministratore di sistema.
