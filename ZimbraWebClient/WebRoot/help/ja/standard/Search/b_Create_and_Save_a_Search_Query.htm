検索クエリを作成して保存する

同じ検索を頻繁に実行する場合は、検索クエリを作成し、それを[検索]フォルダに保存しておくと便利です。その検索を再び実行したいときには、保存した検索フォルダをクリックするだけで検索が実行され、その結果が内容ペインに表示されます。

検索クエリを作成して保存するには

  1. [概要]ペインの[検索]見出しで[編集]をクリックします。 [概要]ペインが更新されて、すべてのフォルダがアルファベット順で表示されます。また、[内容]ペインには[受信箱]の内容が表示されます。

  2. ツールバーの[新しい検索]を選択します。内容ペインに[新しい検索]フォームが表示されます。

  3. [名前]フィールドに、検索の名前を入力します。

  4. [親フォルダ]フィールドで、この検索の保存場所を選択します。 デフォルトはルートフォルダです。このフォルダは、概要ペインの[検索]見出しの下に保存されます。 検索を別のフォルダに保存する場合は、ドロップダウンリストからそのフォルダを選択します。検索フォルダには虫眼鏡のアイコンが表示されます。

  5. [検索クエリ]フィールドで、検索クエリを作成します。使用可能な検索構文の詳細については、「クエリ言語の記述」を参照してください。

  6. [検索を保存]ダイアログで、検索フォルダに名前を付け、検索フォルダの作成場所を指定します。 ルートフォルダに保存した場合は、その検索が[検索]領域に表示されます。別のフォルダを選択した場合は、その検索がそのフォルダ内に保存されます。

  7. [OK]をクリックします。

保存した検索を実行するには、概要ペインでその検索フォルダをクリックします。内容ペインに検索結果が表示されます。
