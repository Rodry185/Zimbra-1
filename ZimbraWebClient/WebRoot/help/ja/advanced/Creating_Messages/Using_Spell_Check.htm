スペルチェックを使用する

メールを送信する前にスペルをチェックできます。 作成ツールバーの[スペルチェック]をクリックします。

送信前にすべてのメッセージで自動的にスペルチェックをするには、[プリファレンス]>[ 作成]ページで[メッセージを送信する前に必ずスペルチェックを行う]を有効にします。

  1. [スペルチェック]をクリックします。正しいスペルでない単語や不明な単語がハイライトされます。

  2. ハイライトされている単語をクリックします。ポップアップが開き、修正候補が表示されます。

  3. 正しい単語を選択します。その単語が別の色でハイライトされます。

  4. 変更を受け入れてスペルチェックを終了するには、[編集を再開]をクリックします。

スペルチェック辞書に語句を追加する

スペルチェック辞書に単語を追加できます。

  1. [スペルチェック]をクリックします。正しいスペルでない単語がハイライトされます。

  2. ハイライトされている単語をクリックします。ポップアップが開き、修正候補が表示されます。

  3. 単語のスペルが正しい場合、その単語をクリックして[追加]をクリックします。その単語が辞書に追加されます。

一度追加した単語は、辞書から削除できません。
