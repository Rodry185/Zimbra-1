Trabajar con archivos en el Maletín

Abrir los archivos incluidos en tu maletín

Puedes abrir los archivos en tu Maletín desde cualquier ordenador. Debes tener el software adecuado para abrir, leer o modificar ese archivo. Los formatos de archivo comunes, como los archivos de texto y los archivos HTML, y las imágenes, como los archivos . gif o . jpg, se pueden abrir con una gran variedad de programas.

Para abrir el archivo, haz clic con el botón derecho del ratón en él y selecciona Abrir.

Nota: Si haces doble clic, también se abre el archivo.

  • Si el tipo de archivo es desconocido, se abrirá un cuadro de diálogo que te solicitará qué deseas hacer con el archivo, guardarlo en el disco o abrirlo con una aplicación específica.

  • Si has compartido el Maletín con otras personas, es posible que quieras enviar el archivo para evitar que otros usuarios editen el archivo en el Maletín.

    • Haz clic con el botón derecho en el archivo y selecciona Enviar archivo. Puedes abrir el archivo o guardarlo en el ordenador.

    • Una vez realizados los cambios, carga el archivo. Si el archivo que se está cargando tiene el mismo nombre que el archivo de la carpeta de tu maletín, se mostrará un cuadro de diálogo denominado Conflicto de carga. Si seleccionas Sí, el archivo se cargará y se creará una nueva versión de los archivos. Si seleccionas No, el archivo no se cargará. Debes cambiar el nombre de archivo o cargarlo a otra carpeta del maletín.

  • Si el tipo de archivo es un documento de Zimbra, haz doble clic para abrirlo en una página del navegador. No se puede editar un documento de Zimbra desde esta vista.  

Editar un archivo de documento de Zimbra

  • Haz clic con el botón derecho en el archivo del documento de Zimbra y selecciona Editar. El archivo se abrirá en una nueva página del navegador.

  • Edita la página y haz clic en Guardar y cerrar para cerrar la página. Si no deseas guardar tu trabajo, cierra la pestaña del navegador.

Nota: Al hacer clic en Guardar o Guardar y cerrar, se crea una nueva versión del archivo.

Mover archivos

Puedes mover los archivos de un maletín a otro.

  1. Selecciona el(los) archivo(s) que quieres mover.

  2. En la barra de herramientas, haz clic en .

  3. Selecciona la carpeta de destino o crea una nueva carpeta del maletín.

  4. Haz clic en Aceptar.

Etiquetar archivos

Puedes etiquetar archivos.

  1. Haz clic en el archivo y, en la barra de herramientas, haz clic en .

  2. Selecciona la etiqueta. Sólo se pueden ver las etiquetas en la vista Detalles.

  3. Haz clic en Aceptar.
