Teclas de acceso directo de Zimbra

Puedes usar las teclas de acceso directo para navegar por el programa Zimbra Web Client (ZWC) y puedes crear tus propias combinaciones de teclas. Ve a Preferencias>Accesos directos para ver una lista de las teclas de acceso directo o para crear tus propias combinaciones. Las cuatro pestañas incluyen las siguientes opciones:

  • La Lista de accesos directos incluye una lista de accesos directos para todas las aplicaciones.

  • Accesos directos a carpetas de correo te permite crear y asignar los accesos directos a tus carpetas de correo.

  • Accesos directos a búsquedas guardadas te permite crear y asignar los accesos directos a las búsquedas que guardas.

  • Accesos directos para las etiquetas te permite crear y asignar los accesos directos para las etiquetas que hayas creado.
