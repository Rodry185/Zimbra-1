Ver el panel de resumen

En el lateral situado a mano izquierda de  la interfaz Cliente web de Zimbra se encuentran tus carpetas, búsquedas, etiquetas y Zimlets (para la versión Cliente web avanzado).

Carpetas

  • La Bandeja de entrada es donde se coloca el correo nuevo de manera predeterminada

  • Enviados contiene los mensajes que has enviado

  • Borrador contiene los mensajes redactados que todavía no se han enviado

  • Spam contiene los mensajes que se han marcado como spam, ya sea marcados por ti o por el sistema

  • Papelera contiene los elementos que has eliminado

  • A continuación, verás las carpetas que has creado. Si tu cuenta de correo se ha migrado desde otro sistema de correo, como Microsoft® Outlook, es posible que veas carpetas adicionales aquí, que representan las carpetas del sistema de correo anterior

Recuento de mensajes no leídos. El número entre paréntesis al lado de cada carpeta indica el número de mensajes no leídos dentro de esa carpeta. Las carpetas de Etiquetas también muestran el número de mensajes no leídos. Cualquier elemento que contenga mensajes sin leer estará resaltado de negrita.

Para ver el número de mensajes y el tamaño total de la carpeta, pasa el cursor encima de dicha carpeta.

.

Buscar elementos compartidos es un enlace para encontrar y añadir rápidamente las carpetas que se han compartido contigo.  Cuando se abra el cuadro de diálogo Añadir carpeta de elementos compartidos, aparecerá una lista de los elementos que se hayan compartido contigo a través de listas de distribución. En el campo Buscar elementos compartidos, puedes introducir una dirección de correo electrónico para ver si se ha compartido algún elemento contigo desde una dirección específica.

Búsquedas

Esta área contiene las búsquedas que has creado y guardado para usar después.

Etiquetas

Aquí se detallan todas las etiquetas que has creado. Haz clic en una etiqueta para ver todos los mensajes a los que has añadido esa etiqueta. Consulta la sección Usar etiquetas para clasificar tus mensajes para más información.

Zimlets

Los Zimlets se han creado como un mecanismo para integrar datos y contenido de terceros con las funciones de @ClientNameLong@. Los Zimlets te permiten trabajar con diferentes tipos de contenido en tus mensajes de correo. En función de la configuración, es posible que no tengas acceso a los Zimlets desde el panel Resumen.
