Cómo borrar los datos de tu dispositivo móvil

Si pierdes o te roban el dispositivo móvil, puedes borrar todo el contenido del dispositivo mediante el enlace Limpiar dispositivo en la página de Preferencias para Dispositivos móviles. Este enlace borrará todos los datos del dispositivo para restablecer la configuración original del fabricante.

  1. Pulsa en Preferencias y selecciona la carpeta Dispositivos móviles

  2. Selecciona el dispositivo en cuestión y pulsa en Limpiar dispositivo.  

El estado cambiará a Borrado pendiente.  El servidor está esperando a que el dispositivo se conecte al servidor  para iniciar la operación.  Cuando el dispositivo se conecte al servidor, se borrarán todos los datos.  El estado cambiará a Borrado finalizado.

  • Nota:  Puedes cancelar tu solicitud de borrar los datos del dispositivo mientras el estado sea Borrado pendiente.

Además de poder iniciar la eliminación de datos del dispositivo desde el servidor, ZCS admite la configuración existente en cuanto a la seguridad del dispositivo. Una de las medidas de seguridad establece los requisitos para el bloqueo del número PIN. La configuración del bloqueo del número PIN determina cuántas veces se puede intentar introducir la contraseña correcta sin éxito antes de realizar el borrado del dispositivo localmente. Esta operación borrará todos los datos en el dispositivo. Sólo el administrador puede determinar esta configuración que no puede ser modificada por el usuario.
