Teclas de acceso directo

Puedes usar las teclas de acceso directo para navegar por la aplicación de Zimbra Web Client (ZWC).

Para ver una lista de las teclas de acceso directo, teclea Ctrl Q. Verás una lista de teclas de acceso directo organizadas por aplicación.  La lista de todas las aplicaciones detalla las teclas de acceso directo que puedes usar en cualquier aplicación.  Por ejemplo, teclea gc para abrir tu agenda o gt para abrir tu lista de tareas.

También puedes imprimir esta lista. Ve a la carpeta Preferencias>Accesos directos.
