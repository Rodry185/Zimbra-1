Usar las teclas de acceso rápido para navegar

Las teclas de acceso directo te permiten usar el programa sin tener que usar el ratón. Para ver una lista de las teclas de acceso directo, teclea Ctrl Q. Las teclas de acceso directo dependen de la aplicación que estés usando en ese momento aunque algunas se pueden usar en cualquier aplicación. Por ejemplo, en cualquier aplicación, teclea gc para abrir tu agenda o ga para abrir tu libreta de direcciones.

Para tener esta lista de teclas a mano, imprímela desde la pestaña  Preferencias>Accesos directos.
