Guardar el contenido de tu cuenta sin conexión

Puedes exportar todos los datos de tu cuenta, incluidos los mensajes, los ficheros adjuntos, tus contactos, la agenda, las tareas y las carpetas del maletín. También puedes exportar sólo determinados elementos de tu cuenta.  A continuación, podrás guardar los datos en tu ordenador u otro lugar para tener una copia de seguridad. Los datos de tu cuenta se guardarán como un archivo con formato comprimido tar-gzipped (tgz) para que lo puedas importar fácilmente y así recuperar tus datos si hiciera falta.

 Nota: Al exportar las carpetas de contactos una por una, se guardan como archivos .csv; las agendas individuales se guardan como archivos .ics.

Este proceso sirve para copiar los datos de la cuenta, no eliminarlos.

Para exportar un archivo, ve a Preferencias> Importar/Exportar.

Para exportar los datos

  1. En la pestaña Preferencias , haz clic en la carpeta Importar/Exportar.

  2. Puedes exportar todos los datos de tu cuenta o seleccionar una carpeta específica que quieras exportar.

    • Para exportar todos los datos de la cuenta, selecciona la opción Cuenta. Nota: No se exportarán la Papelera ni la carpeta Spam.

    • Para exportar una lista de contactos o una agenda específica, selecciona esa opción. Al seleccionar una agenda o lista de contactos, indica en qué formato quieres que se exporte. Consulte las secciones Exportar las libretas de direcciones y Exportar tu agenda iCal.

    • Si quieres seleccionar varios tipos de elementos para exportarlos a la vez, haz clic en Configuración avanzada para ampliar la vista. Activa las casillas de los datos que quieres exportar. Se exportarán todos los elementos de ese tipo de todas las carpetas en la carpeta seleccionada. Si quieres encontrar algún dato específico, utiliza el filtro de búsquedas para especificar el tipo de datos que quieres exportar. Consulta las secciones Descripción del lenguaje de búsqueda y Ejemplos de búsquedas para obtener más información.  

  1. Haz clic en Exportar e indica dónde quieres guardar el archivo.

  2. Haz clic en Aceptar.

Nota: Puedes usar un programa de archivo, como archiver de WinRAR para ver el archivo comprimido de los datos exportados.

Para importar los datos guardados de tu cuenta

Puedes restablecer una cuenta con el archivo que has exportado.  

  1. En la pestaña Preferencias, haz clic en la carpeta Importar/Exportar.

  2. En la sección Importar, localiza el archivo que quieres importar.  Si vas a importar Agendas o Contactos, especifica el formato del archivo que vayas a importar. Consulta las secciones Importar contactos e Importar tu agenda iCal. Si seleccionas la opción Cuenta, sólo podrás importar a tu cuenta el archivo tgz que ya has exportado.  

Selecciona los tipos específicos de elementos que quieres importar del archivo tgz en la sección Tipos de datos. Al hacer clic en Importar sólo se importarán esos tipos de elementos.

  1. Haz clic en Importar.
