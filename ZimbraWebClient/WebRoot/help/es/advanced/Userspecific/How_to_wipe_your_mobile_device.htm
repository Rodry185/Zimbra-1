Cómo borrar los datos de tu dispositivo móvil

Nota: La opción Móvil únicamente está disponible con la Edición de red @CollaborationProductFullName@

Si pierdes o te roban el dispositivo móvil, puedes borrar todo el contenido del dispositivo mediante el enlace Limpiar dispositivo en la página de Preferencias para Dispositivos móviles. Este comando borrará todos los datos del dispositivo para restablecer la configuración original del fabricante.

  1. Haz clic en Preferencias y selecciona la carpeta Dispositivo móvil.

  2. Selecciona el dispositivo en cuestión y haz clic en Limpiar dispositivo.  

El estado cambiará a Borrado pendiente.  El servidor espera que el dispositivo se conecte al servidor  para iniciar la solicitud.  Cuando el dispositivo se conecta al servidor, el dispositivo se borra.  El estado cambia a Borrado completado.

Nota:  Puedes cancelar la solicitud de Borrar dispositivo mientras el estado sea Borrado pendiente.
