Crear agendas

Se crea una agenda para ti y puedes crear más para estar al tanto de los diferentes tipos de funciones.  Por ejemplo, puedes crear una agenda para anotar las actividades del trabajo, otra para tus actividades personales y otra para las de la empresa.  Puedes seleccionar la vista de agenda programada para ver las agendas una al lado de la otra.

Para crear una agenda

  1. Pulsa en la pestaña Agenda.

  2. En el panel de resumen de la agenda, haz clic en , o en la barra de herramientas,  haz clic en la flecha y selecciona Agenda.

  3. En el cuadro de diálogo Crear nueva agenda, introduce un nombre para la agenda y selecciona el color para visualizar las actividades de esa agenda.

  4. Si no quieres que los demás vean las actividades de esa agenda, activa la casilla Excluir esta agenda cuando se informe de tiempo libre/ocupado.

  5. Para crear una agenda y sincronizar tus citas con las de una agenda remota, activa la opción Sincronizar citas desde la agenda remota. Introduce una dirección URL válida iCal para la agenda remota.

  6. Haz clic en Aceptar. Verás la nueva agenda en la lista de agendas.

Las agendas remotas se actualizan de forma periódica según la frecuencia de actualización para la agenda remota que determina el administrador del sistema.
